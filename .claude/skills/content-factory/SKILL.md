# 养生科普内容工厂

自动化内容生产系统，批量生成高质量的养生科普内容（公众号深度 + 小红书风格）。

## 工作流程

```
选题输入 → 资讯搜索 → 文章生成 → 拆解笔记 → 配图制作 → 内容打包
```

## 使用方法

### 模式1：单主题完整生产

```bash
# 生产一个主题的完整内容包
/content-factory "春季养生" --depth 6 --images 5

# 输出：
# - 春季养生_完整文章.md（1500字专业文章）
# - 春季养生_小红书笔记/（6篇拆解笔记）
#   ├── 笔记1_立春/（5张信息图）
#   ├── 笔记2_雨水/（5张信息图）
#   └── ...
```

### 模式2：节气日历自动化

```bash
# 自动生成今日节气内容
/content-factory --today --auto

# 自动检测今天是哪个节气
# 生成对应内容
# 生成配套图片
```

### 模式3：批量生产

```bash
# 批量生产多个主题
/content-factory --batch topics.txt --output factory-output/
```

## 选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `--topic` | 主题名称 | - |
| `--depth` | 拆解笔记数量 | 3-6 |
| `--images` | 每篇笔记图片数 | 3-5 |
| `--style` | 小红书风格 | cute |
| `--output` | 输出目录 | ./factory-output/ |
| `--search` | 是否搜索最新资讯 | true |
| `--article-only` | 只生成文章，不拆图 | false |

## 内容质量标准

### 文章标准（公众号级别）
- 字数：1500-2000字
- 结构：开头 → 主体（分点）→ 结尾
- 专业性：基于中医理论
- 实用性：具体可操作

### 小红书笔记标准
- 每篇：300-500字
- 重点：1个核心观点
- 配图：3-5张信息图
- 风格：卡通、手绘、莫兰迪色

## 输出结构

```
factory-output/
├── [主题]_完整文章.md          # 公众号级深度文章
├── [主题]_小红书笔记/
│   ├── 笔记1_[副标题]/
│   │   ├── 笔记内容.md        # 小红书文案
│   │   ├── 01-封面.png
│   │   ├── 02-内容.png
│   │   └── ...
│   ├── 笔记2_[副标题]/
│   └── ...
└── 生产记录.md                # 元数据、来源、日期
```

## 资讯来源

自动搜索以下权威来源：
- 丁香医生
- 健康时报
- 中医plus
- 养生中国

## 使用示例

```bash
# 示例1：生产春季养生内容
/content-factory "春季养生" --depth 6 --images 5

# 示例2：生产节气内容
/content-factory "谷雨祛湿" --depth 1 --images 3

# 示例3：只生成文章不拆图
/content-factory "夏季养心" --article-only

# 示例4：指定输出目录
/content-factory "秋季润肺" --output ./2024-autumn/
```

## 执行步骤

### 步骤1：资讯搜索
根据主题搜索最新养生资讯，提取核心观点

### 步骤2：文章生成
基于资讯 + 中医理论，生成专业深度文章

### 步骤3：内容拆解
将长文章拆解为多篇小红书笔记

### 步骤4：配图制作
为每篇笔记生成配套的小红书信息图

### 步骤5：内容打包
整理输出，可直接发布

## 注意事项

- 所有内容基于权威来源
- 引用来源标注
- 避免医疗绝对化表述
- 适合中老年受众
