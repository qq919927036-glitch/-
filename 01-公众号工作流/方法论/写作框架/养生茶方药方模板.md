# 养生茶方/药方 - 写作框架

> 来源文章：健脾疏肝茶（2026-02-18）

## 适用场景
- 节气养生（春养肝、夏养心、秋养肺、冬养肾）
- 节后调理（春节、中秋等大餐后）
- 时令健康（流感季、过敏季等）
- 人群痛点（失眠、便秘、三高等）

## 写作框架

### 1. 开头（50-100字）
**目标：建立信任，引出痛点**

**技巧A：应景古诗**
- 例："好雨知时节，当春乃发生。"
- 库存：
  - 春季：春眠不觉晓 / 绿柳才黄半未匀
  - 夏季：小荷才露尖尖角 / 映日荷花别样红
  - 秋季：空山新雨后 / 秋风萧瑟天气凉
  - 冬季：晚来天欲雪 / 瑞雪兆丰年

**技巧B：痛点描述**
- 例："佳节期间，佳肴满桌...给脾胃一份妥贴的调和"
- 结构：场景 + 痛点 + 解决方案

**技巧C：时节关联**
- 结合节气（立春、雨水、惊蛰等）
- 结合节日（春节、中秋、端午等）
- 结合时令（梅雨季、三伏天、数九寒天等）

### 2. 方名（一句话）
**格式：** [功效] + [品类]
- 例：健脾疏肝茶
- 变体：XX养生茶 / XX调理方 / XX养生汤

### 3. 食材和用量
**格式：**
- 清晰列出每种食材和用量
- 用顿号分隔
- 最后一项用"和"或直接结束

**加分项：**
- 个性化加减（"伤食加山楂，夹寒加紫苏"）
- 显示专业度

### 4. 做法（步骤式）
**格式：**
1、第一步；
2、第二步；
3、第三步。

**要点：**
- 用数字序号
- 步骤简洁明了
- 关键参数（水量、温度、时间）

### 5. 功效（列表式）
**格式：**
✦食材名 + 功效1、功效2；

**技巧：**
- 用对号"✦"增加可读性
- 每个食材单独列出
- 专业术语（疏肝解郁、健脾理气等）
- 最后一句总结："诸药合用，共奏XX功效"

### 6. 适宜人群
**格式：**
XX症状导致YY表现...

**要点：**
- 说清楚什么体质、什么症状
- 具体的表现（身体困重、食欲不振等）

### 7. 禁忌（重要！）
**必须列出：**
- 不适合的体质
- 孕妇、经期等特殊人群
- 有相关疾病的人群

**作用：**
- 规避风险
- 显示专业负责

## 标题公式

**公式1：时节 + 痛点 + 解决方案**
- 春节吃撑了？一杯健脾疏肝茶救你
- 雨水节气湿气重，这杯茶帮你祛湿

**公式2：人群 + 痛点 + 方子**
- 中老年人脾胃虚弱，试试这个茶方
- 50岁后睡眠差，这个方子帮你改善

**公式3：疑问 + 解决方案**
- 为什么春天要疏肝？这杯茶告诉你
- 节后脾胃不和？一杯茶调理到位

## 可复用的金句

### 开头类
- "东风送暖，雨润尘芳"
- "不疾不徐，让身心从XX中慢慢归位"
- "一份妥贴的调和"

### 描述类
- "润物无声"（形容温和的食材）
- "添一抹温润回甘"（形容口感和功效）
- "山珍海味后"（对比突出健康）

### 结尾类
- "让身心从节日的喧嚣中慢慢归位"
- [待补充更多]

## 变体方向

**可以写这个系列：**
1. 春季养生系列（疏肝、明目、养阳）
2. 节气养生方（24节气对应茶方）
3. 人群定制方（中老年、女性、上班族）
4. 症状调理方（失眠、便秘、三高）
5. 时令健康方（流感季、过敏季、祛湿季）
