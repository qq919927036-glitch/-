# 节气养生综合内容 - 写作框架

> 来源文章：立春养生（吃三喝三做三）

## 适用场景
- 24节气养生内容
- 大型养生知识科普
- 季节性健康指南

## 文章结构

### 1. 开头（150-200字）
**元素组合：**
- 季节描述（送走了料峭寒冬，迎来了春暖花开）
- 古文引用（《尚书大传》中写道...）
- 身体状况分析（阳气虚弱、代谢减缓）
- 重要性强调（一年关键时刻，为全年打基础）

**公式：** 季节变化 + 古文引用 + 身体分析 + 养生重要性

### 2. 主体内容（模块化结构）
**采用"数字+主题"的形式：**

#### 模块A：吃三样（或饮食养生）
每样内容的结构：
1. 名称（吃春饼）
2. 古文/俗语引用（"立春日啖春饼，谓之咬春"）
3. 历史渊源（民间习俗/皇宫传统）
4. 营养价值（维生素/功效）
5. 适用人群/注意事项

#### 模块B：喝三样（或茶汤养生）
每样内容的结构：
1. 名称（山药小米粥）
2. 为什么喝（立春时节，乍暖还寒...）
3. 食材分析（小米和胃温中，山药味甘...）
4. 制作步骤（具体做法）
5. 功效总结（滋肾益肝，健脾益胃）

#### 模块C：做三样（或行为养生）
每样内容的结构：
1. 名称（春捂，防春寒）
2. 老话/俗语（"春捂秋冻，不生杂病"）
3. 为什么做（气温变化，阳气趋向于表...）
4. 怎么做（分点说明：头颈部、腹部、足膝部）
5. 中医理论（呼应主题）

#### 模块D：养生秘诀（加分项）
穴位按摩类内容：
1. 穴位名称（内关穴）
2. 位置描述（手腕横纹上两寸）
3. 按摩功效（安宁心神）
4. 适用场景（初春烦躁，影响睡眠）

### 3. 结尾（100字）
**元素组合：**
- 古文引用（《宋史乐志》中有云...）
- 升华主题（春回大地，万象更新）
- 情感共鸣（过去种种，都会成为今日奋发的养料）
- 祝福/呼吁（愿大家早睡早起，身体强壮）

**公式：** 古文引用 + 季节升华 + 情感共鸣 + 健康祝福

## 标题公式

**类型1：数字+主题**
```
✅ 立春养生：吃三样、喝三样、做三样
✅ 雨水节气做好这3件事，整年少生病
✅ 春分养生的"3+3+3"法则
```

**类型2：时节+强调**
```
✅ 立春到了！这9件事一定要知道
✅ 春天养生的黄金期，错过等一年
✅ 立春当天，老祖宗留下的养生智慧
```

**类型3：疑问+解答**
```
✅ 立春后该吃什么？老中医这3个建议
✅ 为什么春天要养肝？这篇文章告诉你
✅ 立春养生，为什么说要"春捂"？
```

## 写作技巧

### 1. 模块化组合
- 用数字标号（01、02、03）
- 每个模块内部用小标题（1、2、3）
- 层次清晰，易于阅读

### 2. 引用增强权威性
- 古文引用
- 民间俗语
- 中医经典
- 老话常说

### 3. 实用性优先
- 每个点都有具体做法
- 食材写明用量和步骤
- 穴位写明位置
- 方法简单易行

### 4. 情感共鸣
- "上了年纪"（拉近距离）
- "您不妨试试"（亲切语气）
- "我们中国人"（集体认同）
- 结尾祝福（美好祝愿）
