# 从格深度学习：边界、假从、不忌混

> 学习时间：2026-02-19
> 学习目标：将从格判断从85%提升到95%
> 核心问题：
> 1. 从格的边界在哪里？
> 2. 假从有哪几种形式？
> 3. 为什么从格较不忌混？

---

## 第一部分：从格的边界判断

### 🤔 核心问题

**什么情况能从？什么情况不能从？**

这是从格判断最难的地方，也是最容易出错的地方。

---

### 1.1 边界判断的三个层次

#### 层次一：能从（真从）

**条件（必须同时满足）：**
```
✅ 日干极弱（克泄交加）
✅ 四柱尽是食伤财官
✅ 无印比生扶（"无一点生扶之气"）
✅ 从神大旺
✅ 无刑冲克害破格
```

**典型案例：三戌知县**
```
丙 戊 癸 戊
戌 戌 巳 戌
```

**判断：**
- 癸水生于戌月（土旺克水）
- 地支三戌（土太旺）
- 日坐巳宫（火生土）
- 丙生戊、戊克壬
- **四柱无一点生扶之气**
- 当从戊土正官为格

**关键：** "四柱无一点生扶之气" → 这就是边界！

---

#### 层次二：勉强从（假从）

**条件（满足其一）：**
```
✅ 日干弱，从神旺，但有微弱根气被制
✅ 日干弱，从神旺，但略有混杂能去清
✅ 日干弱，从神旺，但有克害有救应
```

**关键：** **原局为假从，行真从运亦能富贵！**

> "孤儿异性能出类"

**典型案例：假从之例**
```
戊 辛 丙 己
子 酉 申 丑
```

**判断：**
- 日干丙火与月干辛金一合
- 成化气格
- **假化之因：**
  1. 原局有戊己土逆化气化神
  2. 虽经泄化得宜，仍属假化
  3. 壬癸水混杂无法取清

**大运验证：**
- 甲子、乙丑运：非泄壬留癸，即泄癸留壬，去留得宜 → 亦能发福
- 丙寅运：日干丙火与戊土忌神至寅得长生 → 化气格破局

---

#### 层次三：不能从（正格）

**条件（满足其一）：**
```
❌ 日干有印比生扶
❌ 日干虽弱但有根
❌ 从神不旺
❌ 日干与从神力量相当
```

**典型案例：丁壬皆留**
```
丁 丁 壬 壬
巳 未 寅 寅
```

**判断：**
- 时干壬水坐寅
- 月干丁火坐未
- 月时二干丁壬一合
- **丁壬皆留**

**为什么不能从？**
- 壬水既留在原局
- 有比帮身
- **不能做化气论**
- 此以正财为格

---

### 1.2 边界判断的核心标准

#### 标准1："无一点生扶之气"

**崔老师的严格标准：**
> "四柱无一点生扶之气"

**"生扶之气"包括：**
1. **印（生我）：** 金生水、土生金、水生木、木生火、火生土
2. **比（助我）：** 同类五行，如甲乙木、丙丁火

**判断流程：**
```
第一步：找印
  四柱有没有生日干的五行？

第二步：找比
  四柱有没有同类的五行？

第三步：判断
  无印 + 无比 = 无根 = 可以从
  有印 OR 有比 = 有根 = 不能从
```

---

#### 标准2：从神必须大旺

**为什么从神必须大旺？**
> "从是日干无气、财官食伤大旺。不得不从"

**"大旺"的标准：**
1. 占据主导地位
2. 日干无法抗衡
3. 不得不从

**案例对比：**

**能从：庚金四寅**
```
壬 壬 庚 戊
寅 寅 寅 寅
```
- 四寅（甲木大旺）
- 庚金虚浮无根
- 不得不从

**不能从：庚金有根**
```
甲 丙 庚 壬
寅 寅 申 辰
```
- 虽然寅月木旺
- 但日支申金（庚之禄）
- 有根，不能从

---

#### 标准3：阴干易从，阳干难从

**崔老师的经验：**
> "阴日干易从、阳日较难从"

**为什么？**
- **阴干柔弱：** 容易泄尽 → 容易从
- **阳干刚强：** 不容易泄尽 → 不容易从

**但不是绝对！**
- 关键还是看有没有"生扶之气"
- 阳干被泄尽，也能从
- 阴干有根，也不能从

**案例验证：**

**阳干从的案例：**
```
壬 壬 庚 戊
寅 寅 寅 寅
```
- 庚金（阳干）
- 但四寅木太旺
- 庚金虚浮无根
- 也能从

**阴干不从的案例：**
```
甲 乙 乙 癸
寅 卯 卅 亥
```
- 乙木（阴干）
- 但地支寅卯亥（木局）
- 有强根
- 不能从

---

### 1.3 边界情况的快速判断法

#### 方法：三步判断法

**第一步：看日干强弱**
```
日干极弱？
  YES → 进入第二步
  NO → 不能从，以正格论
```

**第二步：看有没有根**
```
有没有印（生）？
  YES → 不能从
  NO → 进入第三步

有没有比（助）？
  YES → 不能从
  NO → 可以从
```

**第三步：看从神旺不旺**
```
从神大旺？
  YES → 真从
  NO → 假从或不能从
```

---

#### 实战判断表

| 日干 | 印 | 比 | 从神 | 判断 |
|------|-----|-----|------|------|
| 极弱 | 无 | 无 | 大旺 | **真从** |
| 极弱 | 无 | 无 | 旺 | **真从** |
| 极弱 | 微 | 无 | 大旺 | **假从**（微根被制） |
| 极弱 | 无 | 微 | 大旺 | **假从**（微根被制） |
| 极弱 | 无 | 无 | 中旺 | **假从**（从神不够旺） |
| 弱 | 有 | 无 | 大旺 | **不能从**（有印） |
| 弱 | 无 | 有 | 大旺 | **不能从**（有比） |
| 中 | 无 | 无 | 大旺 | **不能从**（日干不够弱） |

## 第二部分：假从的多种形式

### 🤔 核心问题

**假从就是不好的命吗？**

**答：不是！假从行真从运，亦能富贵！**

> "真从之家有几人、假从亦可发其身"
> "孤儿异性能出类"

---

### 2.1 假从的定义

**什么是假从？**

原局虽然日干弱、从神旺，但有以下情况之一：
```
1. 有根（印比）但无制化
2. 有混杂（正偏混）但去不清
3. 有克害从神之神但无救应
```

**关键：** **原局是假从，但行真从的运岁，亦能富贵！**

---

### 2.2 假从的三种形式

#### 形式1：有根无制

**定义：**
- 日干有印比根气
- 但无制化印比之神

**为什么是假从？**
- 有根，日干不从
- 但根弱，勉强从
- **原局假从**

**案例：**
```
甲 丙 戊 壬
辰 寅 戌 午
```

**分析：**
- 壬水日干
- 地支寅午戌会火局
- 天干透甲丙戊
- 壬水弱无气

**问题：**
- 壬水虽弱，但天干透壬
- 自有比肩（虽弱）
- **有根无制**

**判断：**
- 以从丁火正财为格
- 但惜有甲克戊生丙混局
- **而为假从**

---

#### 形式2：正偏混杂

**定义：**
- 从正财，混偏财
- 从正官，混七杀
- 混杂但去不清

**为什么是假从？**
- 从格忌混浊
- 正偏混杂，格局不清
- **原局假从**

**案例：**
```
戊 丙 乙 丙
戌 辰 未 戌
```

**分析：**
- 乙木日干
- 辰戌冲、未戌刑
- 土逢四季，得刑冲愈旺
- 干透双丙一戊
- 丙火之气尽泄于戊

**判断：**
- 当以从戊土正财为格
- **此造从正混偏**
- 好在从格较不忌混
- 只求所从之神兴旺
- 命中火生土旺
- 不杂他神
- **从财得真**

**验证：** 荣登甲榜、平步青云

---

#### 形式3：克害从神无解

**定义：**
- 有克害从神之神
- 但无解救之神

**为什么是假从？**
- 从神是命脉
- 从神被克害
- **原局假从**

**案例：**
```
癸 甲 丁 甲
卯 午 卯 午
```

**分析：**
- 辛日干
- 干透甲丙丁
- 地支卯未会
- 卯木生二午
- 官杀混杂
- 日干弱极

**问题：**
- 丙辛之合因天干透财杀破化气
- **化气不成**
- 以丙火正官为格
- 用未土偏印泄杀为用神

**大运：壬申年**
- 流年壬水克正官格神
- 运支卯会年未
- 原局午未合留之未土用神，因解合会卯而去
- 转生午火七杀
- 运支卯木克用神未土
- **用神尽去**
- 七杀忌神大旺
- **凶亡**

---

### 2.3 假从的运岁判断

**核心理论：**
> "假从行真从之运岁，亦能富贵"

**什么是"真从运"？**
```
1. 从神生旺之运
2. 制去根气之运
3. 去清混杂之运
4. 救应从神之运
```

**典型案例：假从之例**
```
戊 辛 丙 己
子 酉 申 丑
```

**原局：** 假化（有戊己土逆化气）

**甲子、乙丑运：**
- 非泄壬留癸
- 即泄癸留壬
- 去留得宜
- **亦能发福**

**丙寅运：**
- 运干与原局争合
- 日干丙火与戊土忌神至寅得长生
- **化气格破局**
- 凶

---

### 2.4 假从vs真从的实战区别

| 对比项 | 真从 | 假从 |
|--------|------|------|
| **原局** | 无根、清纯、从神旺 | 有根/混杂/克害 |
| **大运** | 行从神旺地大发 | 行真从运大发 |
| **本质** | 原局真从 | 原局假从 |
| **富贵** | 容易富贵 | 运到方富贵 |
| **难度** | 真从之家有几人 | 假从亦可发其身 |

**关键理解：**
- **真从：** 原局好，大运配合更容易发
- **假从：** 原局不够好，但大运配合也能发
- **本质：** 都是贵命，只是发的时间不同

## 第三部分："从格较不忌混"原理

### 🤔 核心问题

**崔老师说：**
> "此造从正混偏、好在从格较不忌混"

**为什么从格较不忌混？**
- 正格忌混
- 从格不忌混（相对而言）
- 原理是什么？

---

### 3.1 正格为什么忌混？

**正格的本质：**
> 正格是日干有根，能够独立

**正格忌混的原因：**

1. **官杀混杂**
   - 有正官，有七杀
   - 阴阳混杂，不清纯
   - **必须去留取清**
   - 否则："君子、小人混杂"

2. **财星混杂**
   - 有正财，有偏财
   - 阴阳混杂，不清纯
   - **必须去留取清**
   - 否则：感情混乱、财来财去

**为什么必须去留取清？**
- 正格日干有根，能独立
- 日干能驾驭格神
- 所以要求格神清纯
- 混杂则乱

---

### 3.2 从格为什么不忌混（相对）？

**从格的本质：**
> 从格是日干无根，弃命从人

**从格较不忌混的原因：**

#### 原因1：日干已弃命，不驾驭格神

**理解：**
- 正格：日干有根，驾驭格神 → 要求格神清纯
- 从格：日干无根，弃命从人 → 不驾驭格神 → 格神混一点没关系

**比喻：**
- 正格：老板，要求员工各司其职，不能混乱
- 从格：员工，听命于人，上面混一点不影响执行

---

#### 原因2：关键是所从之神旺不旺

**崔老师的标准：**
> "只求所从之神兴旺、不杂他神、从财得真"

**重点：**
- ❌ 不是：正偏不混
- ✅ 而是：所从之神旺
- ❌ 不是：格局清纯
- ✅ 而是：不杂他神（不同类五行）

**案例：从正混偏**
```
戊 丙 乙 丙
戌 辰 未 戌
```

**分析：**
- 从戊土正财为格
- 但戊土是阳土
- 辰戌未都是土（同类）
- **虽从正混偏，但同类**
- **不杂他神**
- **从财得真**

**验证：** 荣登甲榜、平步青云

---

#### 原因3：从格的重点是"从"不是"纯"

**理解：**
- 正格重点：纯（格局清纯）
- 从格重点：从（彻底顺从）

**为什么？**
- 从格日干已无根，已弃命
- 关键是"从得真不真"
- 不是"纯不纯"

**案例对比：**

**真从（略混）：**
```
戊 丙 乙 丙
戌 辰 未 戌
```
- 从戊土正财
- 虽混辰未土（同类）
- 但土太旺，从得真
- **大贵**

**假从（纯正）：**
```
甲 丙 戊 壬
辰 寅 戌 午
```
- 从丁火正财
- 虽纯正（火）
- 但有甲克戊生丙混局
- **假从**

**结论：**
- 真从 > 假从
- 混一点不影响真从
- 关键是从神旺不旺

---

### 3.3 "较不忌混"不是"完全不忌"

**重要：**
> "从格较不忌混" ≠ "从格完全不忌混"

**从格还是忌混，只是程度较轻：**

**从格忌混的三个层次：**

1. **完全不忌混（同类混杂）**
   ```
   从正财混偏财（都是土）
   从正官混七杀（都是木）
   ```
   - ✅ 不忌混
   - 原因：同类五行

2. **较不忌混（能去清）**
   ```
   从财混杂，有七杀泄偏财
   从官混杂，有食神制七杀
   ```
   - ✅ 较不忌混
   - 原因：有去清之神

3. **还是忌混（不能去清）**
   ```
   从正混偏，无去清之神
   从官混杂，无制化之神
   ```
   - ❌ 还是忌混
   - 原因：混杂无法取清
   - 结果：假从

---

### 3.4 "较不忌混"的实战应用

#### 应用1：从格判断时的权重

**正格判断：**
```
混杂权重：40%（很重要）
```
- 正格必须去清混杂
- 否则降格

**从格判断：**
```
从神旺度权重：60%（最重要）
混杂权重：20%（相对不重要）
```
- 从格关键是所从之神旺不旺
- 混杂是次要问题

---

#### 应用2：取用神时的优先级

**正格取用：**
```
优先级1：去混杂
优先级2：生格神
```

**从格取用：**
```
优先级1：生从神
优先级2：去根气
优先级3：去混杂（如果需要）
```

**案例对比：**

**正格：去混杂优先**
```
癸 甲 丁 甲
巳 子 酉 辰
```
- 以癸水七杀为格
- 会局之壬水正官混杂
- **优先：用甲木泄壬水**
- 去混杂第一

**从格：生从神优先**
```
庚 癸 己 辛
寅 未 酉 未
```
- 从庚金伤官为格
- **优先：取癸水偏财为用神**
- 生从神第一

---

#### 应用3：大运流年的喜忌

**正格大运：**
```
喜：去混杂之运
忌：混杂加临之运
```

**从格大运：**
```
喜：从神生旺之运
   去根气之运
   去混杂之运（如果需要）
忌：日干生旺之运
   克害从神之运
```

**关键：**
- 从格大运，从神旺第一
- 去混杂是次要的
- 如果原局已经清纯，就不需要去混杂

---

### 3.5 总结："较不忌混"的本质

**核心理解：**

1. **相对不忌，不是完全不忌**
   - 同类混杂（正偏混）→ 较不忌
   - 不同类混杂 → 还是忌
   - 有去清之神 → 较不忌
   - 无去清之神 → 还是忌

2. **从神旺度 > 混杂程度**
   - 从神大旺，略混无妨
   - 从神不旺，混杂致命
   - 关键是"从得真不真"

3. **从格重点是"从"不是"纯"**
   - 正格：日干驾驭格神 → 要求纯
   - 从格：日干弃命从人 → 关键是从
   - 从得真 > 格局纯

## 第四部分：综合实战案例

### 案例1：真从但略混

**八字：**
```
戊 丙 乙 丙
戌 辰 未 戌
```

**分析步骤：**

**第一步：判断日干强弱**
- 乙木日干
- 生于辰月（土旺，非木月）
- 辰戌冲、未戌刑
- 土逢四季，得刑冲愈旺
- **日干极弱**

**第二步：判断有没有根**
- 有没有印（水）？无
- 有没有比（木）？无
- **无根**

**第三步：判断从神旺不旺**
- 干透双丙一戊
- 丙火之气尽泄于戊
- 土太旺
- **从神大旺**

**第四步：判断清纯度**
- 从戊土正财为格
- 但戊、辰、未、戌都是土
- **虽从正混偏，但同类**
- **不杂他神**
- **较不忌混**

**结论：**
- 从戊土正财为格
- 从正混偏，但同类
- 好在从格较不忌混
- 从财得真
- **大贵**

**验证：** 荣登甲榜、平步青云

### 案例2：假从（有根无制）

**八字：**
```
戊 甲 壬 丙
辰 寅 戌 午
```

**分析步骤：**

**第一步：判断日干强弱**
- 壬水日干
- 地支寅午戌会火局
- 天干透甲丙戊
- 壬水弱无气

**第二步：判断有没有根**
- 有没有印（金）？无
- 有没有比（水）？天干透壬（自坐比肩）
- **有微根**

**第三步：判断从神旺不旺**
- 寅午戌会火局
- 火太旺
- **从神大旺**

**第四步：判断清纯度**
- 以从丁火正财为格
- 但有甲克戊生丙混局
- **混杂**

**结论：**
- 以从丁火正财为格
- 有微根（壬水）
- 有混杂（甲克戊生丙）
- **假从**

### 案例3：不能从（有根）

**八字：**
```
丁 丁 壬 壬
巳 未 寅 寅
```

**分析步骤：**

**第一步：判断日干强弱**
- 壬水日干
- 坐寅（木，泄水）
- 月干丁火克
- 时干壬水帮
- 日干弱

**第二步：判断有没有根**
- 有没有印（金）？无
- 有没有比（水）？时干壬水，日支寅中藏壬水
- **有根**

**第三步：判断能否从**
- 壬水既留在原局
- 有比帮身
- **不能做化气论**
- **不能从**

**结论：**
- 此以正财为格
- 用比肩壬水制偏财
- **正格，不从**

## 第五部分：学习总结

### 5.1 从格边界判断的核心要点

**关键标准：**
1. **"四柱无一点生扶之气"** - 最严格标准
2. **从神必须大旺** - 不得不从
3. **阴干易从，阳干难从** - 但不是绝对

**快速判断：三步法**
```
第一步：日干极弱？
第二步：无印无比？
第三步：从神大旺？
  YES YES YES → 真从
```

---

### 5.2 假从理解的核心要点

**关键认识：**
1. **假从不是不好**
   - 假从行真从运，亦能富贵
   - "孤儿异性能出类"

2. **假从的三种形式**
   - 有根无制
   - 正偏混杂
   - 克害从神无解

3. **假从vs真从**
   - 真从：原局好，运配合更容易发
   - 假从：原局不够好，运到方发
   - 都是贵命，只是发的时间不同

---

### 5.3 "从格较不忌混"的核心要点

**关键理解：**
1. **相对不忌，不是完全不忌**
   - 同类混杂 → 较不忌
   - 不同类混杂 → 还是忌

2. **从神旺度 > 混杂程度**
   - 从神大旺，略混无妨
   - 从神不旺，混杂致命

3. **从格重点是"从"不是"纯"**
   - 正格：驾驭格神 → 要求纯
   - 从格：弃命从人 → 关键是从

---

### 5.4 实战应用优先级

**从格判断优先级：**
```
优先级1：日干强弱（40%）
优先级2：有没有根（30%）
优先级3：从神旺度（20%）
优先级4：清纯度（10%）
```

**从格取用优先级：**
```
优先级1：生从神
优先级2：去根气
优先级3：去混杂（如果需要）
```
