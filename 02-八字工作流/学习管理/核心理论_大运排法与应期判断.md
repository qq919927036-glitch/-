## 🎯 第一部分：大运排法（顺行、逆行）

### 核心规则（最重要！）

**规则：**
```
阳男阴女：顺行
阴男阳女：逆行
```

---

### 阳年阴年判断

**阳年：**
```
甲、丙、戊、庚、壬 = 阳年
```

**阴年：**
```
乙、丁、己、辛、癸 = 阴年
```

---

### 排法示例

**高际阳案例：**
```
男命，1966年生（丙午年）
丙 = 阳年
男命 + 阳年 = 顺行！
```

**大运顺序：**
```
辛卯 → 壬辰 → 癸巳 → 甲午 → 乙未 → 丙申 → 丁酉 → 戊戌
```

---

### 崔老师原话（行号 332-336）

> "例如甲木为格用喜神，生于戌月，阳男阴女大运顺行。途经亥.子.丑.寅.卯.辰。甲木由长生走到帝旺。气势顺行，就可以真神当真。又假如大运逆行，经酉.申.未.午.巳等甲木死绝之地，气势逆行。"

**理论要点：**
- 顺行 = 气势顺行，真神当真
- 逆行 = 气势逆行，真神不当真

## 🔑 第三部分：大运应期的判断方法

### 核心原则（行号 389-390）

**崔老师原话：**
> "原局格用神，与大运流年顺逆同道，有体用者吉利。与岁运不和克害而无体用者不利。"

**原则：**
- **格用神与大运流年顺逆同道，有体用 → 吉利**
- **与岁运不和克害而无体用 → 不利**

---

### 应期判断要点

**1. 好运应期：**
- 行格神、用神之地
- 行格神之运或行用神之运者 → **吉利**

**2. 坏运应期：**
- 行格神用神死绝之地
- 行忌神得强根之地 → **不利**

**3. 感应触发：**
- 原局有的组合，大运透出时
- **引发该组合的吉凶**

---

### 崔老师原话（行号 329-338）

> "如果原命局假神没有去留，而大运和流年有去留，在这个大运流年里也可以春风得意。滴天髓书中说;『顺逆不齐也，不可逆者,顺其气势而已矣』。这就是『不能逆者、顺其气势而巳矣』之义。所以莫道假神有害处，真神死绝假神当真。"

**理论要点：**
- 假神在大运流年有去留 → 也可以春风得意
- 不能逆者，顺其气势
- 真神死绝假神当真

## 🎯 第六部分：大运吉凶判断的"冲战合好"四字诀

### 崔老师原话（行号 365-382）

> "冲战合好四字，
> 冲：冲去原命忌神者喜，冲喜神不吉。
> 战：有大运旺真神，流年旺假神，或流年凑成真假神混战者。
> 和：是合也，应当合的时候，合就吉。不应当合的时候，而合就凶。
> 好：好是流年吉利的意思，大运流年同为一气，同为吉利的好年运。"

---

### 四字诀详解

**1. 冲（冲去）：**
- **冲去原命忌神 → 喜**
- **冲喜神 → 不吉**

**2. 战（混战）：**
- 大运旺真神
- 流年旺假神
- **或流年凑成真假神混战**

**3. 和（合）：**
- 应当合的时候，合就吉
- 不应当合的时候，合就凶

**4. 好（吉利）：**
- 大运流年同为一气
- **同为吉利的好年运**

---

### 核心原则（再次强调）

**崔老师原话：**
> "原局格用神，与大运流年顺逆同道，有体用者吉利。与岁运不和克害而无体用者不利。"

**原则：**
- 格用神与大运流年顺逆同道 + 有体用 → **吉利**
- 与岁运不和克害 + 无体用 → **不利**
