## 🔍 第二步：定格取用

### 2.1 定格局（识别格神）

#### 格神的定义：

> **格 = 命局中最强、最清纯、最有力的十神 = 真神**

#### 定格四原则：

**原则1：净阴净阳**
```
十神混杂？
→ 需要去留取清
→ 谁清纯谁为格
```

**原则2：察旺衰**
```
谁最旺？
→ 旺者为真神
→ 可以为格
```

**原则3：明变化**
```
有合化？
→ 合去的是假神
→ 合留的是真神
```

**原则4：辨真假**
```
甲木为真神，乙木为假神
→ 有制化乙木之神
→ 以甲木为格
```

---

### 2.2 取用神

#### 用神的定义：

> **用 = 辅佐格神的五行 = 制化忌神的神 = 生扶格神的神**

#### 四种典型情况：

**情况1：官杀混杂**
```
正官为格，七杀混杂
→ 用偏印泄七杀
→ 或用食神制杀
```

**情况2：正官弱**
```
正官为格，但弱
→ 用正财生官
```

**情况3：正官遇伤官**
```
正官为格，伤官克
→ 用正财解救
```

**情况4：七杀格**
```
七杀为格，正官混杂
→ 用伤官制官
→ 或用正印泄官
```

## 💡 第三步：明喜忌

### 3.1 喜神判断

**有利于格神 = 喜**

```
生扶格神 → 喜
制化忌神 → 喜
保护格神 → 喜
```

### 3.2 忌神判断

**不利于格神 = 忌**

```
克制格神 → 忌
泄耗格神 → 忌
混杂格神 → 忌
```

### 3.3 重要原则

**崔老师原话：**
> "忌神被克制越净八字格局越高"
> "格神高大，用神有力，是大富大贵"

---

## 🔑 去留取清方法

### 核心原理：

**崔老师原话：**
> "正宗子平对阴阳五行的具体操作和用法，是去浊留清，净阴净阳"
> "命局的格用喜神要取清而不能混杂，混杂就会引起征战"
> "如果原局混杂无法去留取清者，则下命可决，没有富贵可言"

---

### 操作方法：

**方法1：合化去留**
```
丙辛合 → 丙火去，辛金留
甲己合 → 甲木去，己土留
乙庚合 → 乙木去，庚金留
```

**方法2：冲战去留**
```
子午冲 → 子水去，午火去
冲去忌神 → 喜
冲去喜神 → 凶
```

**方法3：制化去留**
```
用食神制杀 → 七杀去
用偏印泄杀 → 七杀去
用财星破印 → 印星去
```

## 📊 经典案例验证

### 案例1：慈禧太后（劫财格，12级）

**八字：**
```
乙 丁 乙 丁
未 亥 丑 亥
```

**定格取用：**
```
地支亥未会出甲木，天干透出乙木
命局中甲乙木比劫混杂
但此造贵在月时干丁火透出，有力的泄去乙木，留下甲木
甲木劫财非常清纯而且有力，当以甲木劫财为格
以丁火泄乙木，壬水生甲木为用神
```

**核心：**
- 丁火泄乙木 → 去假留真
- 甲木清纯 → 格神高大
- **12级帝王命！**

### 案例2：李鸿章（比肩格，假神代真）

```
此造寅月亥卯未全，甲乙木同旺混杂
幸得月时二干甲己一合，甲木合去
以乙木比肩为格
用癸水偏印生格
```

**假神代真神：**
```
己酉运：己土大运复出合留甲木、克制癸水、酉金冲卯木
甲木留、乙木去。假神代真神，原局依然得清
这是命理奥妙之处
```

**核心：**
- 甲己合去甲木 → 去假留真
- 假神代真神 → 依然得清
- **不凶！**
