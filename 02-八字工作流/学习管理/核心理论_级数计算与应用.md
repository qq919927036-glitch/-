## 🎯 第一部分：级数的基本计算方法

### 完整计算公式

**崔老师公式：**
```
实际级数 = 原局级数 + 暗拱暗会级数 - 忌神扣除级数 - 降级级数
```

**崔老师原话：**
> "以[原局]所得级数，加上[暗拱暗会]的级数。若有[克泄刑害]忌神，虽在原局有制化，亦需扣除。而后所得的，才是实际级数。"

## 💡 第四部分：级数升降（半级制）

### 降半级的条件

**崔老师案例：**

1. **忌神有气未彻底制服**
> "辛金忌神在日为禄地，月令土又生金，这个辛金有气没有彻底被制服，从皇帝位置降半级"

2. **用神被泄但泄力不足**
> "辛金有癸水泄；但是癸水在弱地，泄力不足，降半级"

3. **格神被泄**
> "庚金格神也被癸水泄，降半级"

4. **格神坐绝地**
> "庚金坐绝地，用神戊土力量不足又降半级"

5. **暗藏长生克格神**
> "丙火长生于暗藏寅木克庚金，又降半级"

---

### 降半级触发条件表

| 条件 | 降级 | 理由 |
|------|------|------|
| 忌神有气未彻底制服 | 降半级 | 忌神仍有力量 |
| 用神被泄但泄力不足 | 降半级 | 泄得不彻底 |
| 格神被泄 | 降半级 | 格局受损 |
| 格神坐绝地 | 降半级 | 格神无力 |
| 用神力量不足 | 降半级 | 用神太弱 |
| 暗藏长生克格神 | 降半级 | 隐形伤害 |

## 🎯 第六部分：忌神扣除规则

### 扣除原则

**崔老师原话：**
> "喜神有刑冲克害或混杂为分，无刑冲克害或混杂或得暗拱暗会为级"

**理解：**
- 有刑冲克害或混杂 = 分（不足1级）
- 无刑冲克害或混杂 = 级（满分）
- **扣除 = 从满分中减去**

---

### 扣除案例

**案例1：**
> "辰到未供午四级，丑前拱寅木六级，扣除庚金克乙木二级，余8级"

**计算：**
- 辰到未拱午 = 4级
- 丑前拱寅 = 6级
- 总计：4 + 6 = 10级
- 扣除：庚金克乙木 = 2级
- **实际级数：10 - 2 = 8级（正厅级）**

**案例2：**
> "此命财富以壬水看，月令长生四级，扣除年干戊土克去一级余三级"

**计算：**
- 壬水月令长生 = 4级
- 扣除：年干戊土克去 = 1级
- **实际级数：4 - 1 = 3级（千万富婆）**

## 💡 第七部分：分与级的区别

### 基本定义

**崔老师原话：**
> "喜神有刑冲破坏混杂以分计算，没有刑冲破坏混杂者以级计算，暗拱暗会也可加入计算，**一级是十分**。"

**理解：**
- **级** = 10分 = 满分
- **分** = 不足10分

**转换关系：**
- 10分 = 1级
- 5分 = 0.5级
- 8分 = 0.8级
- 15分 = 1.5级

## 🎯 第八部分：完整计算流程

### 步骤1：定格取用
- 确定格神和用神

### 步骤2：计算原局级数
- 按十二长生计算格神和用神的级数
- 按坐支计算级数

### 步骤3：计算暗拱暗会级数
- 查找可以拱到的库
- 库的级数 = 6级

### 步骤4：计算扣除级数
- 找出所有克制、混杂、刑冲
- 计算需要扣除的级数

### 步骤5：计算降级
- 检查是否有降半级的条件
- 累加所有降级

### 步骤6：得出实际级数
```
实际级数 = 原局级数 + 暗拱暗会级数 - 扣除级数 - 降级级数
```
