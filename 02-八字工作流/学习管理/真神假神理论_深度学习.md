## 第一部分：真神假神的核心概念

### 🎯 什么是真神假神？

**定义：**

**真神：**
- 格局中真正的核心力量
- 被保留、清纯、发挥作用
- "真神得用" = 大富大贵

**假神：**
- 格局中需要被去掉的部分
- 混杂、不清、需要被制化
- "假神去" = 格局清纯

---

### 1.1 真神假神的本质

**崔老师的核心理念：**

> "真神得用，假神去，格局清纯"

**理解：**
- 真神 = 格局的真正主人
- 假神 = 格局的干扰因素
- 去假留真 = 格局清纯
- 格局清纯 = 富贵

---

### 1.2 真神假神的作用

**真神的作用：**
1. **定格**：作为格局的核心
2. **发用**：发挥主导作用
3. **得用**：得到应用和发挥
4. **富贵**：真神得用 = 大富大贵

**假神的作用：**
1. **混杂**：干扰格局清纯
2. **破格**：破坏格局层次
3. **隐患**：随时可能破格
4. **必须去掉**：去假才能存真

---

## 第二部分：真神假神的判断标准

### 2.1 真神的判断标准

**标准1：清纯**
- 无混杂
- 无刑冲克害
- 清晰明确

**标准2：有力**
- 在月令有力
- 得生扶
- 有根有源

**标准3：得用**
- 能发挥作用
- 能主导格局
- 能带来富贵

---

### 2.2 假神的判断标准

**标准1：混杂**
- 与真神混杂
- 正偏混杂
- 阴阳混杂

**标准2：无力**
- 在月令无力
- 被克泄
- 无根无源

**标准3：无用**
- 不能发挥作用
- 干扰格局
- 必须去掉

---

### 2.3 真神假神的判断流程

#### 第一步：识别格局中的混杂

**常见的混杂：**

1. **官杀混杂**
   - 正官 + 七杀
   - 例：壬水正官 + 癸水七杀

2. **财星混杂**
   - 正财 + 偏财
   - 例：戊土正财 + 己土偏财

3. **印星混杂**
   - 正印 + 偏印
   - 例：庚金正印 + 辛金偏印

---

#### 第二步：判断哪个是真神

**判断标准：**

**1. 看月令**
```
在月令有力 → 真神
在月令无力 → 假神
```

**2. 看清纯度**
```
无混杂 → 真神
有混杂 → 假神
```

**3. 看能否得用**
```
能发挥主导作用 → 真神
干扰格局 → 假神
```

---

#### 第三步：去假存真

**去假神的方法：**

**1. 克**
```
假神是什么？
用克假神的五行
例：假神是壬水，用戊土克
```

**2. 泄**
```
假神是什么？
用泄假神的五行
例：假神是壬水，用木泄
```

**3. 合**
```
假神是什么？
用合去假神
例：假神是庚金，用乙合
```

**4. 刑冲**
```
假神在哪里？
用刑冲去掉
例：假神在子，用午冲
```

## 第三部分：真神假神的实战案例

### 案例1：毛泽东（七杀格真神得用）

**八字：**
```
癸 甲 丁 甲
巳 子 酉 辰
```

**分析：**

**第一步：识别混杂**
- 年干：癸水（七杀）
- 地支子辰会局：壬水（正官）
- **官杀混杂**

**第二步：判断真神假神**
- 癸水七杀：月令建禄（子月），又有酉中辛金之生
- 壬水正官：会局之水，混杂

**判断：**
- 癸水七杀 = **真神**
- 壬水正官 = **假神**

**第三步：去假存真**
- 月时干双透甲木正印
- 甲木化泄壬水（假神）
- "泄得干干净净"

**结果：**
- 壬水假神被去掉
- 癸水真神清纯当令
- **真神得用**
- 十二级富贵（帝王之命）

---

### 案例2：岳飞（假神当真，凶）

**八字：**
```
癸 卯 庚 甲
巳 亥 酉 戌
```

**分析：**

**第一步：识别混杂**
- 日支酉中庚金（格神）
- 但原局可能有其他混杂

**第二步：判断真神假神**
- 庚金格神：虽不旺，但无辛金之混
- 可寻暗拱暗会为**真神**
- 但地支可能有甲木假神

**第三步：假神当真**
- 亥运：乙木格神死地
- 甲木假神长生
- **假神当真**
- 凶

**结果：**
- 假神（甲木）长生
- 真神（乙木）死地
- 假神当真
- 凶机暗藏

---

### 案例3：正官格去假留真

**八字：**
```
丁 丙 乙 辛
丑 午 未 巳
```

**分析：**

**第一步：识别混杂**
- 地支巳午未会丙丁火方
- 年时巳丑会庚金正官局
- 时干透辛金七杀
- **官杀混杂**

**第二步：判断真神假神**
- 庚金正官：因会局而有力，归时逢长生
- 辛金七杀：坐死地，又有午中丁火之制

**判断：**
- 庚金正官 = **真神**
- 辛金七杀 = **假神**

**第三步：去假存真**
- 丁火克尽辛金（食神制杀）
- 以庚金正官为格
- 丁火食神制杀为用

**结果：**
- 辛金假神被去掉
- 庚金真神清纯
- 去留得宜
- 十级富贵（大富大贵）

## 第四部分：真神假神的特殊情况

### 4.1 假神当真

**定义：**
假神力量超过真神，假神主导格局

**后果：**
- 真神不能发挥作用
- 假神当道
- 凶

**案例：岳飞**
- 亥运：甲木假神长生
- 乙木真神死地
- 假神当真
- 凶

---

### 4.2 真神变假

**定义：**
原是真神，但大运流年变化，变成假神

**原因：**
- 大运流年扶持假神
- 真神被克泄
- 真假易位

**案例：毛泽东76年**
- 76年走辰运辰年
- 壬水假神闭库不被制化
- 真假神阴阳混杂征战
- 生命到此结束

---

### 4.3 寻暗拱暗会为真神

**定义：**
原局某五行不旺，但可以通过暗拱暗会加强，成为真神

**条件：**
- 无混杂
- 可以暗拱暗会
- 暗拱暗会后有力

**案例：岳飞**
- 庚金虽然局中不旺
- 喜无辛金之混
- 所以可寻暗拱暗会为**真神**
- 子巳紧贴可以暗会丑库庚金
- 为大贵之基因

## 第五部分：真神假神与格局层次

### 5.1 真神得用的层次

**最高层次：真神得用 + 假神全去**
- 例：毛泽东
- 癸水七杀清纯当令（真神得用）
- 壬水正官被甲木化泄（假神全去）
- 十二级富贵（帝王之命）

**次高层次：真神得用 + 假神部分去**
- 例：董建华
- 合去合留，格局清纯
- 十级富贵（大富大贵）

**中层次：真神得用 + 假神有制**
- 假神有制化但不完全去
- 中小富贵

---

### 5.2 假神不去的层次

**假神不去，格局不清**
- 富贵层次降低
- 甚至贫贱

**假神当真**
- 凶
- 可能有大灾

## 第七部分：常见问题解答

### 问题1：真神一定在月令吗？

**答：不一定！**

- 通常真神在月令有力
- 但也可以通过暗拱暗会成为真神
- 关键是：清纯 + 有力 + 得用

---

### 问题2：假神可以不去吗？

**答：不能！**

- 假神不去 = 格局不清
- 格局不清 = 降格
- 必须去假存真

---

### 问题3：如何快速判断真神假神？

**答：三步法**

```
第一步：看月令
  月令有力的 → 真神
  月令无力的 → 假神

第二步：看清纯
  无混杂的 → 真神
  有混杂的 → 假神

第三步：看得用
  能主导格局的 → 真神
  干扰格局的 → 假神
```

---

### 问题4：真神变假怎么办？

**答：这是大运流年的变化**

- 原局真神不变
- 但大运流年可能扶持假神
- 真假易位
- 凶
- 注意大运流年的判断

---

### 问题5：假神能完全去掉吗？

**答：不一定！**

- 有制化之神 = 能去
- 无制化之神 = 不能去
- 去得越净 = 格局越高
- 去不净 = 降格
