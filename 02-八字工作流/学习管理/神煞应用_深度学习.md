## 第一部分：神煞的核心概念

### 🎯 什么是神煞？

**定义：**
神煞是八字中的特殊星曜，用于辅助论命

**崔老师观点：**
> "正宗子平格局不太重视神煞，以格局为主"
> "神煞只能作为辅助参考"

**核心原则：**
```
格局为主（70%） + 神煞为辅（30%） = 完整论命
```

---

### 1.1 神煞的分类

**吉神：**
- 天乙贵人
- 月德
- 文昌
- 食神（吉神意义）

**凶煞：**
- 桃花
- 羊刃
- 亡神
- 劫煞
- 孤寡
- 隔角
- 阴差阳错
- 孤鸾

---

### 1.2 神煞的作用

**辅助作用：**
1. **验证格局**：验证格局判断的准确性
2. **补充细节**：补充格局无法描述的细节
3. **预测流年**：大运流年的吉凶参考
4. **性格分析**：补充性格特征

**不能做的事：**
- ❌ 不能以神煞否定格局
- ❌ 不能以神煞为主论命
- ❌ 不能本末倒置

## 第二部分：常用吉神详解

### 2.1 天乙贵人（最重要）

**查法：**
```
以日干或年干查

甲戊庚 → 丑、未
乙己 → 子、申
丙丁 → 酉、亥
壬癸 → 卯、巳
辛 → 寅、午
```

**作用：**
- 解危救难
- 遇事有人帮
- 提升命局层次
- 化凶为吉

**判断：**
- 原局有贵人 → 吉
- 大运见贵人 → 吉
- 流年见贵人 → 吉

**示例：**
```
日干为甲，地支有丑或未
→ 为天乙贵人
→ 遇事有人帮
```

---

### 2.2 月德

**查法：**
```
以月支查

寅午戌月 → 丙
申子辰月 → 壬
亥卯未月 → 甲
巳酉丑月 → 庚
```

**作用：**
- 化险为夷
- 吉祥如意
- 善德积福
- 遇凶不凶

**判断：**
- 月干有月德 → 吉
- 月支生月德 → 吉

---

### 2.3 文昌

**查法：**
```
以年干或日干查

甲乙 → 巳
丙丁 → 申
戊己 → 申
庚辛 → 寅
壬癸 → 亥
```

**作用：**
- 文笔好
- 聪明
- 利考试
- 学业有成

**判断：**
- 地支见文昌 → 聪明好学
- 文昌得用 → 考试顺利

**示例：**
```
日干为甲，地支见巳
→ 为文昌
→ 聪明利考
```

---

### 2.4 食神（吉神意义）

**查法：**
```
日干生的同性天干
如：甲日干见丙火
```

**作用：**
- 有福气
- 有口福
- 才华横溢
- 性格温和

**判断：**
- 食神透干 → 有福气
- 食神得用 → 才华好

## 第三部分：常用凶煞详解

### 3.1 桃花

**查法：**
```
以年支或日支查

寅午戌 → 见卯（卯桃花）
申子辰 → 见酉（酉桃花）
巳酉丑 → 见午（午桃花）
亥卯未 → 见子（子桃花）
```

**种类：**
- 墙外桃花（年支见）：外缘
- 墙内桃花（月支见）：内缘
- 滚壁桃花（日支见）：自己

**作用：**
- 男命：异性缘好
- 女命：魅力大
- 过旺则淫乱
- 适中则人缘好

**判断：**
- 桃花1个 → 人缘好
- 桃花2个 → 异性缘旺
- 桃花3个 → 可能淫乱
- 桃花 + 合 → 更旺

**示例：**
```
年支为午，地支见卯
→ 为桃花
→ 异性缘好
```

---

### 3.2 羊刃

**查法：**
```
阳干见帝支（帝旺之支）

甲 → 卯
丙 → 午
戊 → 午
庚 → 酉
壬 → 子
```

**作用：**
- 性格刚强
- 容易冲动
- 旺则为权
- 衰则伤身

**判断：**
- 羊刃有制 → 权威
- 羊刃无制 → 刚强伤身

**示例：**
```
日干为甲，地支见卯
→ 为羊刃
→ 性格刚强
```

---

### 3.3 亡神

**查法：**
```
以年支或日支查

寅午戌 → 见亥
巳酉丑 → 见寅
申子辰 → 见巳
亥卯未 → 见申
```

**作用：**
- 性格孤僻
- 容易失败
- 有才华但不得志
- 心理压力大

**判断：**
- 亡神有制 → 减轻
- 亡神无制 → 明显

---

### 3.4 劫煞

**查法：**
```
以年支或日支查

寅午戌 → 见亥
巳酉丑 → 见寅
申子辰 → 见巳
亥卯未 → 见申
```

**作用：**
- 破财
- 意外灾祸
- 注意安全
- 容易被骗

**判断：**
- 劫煞有制 → 减轻
- 劫煞无制 → 明显
- 劫煞 + 桃花 → 破财因色

---

### 3.5 其他凶煞

## 第四部分：神煞的判断原则

### 4.1 崔老师的核心观点

**原则：**
> "正宗子平格局以定格取用为主，神煞为辅"
> "不能以神煞论命，要以格局为主"

**判断顺序：**
```
1. 定格取用（最重要，70%）
2. 净阴净阳（20%）
3. 神煞辅助（10%）
```

---

### 4.2 神煞判断的三大原则

**原则1：不要本末倒置**
- 格局为主，神煞为辅
- 不能用神煞否定格局
- 不能以神煞为主论命

**原则2：看原局是否有救**
- 有吉神，但被克制 = 无用
- 有凶煞，但有制化 = 无碍

**原则3：大运流年配合**
- 吉神运 = 发挥作用
- 凶煞运 = 产生问题

---

### 4.3 神煞的轻重判断

**重的标准：**
- 透干（天干出现）
- 得月令（在月令有力）
- 无制化（没有被制化）

**轻的标准：**
- 藏支（只在藏干中）
- 失月令（不在月令）
- 有制化（被制化）

**判断：**
```
重 → 影响大
轻 → 影响小
```

## 第六部分：神煞实战案例

### 案例1：天乙贵人

**八字：**
```
戊 辛 甲 甲
申 酉 戍 戍
```

**分析：**
- 日干：甲
- 天乙贵人：丑、未
- 原局：无丑未
- → **无天乙贵人**

**作用：**
- 无贵人帮助
- 遇事靠自己

---

### 案例2：桃花

**八字：**
```
男：1990年3月15日 午时
```

**排盘：**
```
年：庚 午
月：己 卯
日：甲 戌
时：甲 午
```

**分析：**
- 年支：午
- 桃花：卯（寅午戌见卯）
- 月支有卯
- → **有墙内桃花**

**作用：**
- 异性缘好
- 但需注意桃花劫

---

### 案例3：羊刃

**八字：**
```
庚 甲 壬 丙
寅 子 寅 午
```

**分析：**
- 日干：壬
- 羊刃：子
- 日支：子
- → **日支坐羊刃**

**作用：**
- 性格刚强
- 容易冲动
- 有制化则为权
