## 📚 阶段1：经典案例深度分析

### 学习目标：
通过分析毛泽东、和珅、岳飞等名人案例，巩固核心理论

### 学习内容：

**案例1：毛泽东 - 七杀格真神得用（12级）**
- 八字：癸巳 甲子 丁酉 甲辰
- 核心：癸水七杀清纯，甲木化泄正官
- 学习：七杀格的最高境界

**案例2：和珅 - 正官格儿能生母（11级）**
- 八字：庚午 乙酉 庚子 壬午
- 核心：乙木长生，儿能生母
- 学习：阴阳长生的特殊用法

**案例3：岳飞 - 劫财格暗拱暗会（10级）**
- 八字：癸未 乙卯 甲子 己巳
- 核心：子巳暗会丑库庚金
- 学习：暗拱暗会的查找

**案例4：慈禧 - 劫财格去假留真（12级）**
- 八字：乙未 丁亥 乙丑 丁亥
- 核心：丁火泄乙木，甲木清纯
- 学习：去假留真的方法

**案例5：李鸿章 - 比肩格甲己合去乙**
- 八字：癸未 甲寅 乙亥 己卯
- 核心：甲己合，甲木去
- 学习：假神代真神

### 学习重点：
1. ✅ 如何识别七杀格、劫财格、比肩格
2. ✅ 如何理解"真神得用"
3. ✅ 如何理解"去假留真"
4. ✅ 如何理解"暗拱暗会"
5. ✅ 如何理解"假神代真神"

### 学习方法：
1. 先阅读崔老师原文分析
2. 总结核心理论点
3. 对比自己的理解
4. 创建学习文档

## 💰 阶段3：级数计算方法

### 学习目标：
掌握富贵级数的精确计算

### 学习内容：

**原局级数计算：**
- 十二长生级数表
- 如何查表
- 如何确定级数

**暗拱暗会级数：**
- 如何判断能暗拱
- 如何计算级数
- 所有库都是6级

**扣除级数：**
- 忌神如何扣除
- 扣除多少级
- 具体案例

**降级处理：**
- 何时降半级
- 如何降级
- 累计降级

**实际级数公式：**
```
实际级数 = 原局级数 + 暗拱暗会级数 - 扣除级数 - 降级级数
```

### 学习重点：
1. ✅ 级数表的记忆
2. ✅ 计算公式的应用
3. ✅ 扣除和降级的判断
4. ✅ 案例计算练习

### 学习方法：
1. 记忆十二长生表
2. 计算5个案例的级数
3. 对比崔老师的计算
4. 总结规律
