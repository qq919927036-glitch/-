## 案例对比分析

### 案例1：万里浪（官杀混杂+假神当真）

**八字：**
```
壬申 癸酉 甲戌 乙丑
```

**基本信息：**
- 性别：男命
- 寿命：38岁（被枪决）

**格局分析：**
- **月令**：酉月（辛金司令），庚辛金禄旺之地
- **地支关系**：申酉戌三会金局
- **天干**：壬癸水混杂，甲乙木混杂

**核心问题：**
1. **官杀混杂**：
   - 辛金正官：15.0级
   - 庚金七杀：17.0级
   - 正官七杀同时透干，无法去留

2. **假神当真**：
   - 申酉戌三会金局 → 假神庚金大旺
   - 真神辛金反而被混杂
   - 假神力量 > 真神力量

3. **日主极弱**：
   - 甲木在申月 = 绝地（0级）
   - 甲木在酉月 = 胎地（1级）
   - 甲木在戌月 = 养地（3级）
   - 甲木在丑月 = 衰地（2级）
   - 日主几乎无根

4. **无印比生扶**：
   - 壬癸水（印星）虽透干，但泄金太重
   - 乙木（比劫）坐丑土，无力
   - 印星救不了日主

**结局：** 38岁被枪决

## 凶命核心特征

### 特征1：官杀混杂（⭐⭐⭐⭐⭐ 最危险）

**定义：**
- 正官和七杀同时透干
- 无法去假留真
- 格局成败未定

**判断标准：**
```python
if 正官透干 and 七杀透干:
    if 无法去留:
        return "官杀混杂，凶险万分"
```

**危险性：**
- 官杀混杂 = 管制混乱
- 性格矛盾，是非不分
- 易走极端，触犯法律
- 易有牢狱之灾

**案例对比：**
- ✅ **富贵命**：慈禧太后（官杀混杂，但有伤官去杀留官）
- ❌ **凶命**：万里浪（官杀混杂，无去留）

---

### 特征2：真衰假旺（⭐⭐⭐⭐⭐ 最危险）

**定义：**
- 格神在月令极弱（0-2级）
- 假神反而大旺（3-6级）
- 假神力量 > 真神力量

**判断标准：**
```python
if 格神级数 <= 2:
    if 假神级数 >= 3:
        if 假神级数 > 格神级数:
            return "真衰假旺，起落不宁"
```

**十二长生级数对应：**
- 帝旺、库 = 6级
- 长生 = 4级
- 临官、冠带、养 = 3级
- 沐浴、衰 = 2级
- 胎、病 = 1级
- 死、绝 = 0级

**危险性：**
- 格神起落不宁
- 忽冷忽热
- 做事不稳
- 易走极端

**案例对比：**
- ✅ **富贵命**：高际阳（丁火真衰，但假神不旺）
- ❌ **凶命**：录儿（癸水真衰，壬水假旺）

---

### 特征3：假神当真（⭐⭐⭐⭐⭐ 最危险）

**定义：**
- 假神力量压倒真神
- 假神占据主导地位
- 真神无法发挥作用

**判断标准：**
```python
if 假神级数 > 真神级数:
    if 假神级数 - 真神级数 >= 3:
        return "假神当真，凶机暗藏"
```

**危险性：**
- 假神当真 = 是非颠倒
- 认知错乱
- 判断失误
- 易犯大错

**案例对比：**
- ✅ **富贵命**：岳飞（庚金用神虽弱，但假神不当真）
- ❌ **凶命**：万里浪（庚金假神17级，辛金真神15级）

---

### 特征4：日主极弱（⭐⭐⭐⭐）

**定义：**
- 日主在月令状态 ≤ 1级
- 无印比生扶
- 日主几乎无根

**判断标准：**
```python
if 日主在月令级数 <= 1:
    if 无印星:
        if 无比劫:
            return "日主极弱，非从不可"
```

**危险性：**
- 日主无力承担
- 容易崩溃
- 难以承压
- 易有大灾

**案例对比：**
- ✅ **从格富贵**：高际阳（日主极弱，但从财成格）
- ❌ **凶命**：万里浪（日主极弱，但不从格）

---

## 识别规则（if-then格式）

### 规则1：官杀混杂识别

```python
if 正官透干 and 七杀透干:
    if 有去留之神:
        # 例如：伤官制杀，或食神合官
        return "格局可成，需看去留是否彻底"
    else:
        if 官杀级数差 <= 2:
            return "【凶险】官杀混杂且无去留，易有牢狱之灾"
        else:
            return "格局不清，起落不宁"
```

**案例验证：**
- ✅ 慈禧太后：官杀混杂，但有伤官去杀留官 → 贵
- ❌ 万里浪：官杀混杂，无去留 → 凶

---

### 规则2：真衰假旺识别

```python
# 计算格神级数
格神级数 = 0
for 地支 in 四柱地支:
    格神级数 += 十二长生级数[格神][地支]

# 计算假神级数
假神级数 = 0
for 地支 in 四柱地支:
    假神级数 += 十二长生级数[假神][地支]

if 格神级数 <= 2:
    if 假神级数 >= 3:
        if 假神级数 > 格神级数 + 2:
            return "【凶险】真衰假旺，起落不宁，易走极端"
        else:
            return "格神偏弱，需扶格神"
```

**案例验证：**
- ✅ 高际阳：丁火格神0级，假神丙火级数不高 → 可从
- ❌ 录儿：癸水格神1级，壬水假神7级 → 凶

---

### 规则3：假神当真识别

```python
if 假神级数 > 真神级数:
    级数差 = 假神级数 - 真神级数

    if 级数差 >= 3:
        return "【凶险】假神当真，是非颠倒，易犯大错"
    elif 级数差 >= 2:
        return "假神偏重，需谨慎"
    else:
        return "假神力量略强，影响不大"
```

**案例验证：**
- ✅ 岳飞：庚金用神虽弱，但假神不当真 → 贵
- ❌ 万里浪：庚金假神17级，辛金真神15级，级数差2 → 凶

### 规则7：凶命综合判断

```python
def judge_dangerous_chart(chart_data):
    """综合判断是否为凶命"""

    凶险特征数 = 0
    凶险特征列表 = []

    # 1. 官杀混杂
    if is_official_and_killing_mixed(chart_data):
        if not has_removal_method(chart_data):
            凶险特征数 += 2
            凶险特征列表.append("官杀混杂无去留")

    # 2. 真衰假旺
    真假神分析 = analyze_true_fake(chart_data)
    if 真假神分析['真神级数'] <= 2 and 真假神分析['假神级数'] >= 3:
        if 真假神分析['假神级数'] > 真假神分析['真神级数'] + 2:
            凶险特征数 += 2
            凶险特征列表.append("真衰假旺")

    # 3. 假神当真
    if 真假神分析['假神级数'] > 真假神分析['真神级数']:
        if 真假神分析['假神级数'] - 真假神分析['真神级数'] >= 3:
            凶险特征数 += 2
            凶险特征列表.append("假神当真")

    # 4. 日主极弱又不从
    if is_self_extremely_weak(chart_data):
        if not is_follow_pattern(chart_data):
            凶险特征数 += 3
            凶险特征列表.append("日主极弱又不从")

    # 5. 阴阳战斗
    if has_yin_yang_fight(chart_data):
        凶险特征数 += 1
        凶险特征列表.append("阴阳战斗")

    # 6. 三会成局克真神
    if has_triple_meeting(chart_data):
        if triple_meeting_attacks_true_god(chart_data):
            凶险特征数 += 2
            凶险特征列表.append("三会成局克真神")

    # 7. 无制化之神
    if has_no_restraint(chart_data):
        凶险特征数 += 1
        凶险特征列表.append("忌神无制")

    # 综合判断
    if 凶险特征数 >= 8:
        return {
            '判断': '【极凶】大凶之命，易有牢狱之灾或生死大灾',
            '凶险特征': 凶险特征列表,
            '凶险分数': 凶险特征数
        }
    elif 凶险特征数 >= 5:
        return {
            '判断': '【凶险】多凶少吉，需格外谨慎',
            '凶险特征': 凶险特征列表,
            '凶险分数': 凶险特征数
        }
    elif 凶险特征数 >= 3:
        return {
            '判断': '【风险】格局不稳，易走极端',
            '凶险特征': 凶险特征列表,
            '凶险分数': 凶险特征数
        }
    else:
        return {
            '判断': '【平稳】格局尚可',
            '凶险特征': 凶险特征列表,
            '凶险分数': 凶险特征数
        }
```

|--------|--------|------|
| **特征** | 官杀混杂，但有去留 | 官杀混杂，无去留 |
| **去留方式** | 伤官制杀、食神合官、印星化杀 | 无 |
| **典型案例** | 慈禧太后（去杀留官） | 万里浪（无去留） |
| **结局** | 大富大贵 | 牢狱之灾/死刑 |
| **判断要点** | 有去留之神 | 无去留之神 |

| 对比项 | 富贵命 | 凶命 |
|--------|--------|------|
| **特征** | 格神偏弱，但假神不旺 | 格神真衰，假神假旺 |
| **格神级数** | 0-2级 | 0-2级 |
| **假神级数** | 0-3级 | ≥ 4级 |
| **级数差** | ≤ 2级 | ≥ 3级 |
| **典型案例** | 高际阳（假神不旺） | 录儿（假神大旺） |
| **结局** | 从格成，富贵 | 走极端，犯罪 |

### 凶命判断标准

**极凶（≥8分）：**
- 易有牢狱之灾
- 易有生死大灾
- 需格外警惕

**凶险（5-7分）：**
- 多凶少吉
- 需谨慎行事
- 易走极端

**风险（3-4分）：**
- 格局不稳
- 需注意防范
- 避免走极端

**平稳（0-2分）：**
- 格局尚可
- 相对平安
