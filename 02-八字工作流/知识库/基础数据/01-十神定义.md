## 核心概念

**十神**：以日干为中心，与其他天干地支的生克关系。

```
日干 = 我（日主）
其他天干/地支藏干 = 与我的关系
```

## 十神对应表

**定义：** 克我者，且阴阳同性

**公式：**
```
阳干克阳干 / 阴干克阴干
```

**代表：** 名誉、地位、权力、官职、丈夫（女命）

**特点：**
- 约束自我
- 正当的规范

## 快速记忆法

### 方法1：生克关系

```
我克：正财（异性）、偏财（同性）
克我：正官（异性）、七杀（同性）
我生：食神（同性）、伤官（异性）
生我：正印（异性）、偏印（同性）
同我：比肩（同性）、劫财（异性）
```

### 方法2：阴阳判断

```
同性 = 偏、七杀、食神、比肩、偏印
异性 = 正、正官、正财、正印、伤官、劫财
```
