## 一、三合局

### 定义
相隔三个地支形成的合局，力量最强

### 三合局表

| 三合局 | 地支 | 合化五行 |
|--------|------|----------|
| **申子辰** | 申-子-辰 | 水局 |
| **亥卯未** | 亥-卯-未 | 木局 |
| **寅午戌** | 寅-午-戌 | 火局 |
| **巳酉丑** | 巳-酉-丑 | 金局 |

### 特点
- 三支全见 → 合化成功
- 只见两支 → 半三合（仍有合意）
- 方局力量最大
- 化气后以化气五行论

### 示例
```
申 + 子 + 辰 = 水局
亥 + 卯 + 未 = 木局
寅 + 午 + 戌 = 火局
巳 + 酉 + 丑 = 金局
```

| **寅亥合** | 木 |
| **卯戌合** | 火 |
| **辰酉合** | 金 |
| **巳申合** | 水 |

### 特点

- 合化以化气五行为主
- 有合留、合去之分

## 实战应用

### 例1：万里浪命例
```
地支：申 酉 戍 戌

分析：
1. 申+酉+戌 = 西方金局（三会）
2. 庚辛金都强旺
3. 官杀混杂
```

### 例2：董建华命例
```
地支：丑 午 未 巳

分析：
1. 巳+午+未 = 南方火局（三会）
2. 丙丁火同旺
3. 巳丑 = 金局（半三合）
```

### 例3：连战命例
```
地支：子 申 已 戌

分析：
1. 申+子 = 水局（半三合）
2. 巳+申 = 合（巳申合水）
3. 巳+戌 = 火局（半三合，暗拱）
```
