## 核心概念

**十二长生**：五行在十二个地支中的生旺死绝状态

**重要口诀：**
```
长生四子中旬半，沐浴一双保吉祥。
冠带临官三子位，旺中五子自成行。
衰中二子病中一，死中至老没儿郎。
绝中取养他人子，入墓之时命夭亡。
胎中头女有姑娘，养中三子只留一。
男女宫中仔细详。
```

## 实战应用

### 例1：万里浪命例
```
月干：辛金（阴金）
月支：酉

查表：辛金在酉 = 临官（3级）
→ "月支坐禄、官星旺相可用"
```

### 例2：甲子运分析
```
辛金（阴金）在子：
查表：辛金在子 = 长生（4级）
→ "辛金到子逢长生"（真神旺）

庚金（阳金）在子：
查表：庚金在子 = 死（0级）
→ "庚金到子为死"（假神死）
```

### 例3：慈禧命例
```
甲木在亥：
查表：甲木在亥 = 长生（4级）
→ "甲木在亥月逢长生"

甲木在亥时：
查表：甲木在亥 = 长生（4级）
→ "甲木在亥时逢长生"

壬水在亥月：
查表：壬水在亥 = 临官（3级）
→ "壬水月时再居禄位"
```

## 特殊规则

### 库地规则
```
戌、丑、辰、未为库地
- 有混杂者 → 无分
- 有克泄者 → 宜闭库，为级
- 无克泄者 → 宜开库
```

### 绝处逢生
```
亥月本为绝（0级）
但若地支得寅合留甲木
→ 甲木生丙火
→ 以级计算（不为0级）
```
