# 八字排盘方法

> 学习时间：2026-02-18
> 重要程度：⭐⭐⭐⭐⭐

---

## 核心概念

**八字** = 年柱 + 月柱 + 日柱 + 时柱

每柱 = 天干 + 地支

```
年：天干 地支
月：天干 地支
日：天干 地支
时：天干 地支
```

---

## 第一步：排年柱

### 方法
根据出生年份的干支

### 年干支表（六十年甲子）

```
甲子  乙丑  丙寅  丁卯  戊辰  己巳  庚午  辛未  壬申  癸酉
甲戌  乙亥  丙子  丁丑  戊寅  己卯  庚辰  辛巳  壬午  癸未
甲申  乙酉  丙戌  丁亥  戊子  己丑  庚寅  辛卯  壬辰  癸巳
甲午  乙未  丙申  丁酉  戊戌  己亥  庚子  辛丑  壬寅  癸卯
甲辰  乙巳  丙午  丁未  戊申  己酉  庚戌  辛亥  壬子  癸丑
甲寅  乙卯  丙辰  丁巳  戊午  己未  庚申  辛酉  壬戌  癸亥
```

### 快速查年干

天干每10年循环一次：
```
年份尾数：
0 → 庚
1 → 辛
2 → 壬
3 → 癸
4 → 甲
5 → 乙
6 → 丙
7 → 丁
8 → 戊
9 → 己
```

**示例：**
```
1970年 → 尾数0 → 庚戌
1990年 → 尾数0 → 庚午
2024年 → 尾数4 → 甲辰
```

| 十一月 | 子 | 大雪 |
| 十二月 | 丑 | 小寒 |

根据年干查月干：

**年干为乙/庚：**
```
正月起戊寅
```

**年干为丙/辛：**
```

```

### 示例

**例1：1970年二月**
```
年干：庚（乙庚起戊寅）
二月：戊寅 → 己卯
月柱：己卯
```

**例2：1990年三月**
```
年干：庚（乙庚起戊寅）
三月：戊寅 → 己卯 → 庚辰
月柱：庚辰
```

## 完整示例

### 例1：万里浪命例

**出生信息：**
```
男：1908年8月21日
时辰：壬时
```

**排盘过程：**

1. **年柱**
```
1908年 → 戊申
（年份尾数8 → 戊）
```

2. **月柱**
```
年干戊（戊癸起甲寅）
八月：甲寅 → 乙卯 → 丙辰 → 丁巳 → 戊午 → 己未 → 庚申
月柱：庚申
```

3. **日柱**
```
查万年历：甲戌日
```

4. **时柱**
```
日干甲（甲己起甲子）
壬时：甲子 → 乙丑 → 丙寅 → 丁卯 → 戊辰 → 己巳 → 庚午 → 辛未 → 壬申 → 癸酉 → 甲戌 → 乙亥
时柱：壬亥
```

**完整八字：**
```
戊 庚 甲 壬
申 申 戌 亥
```

## 练习题

### 题1：1985年三月出生，年月柱是什么？
```
年柱：乙丑（1985尾数5 → 乙）
月柱：己卯（乙庚起戊寅，三月 → 戊寅→己卯）
```

### 题2：甲日出生的午时，时柱是什么？
```
时柱：庚午（甲己起甲子，午时 → 甲子→...→庚午）
```

### 题3：1970年二月卯时，年月时柱是什么？
```
年柱：庚戌
月柱：己卯
时柱：丁卯
```

## 注意事项

1. **节气定月**
   - 月柱以节气为准，不是农历
   - 如：立春前仍是上一年

2. **真太阳时**
   - 需要根据经度调整时间
   - 北京时间需调整

3. **日柱计算复杂**
   - 建议查万年历
   - 不要手工计算
