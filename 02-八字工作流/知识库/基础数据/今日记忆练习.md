```
- 双藏：午、亥、寅、申（每个藏2个）
- 三藏：其余8个（每个藏3个）
```

```

辰水库，戌火库
丑金库，未木库

六冲：
子午冲，丑未冲，寅申冲
```

```
甲木在亥长生 → 冬天树木蓄势待发
乙木在午长生 → 夏天花草茂盛
癸水在卯长生 → 春天雨水滋润
```

```
甲亥、丙寅、戊寅、庚巳、壬申
```
