## 快速记忆法

### 生肖歌诀
```
子鼠 丑牛 寅虎 卯兔
辰龙 巳蛇 午马 未羊
申猴 酉鸡 戌狗 亥猪
```

### 五行分组
```
水局：子（阳水）、亥（阴水）
木局：寅（阳木）、卯（阴木）
火局：巳（阴火）、午（阳火）
金局：申（阳金）、酉（阴金）
土局：辰（水库）、戌（火库）、丑（金库）、未（木库）
```

### 时间对应
```
子（23-1点）    丑（1-3点）     寅（3-5点）
卯（5-7点）     辰（7-9点）     巳（9-11点）
午（11-13点）   未（13-15点）   申（15-17点）
酉（17-19点）   戌（19-21点）   亥（21-23点）
```

### 方位对应
```
子（正北）     午（正南）
卯（正东）     酉（正西）
寅（东北）     申（西南）
辰（东南）     戌（西北）
巳（东南偏南） 亥（西北偏北）
```
