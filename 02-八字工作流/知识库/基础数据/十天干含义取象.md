## 一、甲木（阳木）

### 基本属性
- **五行**：木
- **阴阳**：阳
- **季节**：春天（寅月）
- **颜色**：青色、绿色
- **方位**：东方

### 含义
- **本义**：草木之芽，破土而出
- **象征**：生机、开始、正直
- **特点**：向上生长、宁折不弯

### 取象

**自然界：**
- 大树、栋梁
- 森林、木材
- 竹子、松柏

**人物：**
- 首领、统帅
- 正直的人、有原则的人
- 父亲（男命）
- 丈夫（女命，正官格）

**身体：**
- 胆（中医）
- 头、大脑
- 神经系统
- 脊椎

**物品：**
- 家具、木器
- 纸张、书籍
- 木材、建材

**职业：**
- 教师、公务员
- 军人、警察
- 木材商人
- 园林工作者

### 性格
- 优点：正直、刚毅、有原则
- 缺点：固执、不懂变通

```
土：戊（高山） 己（田园）
金：庚（铁器） 辛（珠宝）
水：壬（江河） 癸（雨露）
```

### 阴阳分组
```
阳干：甲丙戊庚壬（刚强）
阴干：乙丁己辛癸（柔弱）
```

```
东方木：甲乙（春）
南方火：丙丁（夏）
中央土：戊己（四季末）
西方金：庚辛（秋）
北方水：壬癸（冬）
```
