## 核心概念

**藏干**：地支中藏有的天干

每个地支藏有1-3个天干：
- **本气**（主气）：力量最大，决定地支主要性质
- **中气**（余气）：力量次之
- **余气**：力量最小

## 详细解析

### 一、子（只藏癸水）

```
子 = 癸水（本气，100%）
```

**特点：**
- 纯正，只藏一个天干
- 癸水为阴水
- 代表水的本性

**应用：**
- 子水当令时，以癸水为主
- 分析时只看癸水

---

### 二、丑（藏己、辛、癸）

```
丑 = 己土（本气，50%） + 辛金（中气，25%） + 癸水（余气，25%）
```

**力量分析：**
- 己土最强（本气）
- 辛金次之（墓库）
- 癸水最弱（余气）

**应用：**
- 丑月以己土为主
- 辛金可作为金根
- 癸水有水库之意

**问题：** 生于申月，月令本气是什么？

**分析：**
- 申月以庚金为主
- 壬水中气长生
- 戊土余气寄生
