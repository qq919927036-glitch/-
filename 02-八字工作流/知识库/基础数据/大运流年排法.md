## 一、大运排法

### 核心概念

**大运**：人一生中每十年的运势

**起运岁数：**
- 男阳年、女阴年 → 顺行
- 男阴年、女阳年 → 逆行

**起运年龄计算：**
- 从出生到下一个节气，除以3
- 得出的数字为起运岁数

---

### 大运排法步骤

#### 第一步：判断顺逆

**规则：**
```
阳男阴女：顺行
阴男阳女：逆行
```

**阳年阴年判断：**
```
甲、丙、戊、庚、壬 = 阳年
乙、丁、己、辛、癸 = 阴年
```

**示例：**
```
男，生于1990年（庚午）
→ 庚为阳干，阳男
→ 大运顺行

女，生于1990年（庚午）
→ 庚为阳干，阳女
→ 大运逆行
```

---

#### 第二步：排大运

**顺行排法（从月柱往后推）：**
```
月柱：己卯
大运1：庚辰
大运2：辛巳
大运3：壬午
大运4：癸未
...
```

**逆行排法（从月柱往前推）：**
```
月柱：己卯
大运1：戊寅
大运2：丁丑
大运3：丙子
大运4：乙亥
...
```

---

#### 第三步：大运数量

**一般排8步大运：**
```
大运1：1-10岁
大运2：11-20岁
大运3：21-30岁
大运4：31-40岁
大运5：41-50岁
大运6：51-60岁
大运7：61-70岁
大运8：71-80岁
```

## 二、流年排法

### 核心概念

**流年**：每年的干支

**排法：**
- 按照六十甲子循环
- 从出生年开始往后推

---

### 流年排法步骤

#### 第一步：确定出生年干支

**示例：**
```
1990年出生
→ 庚午年
```

#### 第二步：往后推算

**规则：**
```
每年天干地支各进一位
```

**示例：**
```
1990年：庚午（出生年）
1991年：辛未
1992年：壬申
1993年：癸酉
1994年：甲戌
1995年：乙亥
1996年：丙子
1997年：丁丑
1998年：戊寅
1999年：己卯
2000年：庚辰
...
```

## 三、大运流年合参

### 判断原则

**原局 + 大运 + 流年** → 综合判断

### 四种关系

#### 1. 冲

**定义：**
```
大运/流年与原局发生冲撞
```

**判断：**
- 冲去忌神 = 吉
- 冲去喜神 = 凶

**示例：**
```
原局以壬水为喜神
流年遇申（冲寅，不冲壬）
→ 不影响

流年遇午（冲子，子中有壬）
→ 冲去喜神，凶
```

---

#### 2. 战

**定义：**
```
真假神同旺无去留
```

**判断：**
- 真假神混战 = 凶
- 有去留 = 吉

**示例：**
```
原局：庚辛混杂
大运：庚旺、辛旺
→ 混战，凶

大运：庚旺、辛死
→ 有去留，吉
```

---

#### 3. 合（和）

**定义：**
```
大运/流年与原局发生相合
```

**判断：**
- 合留喜神 = 吉
- 合去喜神 = 凶
- 合去忌神 = 吉
- 合留忌神 = 凶

**示例：**
```
原局：丙辛合
→ 丙火合去，丁火独旺
→ 命局转清，吉
```

---

#### 4. 好

**定义：**
```
大运流年同入喜神生旺之地
```

**判断：**
- 同气为吉
- 喜神生旺 = 大吉

---

### 崔老师核心观点

> "原局格用神，与大运流年顺逆同道，有体用者吉利"
> "与岁运不和克害而无体用者不利"
