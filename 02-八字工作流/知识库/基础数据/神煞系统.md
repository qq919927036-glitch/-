## 核心概念

**神煞**：八字中的特殊星曜

**崔老师观点：**
> "正宗子平格局不太重视神煞，以格局为主"
> "神煞只能作为辅助参考"

## 常见吉神

### 一、天乙贵人

**查法：**
```
以日干或年干查

甲戊庚 → 丑、未
乙己 → 子、申
丙丁 → 酉、亥
壬癸 → 卯、巳
辛 → 寅、午
```

**作用：**
- 解危救难
- 遇事有人帮
- 提升命局层次

**示例：**
```
日干为甲，见地支有丑或未
→ 为天乙贵人
```

---

### 二、月德

**查法：**
```
以月支查

寅午戌月 → 丙
申子辰月 → 壬
亥卯未月 → 甲
巳酉丑月 → 庚
```

**作用：**
- 化险为夷
- 吉祥如意
- 善德积福

---

### 三、文昌

**查法：**
```
以年干或日干查

甲乙 → 巳
丙丁 → 申
戊己 → 申
庚辛 → 寅
壬癸 → 亥
```

**作用：**
- 文笔好
- 聪明
- 利考试

---

### 四、食神

**查法：**
```
日干生的同性天干
如：甲日干见丙火
```

**作用：**
- 有福气
- 有口福
- 才华横溢

## 常见凶煞

### 一、桃花

**查法：**
```
以年支或日支查

寅午戌 → 见卯（卯桃花）
申子辰 → 见酉（酉桃花）
巳酉丑 → 见午（午桃花）
亥卯未 → 见子（子桃花）
```

**种类：**
- 墙外桃花（年支见）
- 墙内桃花（月支见）
- 滚壁桃花（日支见）

**作用：**
- 男命：异性缘好
- 女命：魅力大
- 过旺则淫乱

---

### 二、羊刃

**查法：**
```
阳干见帝支

甲 → 卯
丙 → 午
戊 → 午
庚 → 酉
壬 → 子
```

**作用：**
- 性格刚强
- 容易冲动
- 旺则为权

---

### 三、亡神

**作用：**

### 四、劫煞

## 神煞的判断原则

### 崔老师观点

> "正宗子平格局以定格取用为主，神煞为辅"
> "不能以神煞论命，要以格局为主"

### 判断顺序

```
1. 定格取用（最重要）
2. 净阴净阳
3. 神煞辅助
```

### 注意事项

1. **不要本末倒置**
   - 格局为主，神煞为辅
   - 不能用神煞否定格局

2. **看原局是否有救**
   - 有吉神，但被克制 = 无用
   - 有凶煞，但有制化 = 无碍

3. **大运流年配合**
   - 吉神运 = 发挥作用
   - 凶煞运 = 产生问题

## 实战应用

### 例1：天乙贵人

**八字：**
```
戊 辛 甲 甲
申 酉 戍 戍
```

**分析：**
- 日干：甲
- 天乙贵人：丑、未
- 原局：无丑未
- → 无天乙贵人

---

### 例2：桃花

**八字：**
```
男：1990年3月15日 午时
```

**排盘：**
```
年：庚 午
月：己 卯
日：甲 戌
时：甲 午
```

**分析：**
- 年支：午
- 桃花：卯（寅午戌见卯）
- 月支有卯
- → 有墙内桃花
