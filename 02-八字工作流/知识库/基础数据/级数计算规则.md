## 级数计算公式推测

### 原局级数 = 原局状态级数 + 暗拱暗会级数

**示例：和珅**
```
原局级数：
- 午火月令（乙木长生）：4级
- 乙木年支得生：4级

暗拱级数：
- 子午冲去：3级

总计：4 + 4 + 3 = 11级
```

**示例：岳飞**
```
原局级数：
- 癸水真生：4级

暗拱级数：
- 子巳暗会庚金丑库：6级

总计：4 + 6 = 10级
```
