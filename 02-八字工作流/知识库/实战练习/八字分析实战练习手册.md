## 📋 实战练习10题

### 练习1：基础定格

**八字：**
```
男：1990年3月15日 午时
```

**任务：**
1. 排出四柱
2. 标注十神
3. 找出格神（最强最清纯的十神）
4. 找出用神（制化忌神或生扶格神的五行）

**答案：**
```
四柱：庚午 己卯 丙申 丙申
十神：比 食 官 杀 伤
格神：丙火（透干双透，有根基）
用神：庚金（制丙火）
```

---

### 练习2：净阴净阳

**八字：**
```
乙 丁 乙 丁
未 未 丑 亥
```

**任务：**
1. 找出混杂的十神
2. 判断是否去留取清
3. 命局层次（清/浊）

**答案：**
```
混杂：甲乙木混杂（需去留）
取清：丁火泄去乙木
层次：净阴净阳，格局清纯
```

---

### 练习3：暗拱暗会

**八字：**
```
癸 甲 丁 甲
巳 子 酉 辰
```

**任务：**
1. 找出所有三合局
2. 找出所有六合
3. 找出所有暗拱
4. 计算级数

**答案：**
```
三合：
- 巳酉丑（金局）
- 申子辰（水局）

六合：
- 寅亥合

暗拱：
- 拱未库（6级）
- 拱庚金库

级数：6+4+2=12级
```

### 练习6：六亲分析

**八字：**
```
癸 甲 丁 甲
巳 子 酉 辰
```

**任务：**
1. 父母星是谁？关系如何？
2. 兄弟星是谁？数量多少？
3. 妻子星是谁？婚姻如何？
4. 子女星是谁？数量多少？

**提示：**
- 父母：正印为母，偏财为父
- 兄弟：比劫为兄弟星
- 妻子：官杀为夫，财为妻
- 子女：食神伤官为子女

### 练习7：富贵等级判断

**计算你的财富等级：**

**步骤：**
1. 找到格神在哪些地支有长生、临官、帝旺、禄
2. 计算原局级数
3. 查找暗拱暗会
4. 减去刑冲克害
5. 得出实际级数

**对照表：**
```
1级 = 百万以上
2级 = 五百万以上
3级 = 千万以上
...
11级 = 全国首富
```

---

### 练习8：验证时辰

**任务：**
1. 算出你的兄弟姐妹数量
2. 与实际数量对比
3. 如果不符，说明时辰可能有误

**验证方法：**
> "要是命主的兄弟数量有差错，富贵等级也不相符时，
> 即应该将所报时辰上下推一推"

### 练习9：大运流年判断

**当前大运：** ___年___月___日到___年___月___日

**任务：**
1. 用"冲战合好"判断今年
2. 是冲、是战、是和、是好？
3. 具体是什么被冲/战/合/好？

**判断流程：**
```
是否冲喜神？ → 是 = 凶，否 = 吉
是否冲忌神？ → 是 = 吉，否 = 凶
真假神是否同旺？ → 是 = 战
是否合留喜神？ → 是 = 吉
是否同气为吉？ → 是 = 吉
```

## ⚠️ 常见错误

### 错误1：不以日干强弱论命
```
❌ "日干强还是弱？"
✅ "格神是什么？用神是什么？"
```

### 错误2：忽略净阴净阳
```
❌ 十神混杂也算了
✅ 必须去留取清
```

### 错误3：忘记查找暗拱暗会
```
❌ 只看明显的三合六合
✅ 系统扫描所有关系
```

### 错误4：级数和分不分
```
❌ 混淆级和分
✅ 级=完美，分=有瑕疵
```

## 💡 练习技巧

### 技巧1：循序渐进
不要急于求成，先从简单的开始：
1. 先练定格（练习1-2）
2. 再练净阴净阳（练习3）
3. 然后练计算（练习4-5）
4. 最后练完整分析（练习10）

### 技巧2：对比验证
- 分析已知的名人八字
- 对照已有的分析结果
- 找出差异和问题

### 技巧3：记录总结
- 每次练习后记录
- 总结经验和教训
- 持续改进方法

### 技巧4：寻求反馈
- 找老师或同行交流
- 对比分析结果
- 不断提高准确率
