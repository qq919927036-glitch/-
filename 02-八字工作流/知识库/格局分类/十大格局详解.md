## 一、正官格

### 1.1 定义
- 以**正官**为格神（格局的核心）
- 正官代表：名誉、地位、权力、官职

### 1.2 传统方法 vs 子平格局

| 传统方法 | 子平格局 |
|---------|---------|
| 甲木日干生于酉月 | 不限定月令 |
| 丙火日干生于子月 | 察旺衰、明变化、辨真假 |
| 机械公式 | 灵活选择 |

### 1.3 定格方法

**基本规则：**
```
命局中五行十神察旺衰
  ↓
明变化、辨真假
  ↓
选择最清纯有力的正官为格
```

**特殊情况：**

1. **官杀混杂**
   - 有七杀混杂
   - 用**偏印**泄七杀
   - 或用**食神**制七杀

2. **正官弱**
   - 用**正财**生官

3. **比劫过旺**
   - 七杀可以混杂
   - 正官旺七杀衰
   - 寻暗拱暗会为真神

4. **日干衰弱无气**
   - 可以论从

### 1.4 用神取法

| 情况 | 用神 |
|------|------|
| 正官为格混七杀 | 偏印泄七杀 / 食神制杀 |
| 正官为格弱 | 正财生官 |
| 正官为格遇伤官 | 正财解救 |

### 1.5 命例分析

#### 例1：董建华（大富大贵）
```
丁 丙 乙 辛
丑 午 未 巳
```

**定格：**
- 地支巳午未会丙丁火方
- 年时巳丑会庚金正官局
- 时干透辛金七杀（坐死地）
- 庚金正官因会局而有力，归时逢长生

**取用：**
- 丙辛一合，丙火合去
- 丁火克尽辛金（食神制杀）
- 以庚金正官为格
- 丁火食神制杀为用

**结果：**
- 合去合留，格局清纯
- 十级富贵（大富大贵）

#### 例2：连战（富可敌国）
```
丙 丙 辛 戊
子 申 已 戌
```

**定格：**
- 申子会水局克丙火
- 巳申一合，丙火庚金皆留
- 以丙火正官为格

**取用：**
- 戊土正印制壬水护正官
- "儿能生母泄天机"

**特点：**
- 时支丙戊闭库
- 格神深藏不被克泄
- 十一级之贵

## 二、七杀格

### 2.1 定义
- 以**七杀**（偏官）为格神
- 七杀代表：权威、魄力、开拓、武职

### 2.2 定格方法

**基本规则：**
```
七杀为格
  ↓
有正官混杂？
  ↓
取伤官制官 / 正印泄官
```

**特殊情况：**

1. **杀旺身弱**
   - 日干有气不从
   - 食神制杀
   - 或伤官驾杀

2. **杀旺日干弱无气**
   - 从七杀

3. **身杀两停**
   - 杀弱者以杀为用
   - 身弱者以印比劫为用

### 2.3 重要原则

> "去官留杀，去杀留官"

> "阴阳相混，都应该有去有留"

> "正官格有杀混杂，就要制服七杀"

> "七杀格有官混杂，就要去掉正官"

### 2.4 命例分析

#### 例1：毛泽东（十二级富贵，帝王之命）
```
癸 甲 丁 甲
巳 子 酉 辰
```

**定格：**
- 子辰会壬癸水局
- 酉巳会庚辛金局
- 庚辛金生壬癸水
- 年干透癸水七杀
- 杀在月令建禄，又有酉中辛金之生

**问题：**
- 有会局之壬水正官混杂七杀

**取用：**
- 月时干双透甲木正印
- 有力化泄壬水
- 壬水被泄得干干净净
- 癸水七杀清纯当令

**结果：**
- 真神得用
- 十二级富贵（帝王之命）
- 地支拱未库六级，亥水长生四级，甲木坐辰二级

**验证：**
- 76年走辰运辰年
- 壬水假神闭库不被制化
- 真假神阴阳混杂征战
- 生命到此结束

#### 例2：南京交警队长
```
庚 甲 癸 己
戌 申 亥 未
```

**定格：**
- 月令金旺
- 日时亥未会局
- 天干甲己合，庚金克甲，甲木留
- 己土因合也留在局中
- 以己土七杀为格

**取用：**
- 用甲木克戊土
- 戌中丙火克庚金正印

**特点：**
- 甲木坐绝无力
- 但生于立秋后三天，己土还在当令
- 喜神都被合留入库
- 忌神能够转化

**结果：**
- 中小富贵
- 正处级别
- 职业为交警队长（武职）

## 三、正财格

### 3.1 定义
- 以**正财**为格神
- 正财代表：正当财富、妻子、节俭、稳健、实利

### 3.2 定格方法

**基本规则：**
```
以正财为格
  ↓
遇有偏财混格？
  ↓
有七杀？
  → 用七杀泄偏财
  → 或用比肩夺偏财
```

**特殊情况：**

1. **正财有克泄**
   - 用**伤官生正财**

2. **原局无印绶比劫**
   - 可以论从

3. **正财旺偏财弱（真旺假衰）**
   - 寻找暗拱暗会为真神

4. **格神过旺，用神弱**
   - "君不可抗也，贵乎损上以益下"
   - 适合行损格神益用神的运

### 3.3 用神取法

| 情况 | 用神 |
|------|------|
| 正财为格，偏财混格，有七杀 | 七杀泄偏财 |
| 正财为格，偏财混格，无七杀，有比肩 | 比肩夺偏财 |
| 正财有克泄 | 伤官生正财 |

### 3.4 命例分析

#### 例1：康有为（从正财格）
```
戊 乙 辛 乙
午 卯 亥 未
```

**定格：**
- 亥卯未会全木局
- 甲乙木混杂同旺
- 当以从甲木正财为格

**取用：**
- 用丁火七杀泄偏财乙木
- 用壬水伤官生格

**特点：**
- 卯亥暗拱辰库伤官六级
- 文章传世，智慧超群
- 十级富贵

#### 例2：嘉庆皇帝（从财格）
```
辛 丁 庚 丙
卯 酉 午 午
```

**定格：**
- 年月卯酉冲
- 丁火盖头
- 丙丁火克酉金
- 酉金日时病地，被冲克殆尽
- 乙木时上得真生
- 当以乙木正财为格

**取用：**
- 用官杀制劫财
- 因是从财格，官杀不怕混杂

## 四、偏财格

### 4.1 定义
- 以**偏财**为格神
- 偏财代表：意外之财、投资、商业、慷慨、大手大脚

### 4.2 定格方法

**基本规则：**
```
以偏财为格
  ↓
遇有正财混格？
  ↓
有正官？
  → 用正官泄正财
  → 或用劫财夺正财
```

**特殊情况：**

1. **偏财有克泄**
   - 用**食神生偏财**

2. **原局无印绶比劫**
   - 可以论从

3. **偏财旺正财弱（真旺假衰）**
   - 寻找暗拱暗会为真神

4. **格神过旺，用神弱**
   - "君不可抗也，贵乎损上以益下"

### 4.3 用神取法

| 情况 | 用神 |
|------|------|
| 偏财为格，正财混格，有正官 | 正官泄正财 |
| 偏财为格，正财混格，无正官，有劫财 | 劫财夺正财 |
| 偏财有克泄 | 食神生偏财 |

### 4.4 命例分析

#### 例1：郭沫若（大文豪）
```
壬 辛 壬 丙
辰 亥 子 午
```

**定格：**
- 子辰会局
- 亥子并排
- 子午相冲
- 偏财格

**取用：**
- 用食神

#### 例2：审计局长
```
丁 壬 丁 庚
酉 子 丑 戌
```

**定格：**
- 丑戌刑出戊己土
- 酉丑会局庚辛金
- 月柱壬癸水禄旺
- 月日子丑合
- 天干丁壬合，地支子丑合
- 月令戊土比己土旺
- 合壬水留，合子水去
- 丑中己土因有酉金之泄也合留
- 以辛金偏财为格

**取用：**
- 用正官壬水泄庚金正财

**关键：**
- 癸水七杀合去
- 正官壬水旺而可用
- 丁壬合的玄机极大
- 如果不是丁壬合，就破格无用了

#### 例3：军官
```
丙 庚 壬 丙
寅 子 寅 午
```

**定格：**
- 寅午会局
- 双丙透干
- 一坐生一坐旺
- 丙火由衰转旺
- 以丙火偏财为格

**取用：**
- 用子中癸水克丁火
- 用寅中甲木生丙火

**特点：**
- 丙火偏财旺而有力
- 可生戊土偏官七杀
- 丙克庚金印
- 戊土在寅中四级
- 可到团级左右

## 关键原则总结

### 1. 净阴净阳
> "去留得宜，格局才能清纯"

### 2. 官杀混杂
> "正官格有杀混杂，就要制服七杀"
> "七杀格有官混杂，就要去掉正官"

### 3. 举一反三
> "官杀混杂既然要有去留，那么财印伤食比劫混杂是否照样应该要有去留呢？"
> "答案是当然的，是一样需要取清的"

### 4. 格神高大用神有力
> "大贵之人必然有大贵之处"
> "格神高大，用神有力"

### 5. 忌神被制越净格局越高
> "忌神被克制越净八字格局越高"
