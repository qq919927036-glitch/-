### 案例一：和珅（11级富贵）

**八字：**
```
庚 乙 庚 壬
午 酉 子 午
```

**定格取用：**
- 月令庚辛金禄旺
- 乙庚合，皆合留
- 以庚金为格
- 用丁火克酉金

**量化计算：**
- 午火月令：4级
- 乙木年支得生：4级
- 子午冲去：3级
- **总计：11级**

**大运分析：**
- 戊子运：庚金死地，假神当真
- 己丑运：庚金丁火入库，好运
- 庚寅运：寅午会，官杀混杂，失势退
- 辛卯运：壬申年，格用去，遭赐死

### 案例二：李鸿章（去留得宜）

**八字：**
```
癸 甲 乙 已
未 寅 亥 卯
```

**定格取用：**
- 寅月亥卯未全
- 甲乙木同旺混杂
- 甲己合，甲木合去
- 以乙木比肩为格
- 用癸水生格

**核心要点：**
> "真假神在原命局去留得宜，甲木合去乙木得清"

**成功原因：**
- 甲己合 → 甲木合去 ✓
- 乙木清纯 ✓
- 癸水用神得长生 ✓
- **去留得宜，格局清纯**

### 案例三：毛泽东（12级帝王）

**八字：**
```
癸 甲 丁 甲
巳 子 酉 辰
```

**定格取用：**
- 子辰会壬癸水局
- 巳酉会庚辛金局
- 庚辛金生壬癸水
- 年干透癸水七杀

**核心问题：**
- 壬水正官混杂七杀
- **必须去留取清！**

**去留方法：**
- 月时双甲木透出
- 甲木化泄壬水
- **壬水被泄得干干净净！**

**结果：**
- 癸水七杀清纯当令
- 真神得用平生贵
- **12级帝王之命**

**量化计算：**
- 地支拱未库：6级
- 亥水长生：4级
- 甲木坐辰：2级
- **总计：12级**

**应期验证：**
- 未运：完成三大战役，成立新中国
- 辰运辰年（76年）：壬水假神闭库，真假混杂，生命结束

**关键口诀：**
> "令上寻真聚的真，假神休要乱真神"
> "真神得用平生贵，用假终为碌碌人"

## 核心理论总结

### 1. 去留取清的四种方法

**方法1：合留合去**
- 毛泽东案例：甲己合，甲木合去
- 李鸿章案例：甲己合，甲木合去

**方法2：化泄**
- 毛泽东案例：甲木化泄壬水

**方法3：克制**
- 和珅案例：丁火克酉金

**方法4：墓库收**
- 南京交警队长：甲木闭于未库

---

### 2. 真假神辨别

**真神的条件：**
1. 在月令得地
2. 有去留之神
3. 能为格局

**假神的特征：**
1. 在月令不得地
2. 无去留之神
3. 不能为格局

**真神得用：**
> "真神得用平生贵，用假终为碌碌人"

**假神当真：**
> "真神死绝假神当真"

---

### 3. 混杂的严重性

**慈禧太后（成功案例）：**
- 甲乙木混杂
- 丁火泄去乙木
- 甲木清纯
- **12级富贵**

**童命造（失败案例）：**
- 甲乙木混杂同旺
- 无去留取清之神
- **贫贱短寿**

**对比：**
- 同样是甲乙木混杂
- 一个有去留（慈禧）
- 一个无去留（童命造）
- **结果天壤之别！**

---

### 4. 合的玄机

**合留喜神：**
- 毛泽东：甲己合留
- 李鸿章：甲己合留
- **吉利**

**合去喜神：**
- 某女士：寅亥合，壬水留
- **不吉**

**合去忌神：**
- 审计局长：癸水合去
- **吉利**

**合留忌神：**
- 发廊女：壬水留
- **不吉**

**关键原则：**
> "合留喜神，合去忌神就好"
> "合留忌神，合去喜神就会凶"
