# 重新分析三个案例 - 按照崔进祥标准流程

## 案例一：万里浪（38岁被枪决）

### 一、八字排盘

```
年柱：壬申  【水金】
月柱：癸酉  【水金】
日柱：甲戌  【木土】
时柱：乙丑  【木土】
```

### 二、标准分析流程

#### 第一步：全面分析

**1. 天干五行**
- 年干：壬水（阳水）
- 月干：癸水（阴水）
- 日干：甲木（阳木）
- 时干：乙木（阴木）

**关键发现：壬癸水混杂！甲乙木混杂！**

**2. 地支关系**
- 申酉戌三会金局（官杀成势）
- 丑戌刑（开出库神）

**3. 藏干分析**
- 申中：庚金（七杀本气）、壬水、戊土
- 酉中：辛金（正官本气）
- 戌中：戊土、辛金、丁火
- 丑中：己土、癸水、辛金

---

#### 第二步：定格

**月令：** 酉月（辛金当令）

**透干十神：**
- 月干癸水 = 伤官
- 年干壬水 = 食神
- 日干甲木 = 日主
- 时干乙木 = 劫财

**地支三会金局：**
- 申酉戌 = 金局极旺
- 金 = 官杀

**核心问题：申酉戌三会金局，辛金正官、庚金七杀都极强！**

**格神判定：**
- 辛金正官在月令，当令透干
- 庚金七杀在年支申中，得禄旺地
- **官杀混杂！**

**结论：官杀混杂格**

---

#### 第三步：去留取清

**问题：** 辛金正官、庚金七杀同旺，如何去留？

**检查去留之神：**
1. 丁火？在戌中藏干，但力量极弱
2. 食神制杀？壬水在年干，但生在酉月（死地），无力
3. 伤官制官？癸水在月干，生在酉月（病地），无力
4. 印星化杀？无印

**结论：无去留之神！官杀混杂，无法取清！**

---

#### 第四步：判断成败

**根据崔老师理论：**
> "官杀混杂时，则需要去杀留官、或者是去官留杀，就是取清意思"

> "命局中官杀混杂时，有去留之神 = 可成格（降级）"
> "命局中官杀混杂时，无去留之神 = 不成格（大凶）"

**万里浪命局：**
- 官杀混杂 ✓
- 无去留之神 ✓
- **不成格，大凶！**

---

#### 第五步：大运引动

**乙丑大运（39-49岁）：**

**天干：乙木**
- 乙木是劫财
- 无法制官杀
- 无力

**地支：丑**
- 丑刑戌（开出戌库）
- 戌中辛金透出
- 丑酉半合金局
- 金更旺！

**结果：**
- 官杀更旺
- 日主更弱
- 官杀混杂加剧
- **必有凶灾！**

**应期：1946年（丙戌年）**
- 丙火透干，看似制官杀
- 但戌与丑刑
- 开出官杀
- 混杂加剧
- **被枪决！**

---

### 三、总结

**核心问题：官杀混杂，无去留之神**

**判断依据：**
1. 辛金正官、庚金七杀同旺
2. 无丁火制杀
3. 食伤无力（壬癸水生在金月）
4. 无印化杀

**结论：**
- 不成格
- 大凶之命
- 易有牢狱之灾

**验证：** ✓ 38岁被枪决

## 案例二：录儿（17岁抢劫杀人）

### 一、八字排盘

```
年柱：癸亥  【水水】
月柱：己未  【土土】
日柱：庚申  【金金】
时柱：甲申  【金金】
```

### 二、标准分析流程

#### 第一步：全面分析

**1. 天干五行**
- 年干：癸水（阴水）
- 月干：己土（阴土）
- 日干：庚金（阳金）
- 时干：甲木（阳木）

**2. 地支关系**
- 亥未（无直接作用）
- 申申伏吟

**3. 藏干分析**
- 亥中：壬水（本气）、甲木
- 未中：己土（本气）、丁火、乙木
- 申中：庚金（本气）、壬水、戊土

---

#### 第二步：定格

**月令：** 未月（己土当令）

**透干十神：**
- 月干己土 = 正印
- 时干甲木 = 偏财

**地支分析：**
- 日支申 = 庚金禄地
- 时支申 = 庚金禄地
- **庚金极旺！**

**格神判定：**
- 庚金日主，双申为禄
- 庚金极强
- 年干癸水 = 伤官（透干）
- **格神：癸水伤官**

---

#### 第三步：检查真衰假旺

**格神：癸水（伤官）**
- 生于未月 = 墓地（死）
- 归时申 = 病地（弱）
- **真衰！**

**假神：壬水（亥中藏干）**
- 生于亥月 = 禄地（但实际在未月，亥为养地）
- 归时申 = 长生地（旺）
- **假旺！**

**假神当真：**
- 壬水（假神）在亥禄、申长生
- 癸水（真神）在未墓、申病
- **假神欺真！**

---

#### 第四步：判断成败

**根据崔老师理论：**
> "癸水格神月令墓地，归时为死，假神壬水月令逢养，归时长生，真衰假旺之命"
> "凡是真衰假旺之命，一生必多坎坷，好时发光无限，坏时人不如"

**录儿命局：**
- 真衰假旺 ✓
- 假神当真 ✓
- **大凶！**

---

#### 第五步：大运引动

**庚辰年（17岁）：**

**天干：庚金**
- 庚金透干
- 克甲木（用神）
- **用神被制！**

**地支：辰**
- 辰 = 壬水墓库
- 假神入库
- 假神更猖狂！

**连锁反应：**
1. 庚金克甲木 → 用神被制
2. 辰收壬水 → 假神入库
3. 壬水混杂癸水 → 伤官肆虐
4. 伤官无制 → 违纪违法
5. **抢劫杀人！**

---

### 三、总结

**核心问题：真衰假旺，假神当真**

**判断依据：**
1. 癸水真神在未墓、申病（真衰）
2. 壬水假神在亥禄、申长生（假旺）
3. 庚辰年：用神被制、假神入库

**结论：**
- 真衰假旺之命
- 一生坎坷
- 好时发光无限
- 坏时人不如狗

**验证：** ✓ 17岁抢劫杀人，无期徒刑

## 案例三：狄警官（被开除）

```

月柱：乙丑  【木土】

```

### 二、标准分析流程

#### 第一步：全面分析

**1. 天干五行**
- 年干：癸水（阴水）
- 月干：乙木（阴木）
- 日干：癸水（阴水）
- 时干：己土（阴土）

**2. 地支关系**
- 卯未半合木局
- 丑未冲

**3. 藏干分析**
- 卯中：乙木（本气）
- 丑中：己土（本气）、癸水、辛金
- 未中：己土（本气）、丁火、乙木

---

#### 第二步：定格

**月令：** 丑月（己土当令）

**透干十神：**
- 月干乙木 = 食神
- 时干己土 = 七杀

**看得见：**
- 乙木（食神）透月干 ✓

**看不见（关键！）：**

- **甲木制化不掉！**

- 但甲木混杂
- **食伤混杂！**

---

#### 第四步：判断伤官无制

**问题：**
- 己土七杀克癸水
- 需要乙木制己土
- 但甲木混杂

**后果：**
- 乙木不能独担
- 伤官（甲木）无制
- **伤官无制 = 不守法度！**

---

#### 第五步：大运引动

**辛酉运（40-50岁）：**

**天干：辛金（枭神）**
- 辛金克乙木（食神）
- **乙木用神被制去！**

**连锁反应：**
1. 辛金克乙木 → 用神被制
2. 己土无制 → 克癸水
3. 甲木伤官还在
4. 伤官肆无忌惮
5. 不守法度
6. **被开除！**

---

### 三、总结

**核心问题：食伤混杂，伤官无制**

**判断依据：**
1. 乙木（食神）透干
2. 甲木（伤官）在未中藏干
3. 时支又见未，制化不掉
4. 辛酉运：枭神制食

**结论：**
- 食伤混杂
- 伤官无制
- 不守法度之人
- 必有官灾
