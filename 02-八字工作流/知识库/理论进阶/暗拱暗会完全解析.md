## 什么是暗拱暗会

### 定义

> "原局所得级数，加上暗拱暗会的级数。而后所得的，才是实际级数。"

**含义：**
- 八字中虽然没有直接写出某个地支
- 但通过其他地支的"拱"或"会"
- 可以把这个地支"暗藏"的力量计算进来

**价值：**
- 增加级数（6级、4级等）
- 弥补原局不足
- 是"玄机暗藏"的重要来源

## 暗拱暗会的基本原理

### 地支顺序

```
子 → 丑 → 寅 → 卯 → 辰 → 巳 → 午 → 未 → 申 → 酉 → 戌 → 亥
```

### 拱的原理

**1. 间拱（中间拱）**

两个地支之间，可以拱出中间的地支：

| 组合 | 拱出 | 示例 |
|------|------|------|
| 子 + 寅 | 丑 | 子寅间拱丑 |
| 丑 + 卯 | 寅 | 丑卯间拱寅 |
| 寅 + 辰 | 卯 | 寅辰间拱卯 |
| 卯 + 巳 | 辰 | **卯巳间拱辰** ✅ |
| 辰 + 午 | 巳 | 辰午间拱巳 |
| 巳 + 未 | 午 | 巳未间拱午 |
| 午 + 申 | 未 | **午申间拱未** ✅ |
| 未 + 酉 | 申 | 未酉间拱申 |
| 申 + 戌 | 酉 | 申戌间拱酉 |
| 酉 + 亥 | 戌 | 酉亥间拱戌 |
| 戌 + 子 | 亥 | 戌子间拱亥 |
| 亥 + 丑 | 子 | 亥丑间拱子 |

> "得时干真生癸水四级，**午前拱未甲木库六级**。共十级之富贵层次。"

**八字：**
```
丙 甲 丁 癸
子 午 酉 卯
```

**分析：**
- 时干癸水长生 = 4级
- **午前拱未库** = 6级
- 总计：4 + 6 = 10级

**理解：**
- 月支是午
- 午"前"可以拱未
- 未是甲木的库（甲库在未）
- 级数：6级

## 暗拱暗会的规则总结

### 规则1：前拱

**"午前拱未"**
- 午之前的地支可以拱未
- 可能是指顺序上"向前"拱

**"未前拱酉、丑"**
- 未之前的地支可以拱酉、丑

**"丑前拱寅"**
- 丑之前的地支可以拱寅

---

### 规则2：间拱

**"卯亥间暗拱辰"**
- 卯和亥之间暗拱辰
- 注意：卯（4）+ 辰（5）+ 巳（6）+ 午（7）+ 未（8）+ 申（9）+ 酉（10）+ 戌（11）+ 亥（12）
- 不是直接相邻，而是跨越多个地支

**"子巳紧贴可以暗会丑"**
- 子和巳"紧贴"（相邻）可以暗会丑

---

### 规则3：多支拱

**"戌亥巳午中暗拱丑"**
- 四个地支戌、亥、巳、午
- 在它们"中间"暗拱丑

**"地支拱未库"**
- 多个地支共同拱一个库

---

### 规则4：格用同拱

**"格用同拱"**
- 格神和用神都能拱到同一个库
- 这是大贵的标志

## 快速查找暗拱暗会的方法

### 方法1：看两个地支之间

**步骤：**
1. 找到两个地支
2. 看它们之间"缺失"了哪些地支
3. 判断是否能"拱"出来

**示例：**
- 卯 + 巳 → 中间缺失了辰 → 可以拱辰
- 子 + 寅 → 中间缺失了丑 → 可以拱丑

---

### 方法2：看"前"和"后"

**前拱：**
- 午前拱未
- 未前拱酉、丑
- 丑前拱寅

**后拱：**
- 需要更多案例验证

---

### 方法3：看多支组合

**组合拱：**
- 戌 + 亥 + 巳 + 午 → 暗拱丑
- 卯 + 亥 → 暗拱辰
- 辰 + 未 → 拱午

## 级数计算公式

### 基本公式

```
实际级数 = 原局级数 + 暗拱暗会级数 - 忌神扣除级数
```

### 示例：毛泽东

```
原局级数：
- 亥水长生 = 4级
- 甲木坐辰（衰）= 2级

暗拱暗会级数：
- 地支拱未库 = 6级

总计：4 + 2 + 6 = 12级
```
