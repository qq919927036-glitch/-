# 大运流年判断方法（崔进祥方法）

> 学习时间：2026-02-21
> 重要程度：⭐⭐⭐⭐⭐

---

## 📋 基本概念

### 命造的构成

**先天 = 八字原局**
**后天 = 大运**
**命运 = 先天原局生之，后天大运成之**

**必须：先天原局与后天大运，合并计算**

### 流年

**流年 = 五行的岁君**
- 任何人都必须遵守
- 例如：2026年是丙午年，无论任何人，流年都是丙午
- 流年主宰一年五行之衰旺

---

## 🎯 大运流年的作用

### 大运的作用

**后天大运虽然不能改变原局之富贵等级**
**却能引导进入富贵顺逆之境，及生死之途**

**大运掌握：**
- 有十年之久
- 是人命成功与否之关键

### 流年的作用

**流年：**
- 一年的主宰
- 公共的，不论大运或流年，都要依据原命冲战合好

---

## 📋 冲战合好四字

### 冲（Chong）

**定义：**
- 冲去原命忌神者喜
- 冲喜神不吉

**判断标准：**
```python
if 大运冲忌神:
    return "吉"
if 大运冲喜神:
    return "凶"
if 流年冲忌神:
    return "吉"
if 流年冲喜神:
    return "凶"

# 特殊情况
if 大运冲真神 and 流年冲假神:
    return "不吉"
```

### 战（Zhan）

**定义：**
- 大运旺真神，流年旺假神
- 流年凑成真假神混战
- 真假神旺衰稍有分别，就不会形成战斗

**战斗的条件：**
- 真假同旺
- 无去留
- 才会战斗

### 合（He）

**定义：**
- 应当合的时候，合就吉
- 不应当合的时候，而合就凶

**合的类型：**
1. **合留喜神** = 吉
2. **合去喜神** = 凶
3. **合留忌神** = 凶
4. **合去忌神** = 吉

**举例：**
- 李鸿章：甲己合，甲木合去 → 吉
- 某女士：甲己合，己土因此一合也留 → 凶（克害用神）

### 好（Hao）

**定义：**
- 流年吉利的意思
- 大运流年同为一气
- 同为吉利的好年运

## 🎯 大运流年综合判断

### 吉利的条件

**原局格用神，与大运流年顺逆同道，有体用者吉利**

**具体表现：**
1. 大运行格神之运 → 吉利
2. 大运行用神之运 → 吉利
3. 大运行格神用神双全之运 → 大吉
4. 大运能去忌神，旺格用神 → 上吉
5. 原命不好能得大运相顾 → 小草春风

### 不利的条件

**与岁运不和克害而无体用者不利**

**具体表现：**
1. 大运流年同入墓 → 极不吉
2. 大运流年同入坏大运流年 → 极不吉
3. 大运流年冲战合好，刑克会害 → 需详细分析

## 📋 不同格局的大运流年喜忌

### 正官格

**官杀混杂：**
- 用偏印泄七杀遇偏财 → 以比劫岁运最佳
- 用食神制杀逢印 → 以比劫岁运最佳

**正官弱：**
- 用正财生官而逢劫 → 以正官运最佳

**用正官而逢伤官：**
- 以正财运最佳

### 七杀格

**官杀混杂：**
- 以伤官运最佳

**身杀两停：**
- 身弱 → 以比劫运，身强杀运为佳

**杀旺：**
- 日干有气不从 → 以食神制杀或伤官驾杀为佳
- 有财泄食伤 → 以比劫运为最佳

### 财格

**财格混浊：**
- 用比劫
- 有食伤泄比劫，或官杀制比劫 → 以印运最佳

**财弱：**
- 用食伤
- 有正偏印克食伤者 → 以正偏财运最佳

### 食伤格

**食伤格有混浊：**
- 以正偏印为用
- 遇有正偏财克制者 → 以比劫运最佳
- 以正偏财泄食伤为用，遇有比劫克制者 → 以食伤运最佳

### 印格

**印格有混浊：**
- 用偏正财去留，而逢比劫者 → 则以官杀或食伤运最佳

### 比劫格

**比劫格有混浊：**
- 则以食伤与官杀为用
- 遇有食伤克制官杀，或正偏印克制食伤者 → 则以比劫运及食伤运最佳

### 从格

**从格忌混浊，更怕日干生旺之运**

**从官杀：**
- 原局有忌印，泄官杀生日干 → 则以财运最佳

### 化气格

**化气格亦忌混浊**

**干头要透出化神**
**生化神之神**
**或地支成化气之局**

**如有克泄：**
- 则以制克，制泄，生化气之神之运为好运
- 混浊则以去混之运为好运

## 📋 特殊情况

### 喜神原局有刑冲克泄

**刑冲：**
- 刑冲以解刑解冲为好运

**克泄：**
- 克泄要行克制克泄之运

**当然：**
- 行合留之运，或入库之运是最佳

### 君臣关系

**君赖臣生：**
- 喜行臣生旺运

**儿能生母：**
- 喜行儿母生旺运

**君过旺：**
- 喜行损上益下运

**臣过旺：**
- 喜行损下益上运

### 原局有阻挡

**宜行通关之运**

**原局格用生旺，又无缺痕，行运又顺：**
- 除行克害格用运岁，稍有不利之外
- 其余到处春风

**但有刑冲克泄破害者，行运又逆：**
- 则好运难成

**原命局格用破败：**
- 行好运也没什么作用

## 🎯 实战案例

### 案例1：和珅

**八字：**
```
庚 乙 庚 壬
午 酉 子 午
```

**戊子运：**
- 庚金死辛金生
- 格神死地假神当真
- 子冲午
- 辛金假神无受破害，可以当真

**己丑运：**
- 庚金丁火格用神入库
- 这二十年是好运，威权显奕

**庚寅运：**
- 庚运尚可
- 寅运会午，官杀混杂，制比刃
- 失势而退

**辛卯运：**
- 壬申年，运干辛金克乙木
- 运支卯酉冲，壬申年申子会局
- 壬水泄庚金，格用去，比刃体神也去
- 遭赐死

### 案例2：杨椒山

**八字：**
```
丙 甲 丁 癸
子 午 酉 卯
```

**己亥运：**
- 己土克癸水用神
- 合去甲木用神
- 丙火格神到绝地
- 格用尽去而凶亡
