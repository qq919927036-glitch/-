## 案例2：判断流年吉凶

### 命造信息
```
慈禧太后
八字：乙未 丁未 乙丑 丁亥
格神：甲木（劫财）
用神：丁火泄乙木，壬水生甲木
```

### 练习要求

**问题：慈禧1881年（辛巳年）发生什么？**

```
Step 1: 分析辛巳年
  - 辛金：正官
  - 巳火：食神

Step 2: 看与格神关系
  - 格神：甲木
  - 辛金克甲木 → 不利

Step 3: 看与用神关系
  - 用神：丁火
  - 巳火扶丁火 → 有利

Step 4: 看与大运关系
  - 当时大运：己亥（约40岁）
  - 亥未会甲木 → 助格神
  - 巳亥冲 → 冲去亥木

Step 5: 综合判断
  → 请判断吉凶
```

### ✅ 历史事实

1881年（辛巳年）：
- 慈禧47岁
- 东太后慈安去世
- 慈禧开始独掌大权

**分析**：
- 辛金正官混杂本应不利
- 但巳亥冲冲去亥木（助格神者被冲）
- 实际结果是：慈安去世，慈禧独掌大权
- 对慈禧而言：得权（虽然手段不光彩）

## 案例3：真假神在大运的变化

### 命造信息
```
录儿（真衰假旺案例）
八字：壬寅 癸丑 壬辰 壬亥
格神：癸水（真神，1级）
假神：壬水（7级）
原局：无去留 → 大凶
```

### 练习要求

**问题：在丙午运（火运）会发生什么？**

```
Step 1: 分析丙午运
  - 丙火：伤官
  - 午火：食神
  - 火旺泄水

Step 2: 看与真神关系
  - 真神：癸水（1级）
  - 丙午泄癸水 → 真神更弱

Step 3: 看与假神关系
  - 假神：壬水（7级）
  - 丙午泄壬水 → 假神得制

Step 4: 判断变化
  - 假神被制，真神显露
  - → 虽然大运泄水，但去假留真
  - → 这步大运反而有利

请判断：此运吉凶如何？
```

### ✅ 答案

**丙午运（约30-39岁）**：
- 火旺泄水，制去假神壬水
- 虽然也泄真神癸水
- 但假神得制后，真神能够发挥作用
- **这步大运反而比较顺利**

**启示**：
- 真衰假旺的命局
- 大运流年如果能去假留真
- 即使不生扶格神也能有利

## 案例4：冲战合好判断

### 命造信息
```
八字：甲子 丙寅 甲午 丁卯
格神：甲木（比肩）
用神：丙火食神
```

### 练习要求

**问题1：庚申年（金年）是冲是战？**

```
分析：
- 大运：假设行丁卯运（助甲木）
- 流年：庚申（金）
- 庚金：七杀
- 申金：七杀禄地

判断：
- 庚申冲克甲木 → 冲
- 但申子辰三合水 → 合
- 冲合同时出现

→ 是冲为主还是合为主？
```

**问题2：此年吉凶如何？**

```
提示：
1. 冲去格神 → 不利
2. 但有合化 → 减轻冲击
3. 需要看冲合哪个力量大
```

### ✅ 答案

**庚申年**：
- 庚申金克甲木 → 主要力量是冲
- 申子辰合水 → 次要力量是合
- 冲大于合 → 格神受损
- **此年不利，可能有灾**

## 案例5：合留合去的区别

### 命造信息
```
李鸿章
八字：癸未 甲寅 乙亥 乙卯
格神：乙木（比肩）
特殊：月时甲己合，甲被合去
```

### 练习要求

**问题1：为什么甲己合是合去甲木？**

```
分析：
- 原局：甲木、乙木同透
- 甲乙比劫混杂
- 月时甲己一合

判断：
- 甲木是否遭到克泄？
- 如果是 → 合留
- 如果否 → 合去

→ 请判断并说明理由
```

**问题2：如果甲木被合去，为什么还是好命？**

```
提示：
1. 合去甲木后，留下什么？
2. 乙木能否为格？
3. 用神是什么？
```

### ✅ 答案

**问题1**：
- 甲木在原局没有遭到克泄
- 与己金作合 → **合去**
- 甲木被绊住，不能发挥作用

**问题2**：
- 甲木被合去后，留下乙木
- 乙木为格（比肩）
- 用癸水偏印生格
- 真假神在原命局去留得宜
- 虽然甲被合去，但格局清纯
- **所以仍是好命**

**需要的大运流年**：

3. **燥土运**：戌未土克去癸水

**如果能遇到这样的运**：
- 假神癸水被去
- 真神壬水留存
- 格局清纯

**启示**：

## 案例9：战斗的判断

### 命造信息
```

格神：甲乙混杂

```

**问题1：甲申大运、乙酉流年，是战吗？**

```

- 大运：甲申（扶甲木真神）

判断条件：
1. 真神（甲木）旺？
```

```
- → 大凶
```

- 乙木在流年得酉中辛金生 → 假神也旺
- 原局无去留之神
- **形成战斗 → 大凶**

**此年可能**：

## 💡 练习总结

### 判断大运流年的核心要点

1. **定格取用是基础**
   - 必须先确定格神用神
   - 否则无法判断吉凶

2. **看与格神关系**
   - 生扶格神 → 好
   - 克泄格神 → 坏

3. **看与用神关系**
   - 生扶用神 → 好
   - 克泄用神 → 坏

4. **看与忌神关系**
   - 制忌神 → 好
   - 扶忌神 → 坏

5. **综合判断**
   - 原局 + 大运 + 流年
   - 三者综合判断

### 常见错误

❌ 只看流年不看大运
❌ 只看大运不看原局
❌ 忽视合化的作用
❌ 混淆冲战合好
❌ 不懂真假神的消长

### 进阶方向

1. **精确应期**
   - 什么时候发生好事？
   - 什么时候发生坏事？
   - 需要看流月流日

2. **量化吉凶程度**
   - 小吉、中吉、大吉？
   - 小凶、中凶、大凶？
   - 需要结合级数分

3. **特殊格局处理**
   - 从格的大运流年
   - 化格的大运流年
   - 需要特殊判断
