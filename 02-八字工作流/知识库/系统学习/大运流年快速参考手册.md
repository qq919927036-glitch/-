## 一、大运排法

### 1.1 顺行

```
适用：
  • 阳年出生的男性（甲丙戊庚壬）
  • 阴年出生的女性（乙丁己辛癸）

方法：
  从月令向下推

示例：
  月令：辛巳
  大运：壬午 → 癸未 → 甲申 → 乙酉...
```

### 1.2 逆行

```
适用：
  • 阴年出生的男性（乙丁己辛癸）
  • 阳年出生的女性（甲丙戊庚壬）

方法：
  从月令向上推

示例：
  月令：辛巳
  大运：庚辰 → 己卯 → 戊寅 → 丁丑...
```

### 1.3 起运计算

```
公式：
  到节气天数 ÷ 3 = 起运年数

顺行：
  从出生日算到下一个节气

逆行：
  从出生日算到上一个节气

示例（顺行）：
  出生：5月15日
  节气：芒种（6月6日）
  天数：22天
  起运：22 ÷ 3 ≈ 7年4个月
```

## 二、流年特点

### 2.1 核心地位

```
流年 = 天子地位
  • 任何人今年的流年都一样
  • 大运无法更改流年
  • 一年休咎尤系乎岁
```

### 2.2 流年作用

```
好流年：
  ✓ 生扶格神
  ✓ 生扶用神
  ✓ 制忌神
  ✓ 去假神

坏流年：
  ✗ 冲克格神
  ✗ 冲克用神
  ✗ 扶忌神
  ✗ 假神当真
```

## 三、冲战合好

### 3.1 冲

```
地支相冲：
  子午  丑未  寅申
  卯酉  辰戌  巳亥

判断：
  冲去忌神 → 喜
  冲去喜神 → 凶
  冲去格神 → 大凶
  冲去用神 → 大凶
```

### 3.2 战

```
战斗条件：
  1. 真神旺
  2. 假神也旺
  3. 无去留之神

判断：
  大运扶真神 + 流年扶假神 → 战
  大运扶假神 + 流年扶真神 → 战
  真假神旺衰有差 → 不战

结果：
  战斗 → 大凶
```

### 3.3 和（合）

```
合的种类：
  天干五合：甲己 乙庚 丙辛 丁壬 戊癸
  地支六合：子丑 寅亥 卯戌 辰酉 巳申 午未
  地支三合：申子辰 亥卯未 寅午戌 巳酉丑
  地支三会：亥子丑 寅卯辰 巳午未 申酉戌

判断：
  合留喜神、合去忌神 → 吉
  合留忌神、合去喜神 → 凶
```

### 3.4 好

```
好的定义：
  大运流年同为一气 = 吉

判断：
  大运流年同气 → 吉
  同入喜神生旺之地 → 大吉
  同入坏地 → 极不吉
```

## 四、合留合去

### 4.1 合留

```
定义：
  格神或用神遭到克泄
  命局有救助之神来合
  → 合留 = 吉

示例：
  格神甲木被庚克
  命局有乙木来合庚
  → 乙合庚 = 合留 = 保护格神
```

### 4.2 合去

```
定义：
  格用神没遭克泄
  而与他神作合
  → 合去 = 凶

示例：
  格神甲木本无克
  命局有己土来合甲
  → 己合甲 = 合去 = 格神被绊住
```

### 4.3 合化

```
条件：
  1. 得时（月令）
  2. 得地（地支）
  3. 不被克

示例：
  甲己合化土
  必须在辰戌丑未月（土月）
  地支要有辰戌丑未
  不能有木来克土

不满足条件：
  → 只合不化
  → 但仍有合留合去作用
```

## 五、判断流程

### 5.1 大运判断流程

```
Step 1: 定格取用
  ↓
Step 2: 看大运五行
  ↓
Step 3: 判断与格神关系
  - 生扶格神 → 好
  - 克泄格神 → 坏
  ↓
Step 4: 判断与用神关系
  - 生扶用神 → 好
  - 克泄用神 → 坏
  ↓
Step 5: 判断与忌神关系
  - 制忌神 → 好
  - 扶忌神 → 坏
  ↓
Step 6: 综合判断
  - 格用神都得生 → 大吉
  - 格用神一得一生 → 中吉
  - 格用神都受损 → 大凶
```

### 5.2 流年判断流程

```
Step 1: 看流年五行
  ↓
Step 2: 看与原局关系
  - 生扶格神 → 吉
  - 克泄格神 → 凶
  ↓
Step 3: 看与大运关系
  - 大运流年同气 → 吉
  - 大运流年相战 → 凶
  ↓
Step 4: 看合化冲刑
  - 合去忌神 → 吉
  - 合去喜神 → 凶
  ↓
Step 5: 综合判断
  - 三好 → 大吉
  - 两好一平 → 中吉
  - 一好两坏 → 小凶
  - 三坏 → 大凶
```

## 六、常见情况

### 6.1 原局坏得好运救

```
情况：
  原局有混杂，无去留
  命局不好

救应：
  大运流年有去留之神

结果：
  可以小草春风
  在这步运中发达
```

### 6.2 好命行坏运

```
情况：
  原局格局清纯
  但行坏运

表现：
  格神受损
  用神被制
  英雄无用武之地

结果：
  怀才不遇
  郁郁不得志
```

### 6.3 真假神变化

```
真衰假旺：
  原局真神弱，假神旺
  → 不是好命

大运救应：
  制去假神
  真神显露
  → 这步运反而好

假神当真：
  大运扶假神，制真神
  → 大凶
```

### 6.4 冲合同时出现

```
情况：
  大运流年与原局既有冲也有合

判断：
  看哪个力量大
  冲大于合 → 以冲为主
  合大于冲 → 以合为主

一般：
  地支三合三会力量 > 六冲
  六冲力量 > 六合
```

### 6.5 大运流年相战

```
情况：
  大运扶真神
  流年扶假神
  → 战斗

结果：
  大凶
  内外交困
  做事不成
```

## 🎯 实战技巧

### 技巧1：抓主要矛盾

```
大运流年判断时：
  先看与格神的关系
  再看与用神的关系
  最后看与忌神的关系

格神是第一位的！
```

### 技巧2：看力量对比

```
不是简单的吉凶判断
要看力量大小

例：
  大运好、流年坏
  看哪个力量大
  力量大的一方主导
```

### 技巧3：重视合化

```
合化改变格局
不要忽视

即使不合化
也有合留合去的作用
```

### 技巧4：注意真假神

```
大运流年会影响真假神
可能真神变假
可能假神变真

需要重新判断
```

### 技巧5：综合判断

```
不要单看一点
要综合判断

原局 + 大运 + 流年
三者合一
才是准确判断
```
