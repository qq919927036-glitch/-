## 🎯 一、核心原理（3句话讲清楚）

### 1. 命运构成

```
先天 = 八字原局（基础）
后天 = 大运（从原局引出）
结果 = 先天生之，后天成之
```

**白话**：
- 八字原局是你的起点（好坏已定）
- 大运是你的过程（如何走）
- 先天决定了富贵等级
- 后天决定了能否达到

### 2. 地位对比

```
大运 = 后天主宰（掌握10年）
流年 = 年中天子（主宰1年）
流年 > 大运（流年无法更改）
```

**核心**：
> **"大运无权更改流年"**
> **"一年休咎尤系乎岁"**

### 3. 判断总纲

```
原局格用神 + 大运流年
  → 顺逆同道，有体用 = 吉
  → 与岁运不和，无体用 = 凶
```

**3个判断点**：
1. 原局格用神是什么？
2. 大运对格用神的影响？
3. 流年对格用神的影响？

## 💡 二、冲战合好（4字诀）

### 冲

```
冲去忌神 → 喜 ✓
冲去喜神 → 凶 ✗

例：
  原局忌神在午
  流年子水来冲
  → 冲去忌神 → 吉
```

**6冲**：
```
子午冲  丑未冲  寅申冲
卯酉冲  辰戌冲  巳亥冲
```

### 战

```
战斗 = 真假神同旺无去留

条件（缺一不可）：
1. 真神旺
2. 假神也旺
3. 无去留之神

结果：大凶 ✗

例：
  原局甲乙混杂同旺
  大运扶甲（真神）
  流年扶乙（假神）
  → 真假神混战 → 大凶
```

**关键**：
> **"真假神旺衰稍有分別，就不会形成战斗"**
> **"战斗必然是真假同旺无去留"**

### 和（合）

```
合留喜神、合去忌神 → 吉 ✓
合留忌神、合去喜神 → 凶 ✗

例1（合留）：
  格神甲木被庚克
  命局有乙木合庚
  → 乙合庚 = 合留 = 保护格神

例2（合去）：
  格神甲木本无克
  命局有己土合甲
  → 己合甲 = 合去 = 格神被绊
```

### 好

```
好 = 流年吉利

判断：
  大运流年同为一气 → 吉
  大运流年同入喜神生旺之地 → 大吉
  大运流年同入坏地 → 极不吉
```

## 🔥 三、核心心法（背下来）

### 心法1：富贵等级

```
原局 = 基础等级（不能改变）
大运 = 能否达到（引导过程）
流年 = 具体实现（每年情况）
```

**重要**：
> **"后天大运虽然不能改变原局之富贵等级，却能引导进入富贵顺逆之境"**

### 心法2：成功关键

```
大运掌握10年
有的二三十年好运
有的只有五年十年好运
→ 这就是人生差距
```

**例**：
```
原命以戊土为格，用丙火，生于正月

顺行大运：
  由长生入帝旺
  → 一生都是顺运顺境

逆行大运：
  最好的运是戌运库运
  → 只有五年或十年好运
```

### 心法3：冲战合好不能完全代表

```
除了冲战合好
还要看：刑克会合

刑：刑喜神 vs 刑忌神
克：克喜神 vs 克忌神
会：会喜神 vs 会忌神
合：合留喜神 vs 合去喜神
```

### 心法4：原局缺陷

```
看大运流年之前
必须知道原局缺陷
→ 才能知道需要什么五行来相助
```

**顺序**：
```
1. 看原局缺陷
2. 看需要什么五行
3. 看大运流年有没有
4. 判断吉凶
```

## 🎓 五、实战要点（崔进祥原话）

### 要点1：格用神的重要性

```
原局格用神，与大运流年顺逆同道，有体用者吉利
与岁运不和克害而无体用者不利

大运行格神之运，或行用神之运者吉利
大运行格神用神双全之运大吉
大运能去忌神，旺格用神也是上吉
```

### 要点2：原命坏得好运救

```
原命不好能得大运相顾，也可以小草春风
```

**例**：
```
原局有混杂，命局不好
但大运流年有去留之神
→ 在这步运中也能发达
```

### 要点3：好命行坏运

```
原命格用破败，行好运也没什么作用
```

**例**：
```
原局格局清纯
但行坏运
→ 英雄无用武之地
```

### 要点4：老命行好运

```
如果大运行入格用之乡
即使年已老迈
也必是童颜鹤发精神奕奕
```

### 要点5：入坏地大凶

```
人命行入喜神衰竭，凶神生旺之方
如不遭凶亡，也必然患病连年，破财刑耗
或未老身衰而亡
```

## 🌟 六、不同格局的大运流年取用

### 正官格

```
正官为格混七杀
→ 用偏印泄七杀遇偏财
→ 以比劫岁运最佳

正官为格弱
→ 用正财生官而逢劫
→ 以正官运最佳

用正官而逢伤官
→ 以正财运最佳
```

### 七杀格

```
七杀为格混正官
→ 以伤官运最佳

七杀为格混官
→ 用正印泄官逢财
→ 或用伤官制官逢印
→ 皆以比劫运最佳
```

### 财格

```
以财为格混浊
→ 用比劫
→ 若有食伤泄比劫，或官杀制比劫
→ 以印运最佳

财弱用食伤
→ 有正偏印克食伤者
→ 以正偏财运最佳
```

### 食伤格

```
以食伤为格有混浊
→ 以正偏印为用
→ 而遇有正偏财克制者
→ 以比劫运最佳
```

### 印格

```
以印为格有混浊
→ 用偏正财去留
→ 而逢比劫者
→ 则以官杀或食伤运最佳
```

### 比劫格

```
以比劫为格有混浊
→ 则以食伤与官杀为用
→ 遇有食伤克制官杀
→ 或正偏印克制食伤者
→ 则以比劫运及食伤运最佳
```

### 从格

```
从格忌混浊
更怕日干生旺之运

如从官杀
→ 原局有忌印，泄官杀生日干
→ 则以财运最佳
```

### 化气格

```
化气格亦忌混浊
干头要透出化神
生化神之神
或地支成化气之局

如有克泄
→ 则以制克、制泄、生化气之神之运为好运

混浊
→ 则以去混之运为好运
```
