## 📖 第一部分：大运

### 1.1 大运的排法

#### 阳年男阴年女（顺行）

```
甲、丙、戊、庚、壬年出生的男性
乙、丁、己、辛、癸年出生的女性

→ 大运顺行（从月令向下推）
```

**示例**：
```
男命，1990年（庚午年）5月生

月令：辛巳
大运：壬午 → 癸未 → 甲申 → 乙酉...
```

#### 阴年男阳年女（逆行）

```
乙、丁、己、辛、癸年出生的男性
甲、丙、戊、庚、壬年出生的女性

→ 大运逆行（从月令向上推）
```

**示例**：
```
女命，1990年（庚午年）5月生

月令：辛巳
大运：庚辰 → 己卯 → 戊寅 → 丁丑...
```

### 1.2 大运起运计算

#### 公式

```
到节气天数 ÷ 3 = 起运年数

顺行：从出生日算到下一个节气
逆行：从出生日算到上一个节气
```

**示例1（顺行）**：
```
男命，1990年5月15日生
节气：芒种（6月6日）

计算：
  5月15日 → 6月6日 = 22天
  22 ÷ 3 = 7.33年

起运：约7年4个月开始大运
```

**示例2（逆行）**：
```
女命，1990年5月15日生
节气：立夏（5月6日）

计算：
  5月6日 → 5月15日 = 9天
  9 ÷ 3 = 3年

起运：约3年开始大运
```

### 1.3 大运的作用

#### 1. 掌握十年之久
```
大运1：8-17岁
大运2：18-27岁
大运3：28-37岁
大运4：38-47岁
大运5：48-57岁
大运6：58-67岁
大运7：68-77岁
大运8：78-87岁
```

#### 2. 人命成功与否的关键
- 有的人二三十年好运
- 有的人只有五年十年好运
- 全看大运配合

#### 3. 大运的性质
```
好大运：
  - 行格神之运
  - 行用神之运
  - 去忌神之运
  - 旺格用神之运

坏大运：
  - 冲克格神
  - 冲克用神
  - 扶起忌神
  - 假神当真
```

### 1.4 大运的四种关系

#### 冲
```
冲去原命忌神者 → 喜
冲喜神 → 不吉

大运冲真神，流年冲假神 → 不吉
大运冲假神，流年冲真神 → 更不吉
```

#### 战
```
战斗 = 真假神同旺无去留

大运旺真神，流年旺假神 → 战斗
大运旺假神，流年旺真神 → 战斗

真假神旺衰稍有分别 → 不会战斗
```

#### 和（合）
```
合留喜神、合去忌神 → 吉
合留忌神、合去喜神 → 凶

应当合的时候，合就吉
不应当合的时候，合就凶
```

#### 好
```
大运流年同为一气 → 吉利
同入喜神生旺之地 → 大吉
同入坏地 → 极不吉
```

## 📖 第二部分：流年

### 2.1 流年的特点

#### 1. 公共性
- 任何人今年的流年都一样
- 2026年是丙午年，所有人都是丙午

#### 2. 天子地位
- 流年是一年之主宰
- 大运无法更改流年
- 一年休咎尤系乎岁

#### 3. 流年的作用
```
好流年：
  - 生扶格神
  - 生扶用神
  - 制忌神
  - 去假神

坏流年：
  - 冲克格神
  - 冲克用神
  - 扶忌神
  - 假神当真
```

### 2.2 流年判断要点

#### 1. 看与原局的关系
```
流年生扶格神 → 吉
流年克泄格神 → 凶

流年喜神到位 → 发达
流年忌神到位 → 灾祸
```

#### 2. 看与大运的关系
```
大运流年同气 → 吉
大运流年相战 → 凶

大运好，流年坏 → 中吉
大运坏，流年好 → 小吉
大运好，流年好 → 大吉
```

#### 3. 看合化冲刑
```
流年合喜神 → 凶（喜神被合去）
流年合忌神 → 吉（忌神被合去）

流年冲喜神 → 凶
流年冲忌神 → 吉

流年刑害喜神 → 凶
流年刑害忌神 → 吉
```

## 📖 第三部分：冲战合好详解

### 3.1 冲

#### 冲的定义
```
地支相冲：
子午冲  丑未冲  寅申冲
卯酉冲  辰戌冲  巳亥冲
```

#### 冲的判断
```
1. 冲去忌神 → 喜
   例：原局有忌神在午
   流年子水来冲 → 去忌神 → 吉

2. 冲去喜神 → 凶
   例：原局有喜神在子
   流年午火来冲 → 去喜神 → 凶

3. 冲去格神 → 大凶
   例：格神为子水
   流年午火来冲 → 破格局 → 大凶

4. 冲去用神 → 大凶
   例：用神为卯木
   流年酉金来冲 → 破用神 → 大凶
```

#### 冲的特殊情况
```
1. 旺冲衰
   例：子水旺，午火衰
   流年午来冲子 → 子不败 → 午受伤

2. 衰冲旺
   例：子水衰，午火旺
   流年子来冲午 → 子受伤 → 午不败

3. 两冲
   例：原局有子午冲
   流年又逢子或午 → 加重冲 → 凶
```

### 3.2 战

#### 战的定义
```
战斗 = 真假神同旺无去留
```

#### 战的判断
```
条件1：真神旺
条件2：假神也旺
条件3：无去留之神
→ 战斗产生

例：
  原局：甲木真神、乙木假神同旺
  大运：扶甲木（真神）
  流年：扶乙木（假神）
  → 真假神混战 → 大凶
```

#### 战的避免
```
1. 有去留之神
   例：有丁火泄乙木
   → 假神被去 → 不战

2. 真假旺衰有差
   例：甲木很旺，乙木稍弱
   → 不形成战斗

3. 大运流年同扶一神
   例：都扶甲木或都扶乙木
   → 不战
```

### 3.3 和（合）

#### 合的种类
```
天干五合：
甲己合  乙庚合  丙辛合
丁壬合  戊癸合

地支六合：
子丑合  寅亥合  卯戌合
辰酉合  巳申合  午未合

地支三合：
申子辰合水  亥卯未合木
寅午戌合火  巳酉丑合金

地支三会：
亥子丑会水  寅卯辰会木
巳午未会火  申酉戌会金
```

#### 合留 vs 合去
```
合留：
  格神或用神遭到克泄
  命局有救助之神来合
  → 合留 = 吉

  例：格神甲木被庚克
  命局有乙木来合庚
  → 乙合庚 = 合留 = 保护格神

合去：
  格用神没遭克泄
  而与他神作合
  → 合去 = 凶

  例：格神甲木本无克
  命局有己土来合甲
  → 己合甲 = 合去 = 格神被绊住
```

#### 合化的条件
```
1. 得时（月令）
   例：甲己合化土
   必须在辰戌丑未月（土月）

2. 得地（地支）
   例：甲己合化土
   地支要有辰戌丑未

3. 不被克
   例：甲己合化土
   不能有木来克土

如果条件不满足：
  → 只合不化
  → 但仍然有合留合去的作用
```

### 3.4 好

#### 好的定义
```
好 = 流年吉利
```

#### 好的判断
```
1. 大运流年同为一气
   例：大运甲木，流年甲木
   → 同气相求 → 吉

2. 同入喜神生旺之地
   例：格神甲木，大运流年都入寅卯
   → 生旺之地 → 大吉

3. 同入坏地
   例：格神甲木，大运流年都入申酉
   → 克害之地 → 极不吉

4. 原局格用神与大运流年顺逆同道
   例：格神甲木，大运流年都生扶甲木
   → 有体用 → 吉利

5. 原局格用神与大运流年不和
   例：格神甲木，大运流年都克甲木
   → 无体用 → 不利
```

## 📖 第五部分：判断流程

### 5.1 大运判断流程

```
Step 1: 定格取用
  ↓
Step 2: 看大运五行
  ↓
Step 3: 判断与格神关系
  - 生扶格神 → 好
  - 克泄格神 → 坏
  ↓
Step 4: 判断与用神关系
  - 生扶用神 → 好
  - 克泄用神 → 坏
  ↓
Step 5: 判断与忌神关系
  - 制忌神 → 好
  - 扶忌神 → 坏
  ↓
Step 6: 综合判断
  - 格用神都得生 → 大吉
  - 格用神一得一生 → 中吉
  - 格用神都受损 → 大凶
```

### 5.2 流年判断流程

```
Step 1: 看流年五行
  ↓
Step 2: 看与原局关系
  - 生扶格神 → 吉
  - 克泄格神 → 凶
  ↓
Step 3: 看与大运关系
  - 大运流年同气 → 吉
  - 大运流年相战 → 凶
  ↓
Step 4: 看合化冲刑
  - 合去忌神 → 吉
  - 合去喜神 → 凶
  - 冲去忌神 → 吉
  - 冲去喜神 → 凶
  ↓
Step 5: 综合判断
  - 原局好、大运好、流年好 → 大吉
  - 两好一平 → 中吉
  - 一好两坏 → 小凶
  - 三坏 → 大凶
```

## 📖 第六部分：实战要点

### 6.1 判断吉凶的核心

```
原局格用神 + 大运流年
  ↓
顺逆同道，有体用 → 吉
  ↓
与岁运不和克害无体用 → 凶
```

### 6.2 好大运流年的标准

```
✅ 大运行格神之运
✅ 大运行用神之运
✅ 大运能去忌神
✅ 大运能旺格用神
✅ 流年生扶格神
✅ 流年生扶用神
✅ 流年制忌神
✅ 大运流年双全
```

### 6.3 坏大运流年的标准

```
❌ 大运冲克格神
❌ 大运冲克用神
❌ 大运扶起忌神
❌ 大运假神当真
❌ 流年冲克格神
❌ 流年冲克用神
❌ 流年扶起忌神
❌ 流年假神当真
```

### 6.4 特殊情况处理

```
原命不好 + 好大运好流年
  → 可以小草春风

原命好 + 坏大运坏流年
  → 英雄无用武之地

原命中等 + 大运流年参半
  → 起伏不定
```
