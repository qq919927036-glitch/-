## 🏆 一、崔进祥的5个核心洞察

### 洞察1：八字玄机在哪里？

> **"命理的玄机奥秘都在五行的生、合、库、及刑冲制化之中。都在阴阳二气的生死互换之内。"**

**解读**：
- 不是看表面十神
- 而是看五行生克
- 不是看天干地支
- 而是看刑冲制化

### 洞察2：如何判断富贵层次？

> **"只要我们调整思路，抓住重点，阴阳之理大道至简，断人的富贵层次，吉凶祸福不是难事。"**

**关键**：
- 抓住重点 = 格神用神
- 阴阳之理 = 净阴净阳
- 大道至简 = 去假留真

### 洞察3：暗拱暗会的重要性

> **"当我们水平提高后，能看出八字的暗拱暗会时，八字就不仅仅是这八个字的问题了。"**

**例**：
```
和珅案例：
  表面：庚午 乙酉 庚壬 子午
  实际：子午暗拱暗会 = 11级富贵
```

### 洞察4：合留合去的玄机

> **"合有宜不宜，合多不为奇。所以合留喜神，合去忌神就好。合留忌神，合去喜神就会凶。"**

**判断**：
```
合留：
  格神或用神遭到克泄
  命局有救助之神来合
  → 合留 = 吉

合去：
  格用之神在命局中并没有遭到克泄
  而与他神作合
  → 合去 = 凶
```

### 洞察5：格用尽去的后果

> **"己亥运，己土克癸水用神，合去甲木用神，丙火格神到绝地，格用尽去而凶亡。"**

**明朝宰相杨椒山案例**：
```
原局：丙子 甲午 丁酉 癸卯
格神：丙火（刧财）
用神：癸水（制丁）、甲木（泄丙）

己亥运：
  己土克癸水用神 ✗
  合去甲木用神 ✗
  丙火格神到绝地 ✗
  → 格用尽去而凶亡
```

## 💎 二、实战案例深度解析

### 案例1：和珅（大富大贵）

```
八字：庚午 乙酉 庚壬 子午
```

**定格**：
```
月令庚辛金禄旺之地
天干乙庚合，皆合留在局
乙木可以泄子水生丁火
→ 以庚金为格
```

**用神**：
```
用丁火克酉金
乙木之所以可用，是因：
  乙木年支得长生
  （阴干之长生带泄）
  （儿能生母泄天机）
```

**暗拱暗会**：
```
日时子午相冲
→ 子午皆冲去
→ 子水不会破害年支午火
→ 可以寻暗拱暗会为真神
```

**级数计算**：
```
午火月令：4级
乙木年支得生：4级
子午冲去：3级
共：11级（亿万富翁）
```

**大运验证**：

```
戊子运：
  庚金死辛金生
  格神死地假神当真
  子冲午
  → 辛金假神无受破害，可以当真

己丑运：
  庚金丁火格用神入库
  → 这二十年是好运，威权显奕

庚寅运：
  庚运尚可
  寅年会午
  → 官杀混杂，制比刃，失势而退

辛卯运：
  壬申年
  运干辛金克乙木
  运支卯酉冲
  壬申年申子会局
  壬水泄庚金
  → 格用去，比刃体神也去，遭赐死
```

**核心心法**：
```
1. 假神当真 = 可以暂时发达
2. 格用入库 = 好运20年
3. 格用尽去 = 凶亡
```

### 案例2：重庆政协副主席

```
八字：戊戌 辛酉 丁未 癸卯
```

**定格**：
```
地支卯未会局
甲乙木混杂不清
月令金旺看似卯木不能成格

玄机在于：
  时干癸水坐长生
  有酉金克卯木
  癸水得长生可以泄酉金生卯木
  → 乙木偏印仍然可以成格
```

**用神**：
```
用癸水和戌中丙火
戌中丙火：6级
癸水长生：4级
共：10级（省部级）
```

**核心心法**：
```
1. 混杂不清不一定不能成格
2. 看玄机（癸水长生）
3. 有去留（酉克卯、癸泄酉生卯）
4. 仍可成格
```

## 🌟 三、崔进祥的实战技巧

### 技巧1：如何判断玄机？

```
看五行长生奥秘：
  癸水在月令得长生
  → 泄酉金生卯乙木

看暗拱暗会：
  年时二子会局
  → 增加级数

看刑冲制化：
  卯戌合、酉酉自刑
  → 产生特殊作用
```

### 技巧2：如何判断成格？

```
表面看不成格
但有玄机：
  1. 五行得长生
  2. 暗拱暗会
  3. 刑冲制化
  → 仍可成格
```

### 技巧3：如何计算级数？

```
原局级数 + 暗拱暗会级数 - 忌神扣除 = 实际级数

例：
  午火月令：4级
  乙木年支：4级
  子午冲去：3级
  = 11级
```

### 技巧4：如何判断大运？

```
看格神在大运的状态：
  生旺 → 好
  克泄 → 坏

看用神在大运的状态：
  生旺 → 好
  克泄 → 坏

看忌神在大运的状态：
  受制 → 好
  生旺 → 坏

综合判断：
  格用神都得生 → 大吉
  格用神一得一生 → 中吉
  格用神都受损 → 大凶
```

### 技巧5：如何预测死亡？

```
格用尽去 → 死亡

条件：
  1. 格神被克泄
  2. 用神被克泄
  3. 无救应之神

例：杨椒山
  己亥运：
  己土克癸水用神
  合去甲木用神
  丙火格神到绝地
  → 格用尽去而凶亡
```

## 🔥 四、142个名人案例验证的核心发现

### 发现1：格用尽去必死

**验证案例**：
```
杨椒山：格用尽去 → 凶亡
和珅：格用尽去 → 被赐死
```

**结论**：
```
原局格用神 + 大运流年
  如果大运流年让格用神完全失去作用
  → 必死无疑
```

### 发现2：假神当真可发

**验证案例**：
```
和珅戊子运：
  庚金死辛金生
  格神死地假神当真
  → 可以暂时发达
```

**结论**：
```
假神当真 ≠ 好命
但假神当真时可以发达
等到真神出现 → 灾祸降临
```

### 发现3：格用入库好运

**验证案例**：
```
和珅己丑运：
  庚金丁火格用神入库
  → 这二十年是好运，威权显奕
```

**结论**：
```
格用神入库：
  = 好运10-20年
  = 发达、有权、有势
```

### 发现4：混杂不一定不格

**验证案例**：
```
重庆政协副主席：
  甲乙木混杂不清
  但有癸水长生玄机
  → 仍可成格，10级富贵
```

**结论**：
```
混杂 ≠ 不能成格
关键看：
  1. 有无去留之神
  2. 有无玄机暗藏
  3. 有无长生奥秘
```

### 发现5：长生奥秘很重要

**验证案例**：
```
多个案例都显示：
  癸水在月令得长生
  → 泄酉金生卯木
  → 乙木成格
```

**结论**：
```
长生 ≠ 简单的"生"
长生 = 暗藏的生机
    = 玄机所在
    = 可以改变格局
```
