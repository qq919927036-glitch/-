## 一、核心理念

### 1.1 理论基础
**正宗子平格局** vs **传统命理**

| 传统命理 | 子平格局 |
|---------|---------|
| 以日干强弱论身强身弱 | 以八字间架结构论命 |
| 取扶抑、病药、通关等 | 定格局、取用神、明喜忌 |
| 平衡用神论命体系 | 净阴净阳、去浊留清 |

### 1.2 核心精髓
**「阴阳」二字**
- 阴阳各不同途，各行各的轨道
- 阳有阳党，阴有阴党
- **净阴净阳**：去浊留清，不能混杂
- **战斗之患**：阴阳混杂无去留则产生征战

## 二、操作程序

### 2.1 三步法
```
八字到手
  ↓
1. 全面分析阴阳五行
   - 旺相休囚
   - 生克制化
  ↓
2. 定格取用
   - 定格局
   - 取用神
   - 明喜忌
  ↓
3. 断命
   - 富贵贫贱等级
   - 吉凶祸福
   - 六亲缘分
```

### 2.2 净阴净阳原理

**为什么要取清？**
```
命局去浊留清
  = 国家有英明领袖和强有力的中央集权
  = 农民锄草，庄稼茁壮成长
  = 阴阳不混杂，格用喜神清纯
```

**如何取清？**

例1：官杀混杂
- 去杀留官
- 或去官留杀

例2：财印伤食比劫混杂
- 同样需要去留取清
- 举一反三，十神皆然

**取清失败的结果：**
- 下命可决
- 没有富贵可言
- 阴阳征战而为凶

## 四、五行真假

### 4.1 真神 vs 假神

**定义：**
- 甲木为阳木，乙木为阴木
- 阴阳各不同途
- 如果甲木比乙木旺 → 甲木为真神（格），乙木为假神

**取格原则：**
```
甲木旺
  ↓
有制化乙木之神
  ↓
以甲木为格（真神）
以制化乙木之五行为用神
```

**真衰假旺：**
```
原局没有制化假神之神
反而有制化真神之神
  ↓
以假神为格
  ↓
真衰假旺，不是好命
```

### 4.2 大运流年的影响

**原局假神未去留：**
- 大运流年有去留
- 在这个大运流年里也可以春风得意

**真神当真：**
- 大运顺行：真神当真
- 大运逆行：假神当真（莫道假神有害处）

**滴天髓：**
> 「顺逆不齐也，不可逆者，顺其气势而已矣」

## 五、大运流年

### 5.1 关系
- **先天**：八字原局
- **后天**：大运（从原局引出）
- **命运**：先天生之，后天成之

### 5.2 冲战合好

#### 冲
- 冲去原命忌神者 → 喜
- 冲喜神 → 不吉
- 大运冲真神，流年冲假神 → 不吉

#### 战
- 大运旺真神，流年旺假神 → 战斗
- 真假同旺无去留 → 必然战斗
- 真假旺衰稍有差别 → 不会战斗

#### 和（合）
- 应当合的时候，合就吉
- 不应当合的时候，合就凶
- 合留喜神、合去忌神 → 吉
- 合留忌神、合去喜神 → 凶

#### 好
- 大运流年同为一气 → 吉利
- 同入喜神生旺之地 → 大吉
- 同入坏地 → 极不吉

### 5.3 流年特点
- **公共性**：任何人今年的流年都一样
- **天子地位**：年中天子
- 大运无法更改流年
- 一年休咎尤系乎岁

### 5.4 大运特点
- 掌握十年之久
- 人命成功与否的关键
- 有的人二三十年好运
- 有的人只有五年十年好运

## 七、命例分析

### 7.1 阴阳不能取清（下命）

**浙江湖州 童民联命造：**
```
男: 63, 11, 15, 寅时
癸 甲 丁 壬 癸 壬 辛 庚
卯 子 未 寅 亥 戍 酉 申
```

**现状：**
- 父母健在、无兄弟、有姐姐
- 光棍未婚
- 申运已丑年突然死亡

**分析：**
- 官印相生，日主不弱
- 但甲乙木混杂同旺，没有去留取清之神
- 正偏印阴阳交战，互不相让
- 印星为母，印星混乱为"母慈灭子"
- 结果：贫贱孤独夭寿

**传统理论判断：**
- 日主不弱
- 官印双全
- 杀化印生
- 应是富贵命

**现实：**
- 无官无财贫贱短寿

### 7.2 阴阳去留最清（上命）

**慈禧命造：**
```
乙 丁 乙 丁
未 未 丑 亥
```

**分析：**
- 地支亥未会出甲木
- 天干透出乙木
- 甲乙木比劫混杂
- 贵在月时干丁火透出，有力泄去乙木
- 留下甲木，甲木劫财非常清纯而且有力

**定格：**
- 格：甲木劫财
- 用神：丁火泄乙木，壬水生甲木
- 格神甲木：地位和富贵
- 壬水正印：管辖疆域
- 丁火食神：治理国家能力

**计量：**
- 格神甲木在月时两逢长生
- 壬水月时再居禄位
- 丁火清而透干
- 格神高大，用神有力，忌神有制
- **十二级富贵**（国家元首级别）
