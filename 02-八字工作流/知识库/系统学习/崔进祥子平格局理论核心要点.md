## 理论核心：净阴净阳

### 核心口诀

> **"净阴净阳，去浊留清"**

### 理论阐述

**阴阳不能混杂！**
- 阴有阴党，阳有阳党
- 阴阳不同途，各行各的轨道
- 阴不生克阴，阳不生克阳
- **阴阳混杂 = 战斗之患**

**为什么不能混杂？**
```
类比现实：
- 一个国家不能有两个政党同时执政
- 春秋战国、三国鼎立 = 内乱 = 国力衰败

八字同理：
- 阴阳混杂 = 群龙无首 = 大乱之象
- 必须去假留真，去浊留清
- 就像农民锄草，锄掉杂草，庄稼才能茁壮成长
```

**净阴净阳的操作：**
```python
if 官杀混杂:
    # 必须去杀留官，或去官留杀
    去假留真()

if 财印混杂:
    # 必须去假留真
    去假留真()

if 伤食混杂:
    # 必须去假留真
    去假留真()

if 比劫混杂:
    # 必须去假留真
    去假留真()
```

## 格局的本质

### 什么是格局？

**格局 = 五行的司令**

**类比：**
- 格局 = 公司董事长/总裁
- 用神 = 宰相和大臣
- 没有格局 = 群龙无首 = 难成好命

### 格局的作用

**格局代表：**
1. 命主的富贵等级
2. 命主的社会地位
3. 命主的事业成就
4. 命主的综合层次

**八字格局的高与低、大与小、好与差 = 命主本身的好与差**

### 传统命理的错误

**错误方法：**
- 以日干为主论身强身弱
- 取扶抑、病药、通关、中和、调候等寻找用神
- 五行数量的加减定强弱

**问题：**
- 都在日干强弱的框架束缚内转圈圈
- 很难达到准确和精确的程度
- 根本不可能探求到命理的细微深处

**子平格局的正确方法：**
- 舍弃以日干为主
- 改采八字间架结构及整体干支五行变化
- **定格局、取用神、明喜忌**

## 格神用神喜神忌神

### 格神（格局之神）

**定义：**
- 八字间架结构中的领导者
- 五行的司令
- 命造中最清纯、最有力的十神

**判断标准：**
1. 月令本气透干
2. 或司令分日透干
3. 或通过去假留真后留存

**举例：**
- 慈禧：甲木劫财为格（月时亥未会出甲木）
- 和珅：庚金正财为格（月令庚辛金禄旺之地）
- 毛泽东：特殊格局（需深度分析）

### 用神（维护格局之神）

**定义：**
- 维护格局的十神
- 对格神有利的五行

**用神的作用：**
1. 生扶格神
2. 制忌神
3. 去假神

**选取标准：**
- 看格局需要什么
- 能维护格局的就是用神

**举例：**
- 慈禧：用丁火泄乙木，壬水生甲木
- 和珅：用丁火克酉金

### 喜神（喜用之神）

**定义：**
- 对格局有利的十神
- 包括格神和用神

**喜神的特点：**
- 越旺越好
- 逢生旺之地大吉
- 遇克泄之地凶

### 忌神（忌讳之神）

**定义：**
- 对格局不利的十神
- 破坏格局的十神

**忌神的特点：**
- 越弱越好
- 必须制化
- 若制化不成，格局必败

**举例：**
- 万里浪：辛金真神、庚金假神混杂，无去留 = 大凶
- 录儿：癸水真神、壬水假神混杂，无去留 = 大凶

## 阴阳五行的真与假

### 真神与假神

**定义：**
- 真神：能为格，为主，为司令
- 假神：不能为格，为辅，为从属

**判断标准：**
```python
if 甲木比乙木旺:
    if 有制化乙木之神:
        return "甲木为真神，为格"
        return "乙木为假神，为忌"
    elif 有制化甲木之神:
        return "乙木为真神，为格（真衰假旺）"
        return "甲木为假神，为忌"
    else:
        return "阴阳混乱不清，战斗之患，大凶"
```

**真衰假旺：**
- 真神衰弱，假神旺
- 假神当真
- 不是好命
- 需大运流年去假留真

**举例：**
- 高际阳：丁火真衰（0级），丙火假旺
- 录儿：癸水真衰（1级），壬水假旺（7级）

### 真长生 vs 假长生

**阳干长生：**
- 阳干长生在地支的寅申巳亥
- 阳干之长生为真生
- 但寅申巳亥中藏有他神
- **必须天干有透，地支有会，才算真长生**

**阴干长生：**
- 阴干长生在地支的子午卯酉
- 阴干之长生带泄
- 如果遇有克泄阴干之神，儿能生母才算真生
- **惟有丁火长生算是真生**

**举例：**
- 丙火长生在寅：寅中有甲丙戊，天干透甲、地支会戌 = 真长生
- 乙木长生在午：乙木生午，因有酉金之克，得午火制酉护乙 = 真长生（儿能生母）

## 合的玄机

### 合的本质

**合 = 变化**

- 中草药的配方 → 配合产生变化
- 化学元素的合成 → 配合产生变化
- 干支五行的合 → 也会产生变化

### 合留 vs 合去

**合留：**
- 格神或用神遭到克泄
- 命局中有救助之神来合
- 此一合就是合留
- **喜神合留则有福**

**合去：**
- 格用之神在命局中并没有遭到克泄
- 而与他神作合
- 此一合即为合去
- **喜神被合去 = 英雄无用武之地**

**口诀：**
> **合留喜神，合去忌神就好**
> **合留忌神，合去喜神就会凶**

### 天干五合

- 甲己合
- 乙庚合
- 丙辛合
- 丁壬合
- 戊癸合

### 地支六合

- 子丑合
- 寅亥合
- 卯戌合
- 辰酉合
- 巳申合
- 午未合

### 合的实战案例

**案例1：李鸿章**
```
癸 甲 乙 己
未 寅 亥 卯
```
- 月时二干甲己一合
- 甲木合去
- 以乙木比肩为格
- 用癸水偏印生格
- **真假神在原命局去留得宜**

**案例2：某女士**
```
乙 己 乙 甲
巳 卯 亥 申
```
- 时上申庚克甲
- 甲木因合留仍然混杂在命局中
- 造成真假神不清之弊
- 壬水正印为忌神，与母亲沟通欠佳

## 冲战合好

### 冲

**冲去原命忌神者喜**
**冲喜神不吉**

**举例：**
- 大运冲忌神 → 喜
- 大运冲喜神 → 凶
- 流年冲忌神 → 喜
- 流年冲喜神 → 凶

### 战

**战斗 = 真假神同旺无去留**

**举例：**
- 大运旺真神，流年旺假神
- 流年凑成真假神混战
- 真假神旺衰稍有分别，就不会形成战斗

**战斗必然是：**
- 真假同旺
- 无去留
- 才会战斗

### 合

**合有宜不宜，合多不为奇**

**应当合的时候，合就吉**
**不应当合的时候，而合就凶**

### 好

**好 = 流年吉利**

**大运流年同为一气 = 同为吉利的好年运**

### 综合判断

**原局格用神，与大运流年顺逆同道，有体用者吉利**
**与岁运不和克害而无体用者不利**

**大运行格神之运，或行用神之运者吉利**
**大运行格神用神双全之运大吉**
**大运能去忌神，旺格用神也是上吉**

**原命不好能得大运相顾，也可以小草春风**

## 核心总结

### 子平格局的精髓

**1. 净阴净阳（最重要）**
- 阴阳不能混杂
- 去假留真，去浊留清
- 混杂 = 战斗 = 凶

- 格神 = 命主的富贵等级
- 用神 = 维护格局
- 喜神 = 越旺越好
- 忌神 = 必须制化

- 状态表 + 量化工具
- 精确计算富贵等级
- 精确计算寿命、六亲

- 合留喜神 = 吉
- 合去喜神 = 凶
- 合去忌神 = 吉
- 合留忌神 = 凶

**5. 大运流年**
- 冲战合好四字
- 原局格用神与大运流年顺逆同道 = 吉
- 与岁运不和 = 凶
