## 📋 格局成功的条件

### 成功条件1：格神清纯

**定义：**
- 格神没有混杂
- 或者混杂但有去留

**判断标准：**
```python
if 格神无混杂:
    return "格局纯，成格"
elif 格神有混杂:
    if 有去留之神:
        return "去假留真，格局纯，成格"
    else:
        return "混杂不清，格局败，不成格"
```

**举例：**
- ✅ 慈禧：甲乙木混杂，但有丁火去乙留甲 → 成格
- ❌ 万里浪：官杀混杂，无去留 → 不成格

---

### 成功条件2：用神有力

**定义：**
- 用神在月令得地
- 用神透干有力
- 用神有生扶

**判断标准：**
```python
用神月令级数 = 十二长生[用神][月令]
用神是否透干 = 用神 in 天干

if 用神月令级数 >= 3 and 用神是否透干:
    return "用神有力，格局成"
elif 用神月令级数 >= 2:
    return "用神偏弱，格局成但层次不高"
else:
    return "用神无力，格局败"
```

**举例：**
- ✅ 慈禧：壬水月时再居禄位，丁火清而透干 → 用神有力
- ❌ 南京交警：甲木坐绝，为何还能富贵？因甲木合留在局

---

### 成功条件3：忌神受制

**定义：**
- 忌神弱
- 忌神有制化
- 忌神不入墓

**判断标准：**
```python
忌神月令级数 = 十二长生[忌神][月令]
有制忌神 = 检查食伤官杀印()

if 忌神月令级数 <= 2:
    return "忌神弱，格局成"
elif 有制忌神:
    return "忌神受制，格局成"
else:
    return "忌神无制，格局败"
```

**举例：**
- ✅ 和珅：酉金被丁火克 → 忌神受制
- ❌ 录儿：壬水假神无制 → 忌神无制

---

### 成功条件4：格神生旺

**定义：**
- 格神在月令得地
- 格神有长生帝旺
- 格神有暗拱暗会

**判断标准：**
```python
格神月令级数 = 十二长生[格神][月令]
格神总级数 = sum(十二长生[格神][所有地支])

if 格神月令级数 >= 4:
    return "格神生旺，大贵"
elif 格神月令级数 >= 2:
    return "格神有根，中贵"
elif 格神总级数 >= 10:
    return "格神暗拱暗会，大贵"
else:
    return "格神偏弱，小贵或不成格"
```

**举例：**
- ✅ 慈禧：甲木月时两逢长生 → 格神生旺
- ❌ 高际阳：丁火月令死地（0级）→ 格神极弱

## 📋 格局失败的条件

### 失败条件1：混杂不清

**定义：**
- 阴阳混杂
- 同十神混杂
- 无法去假留真

**判断标准：**
```python
if 阴阳混杂 and 无去留:
    return "混杂不清，大凶"
if 官杀混杂 and 无去留:
    return "官杀混杂，大凶"
if 财印混杂 and 无去留:
    return "财印混杂，大凶"
# ... 其他混杂
```

**举例：**
- ❌ 童民联：甲乙木混杂同旺，无去留 → 贫贱短寿

---

### 失败条件2：用神无力

**定义：**
- 用神在月令不得地
- 用神不透干
- 用神受制

**举例：**
- ❌ 某女命：用神甲木坐绝，因合留仍然混杂

---

### 失败条件3：忌神无制

**定义：**
- 忌神大旺
- 忌神无制化
- 忌神入库

**举例：**
- ❌ 录儿：壬水假神大旺，无制

---

### 失败条件4：格神极弱

**定义：**
- 格神在月令绝地
- 格神无根
- 无法从格

**举例：**
- ❌ 万里浪：甲木在申月绝地（0级），无印比，不从格

---

## 🎯 格局层次判断

### 层次1：上等格局（9-12级）

**条件：**
- 格神清纯
- 用神有力
- 忌神受制
- 格神生旺
- 无刑冲破害

**举例：**
- 慈禧：12级（帝王级）
- 和珅：11级（首富级）

---

### 层次2：中等格局（5-8级）

**条件：**
- 格神清纯
- 用神偏弱
- 忌神受制
- 格神有根

**举例：**
- 枣庄市财政局长：7级（副厅级）
- 南京交警队长：6级（处级）

---

### 层次3：下等格局（1-4级）

**条件：**
- 格神偏弱
- 用神无力
- 忌神有制但不够

**举例：**
- 小康、富裕之家

---

### 层次4：不成格（0级）

**条件：**
- 混杂不清
- 用神无力
- 忌神无制
- 格神极弱

**举例：**
- 万里浪：官杀混杂，无去留
- 录儿：真假神混杂，无去留
- 童民联：甲乙木混杂，无去留
