## 📋 混杂的类型

### 类型1：阴阳混杂（最严重）

**定义：**
- 同五行阴阳同透
- 例如：甲乙木混杂、丙丁火混杂

**判断标准：**
```python
if 甲在天干 and 乙在天干:
    return "甲乙木混杂"
if 丙在天干 and 丁在天干:
    return "丙丁火混杂"
if 戊在天干 and 己在天干:
    return "戊己土混杂"
if 庚在天干 and 辛在天干:
    return "庚辛金混杂"
if 壬在天干 and 癸在天干:
    return "壬癸水混杂"
```

**严重程度：** ⭐⭐⭐⭐⭐

**去留方法：**
- 甲乙木混杂 → 用火泄乙木，或用金克甲木
- 丙丁火混杂 → 用土泄丙火，或用水克丁火
- 庚辛金混杂 → 用火克庚金，或用水泄辛金
- 壬癸水混杂 → 用木泄壬水，或用土克癸水
- 戊己土混杂 → 用金泄戊土，或用木克己土

**举例：**
- ✅ 慈禧：甲乙木混杂，用丁火泄乙木 → 去假留真
- ❌ 童民联：甲乙木混杂，无去留 → 混杂不清，大凶

---

### 类型2：官杀混杂

**定义：**
- 正官和七杀同时透干
- 例如：甲木见庚辛金

**判断标准：**
```python
if 正官透干 and 七杀透干:
    return "官杀混杂"
```

**严重程度：** ⭐⭐⭐⭐⭐

**去留方法：**
- 去杀留官：用伤官制杀，或用印星化杀
- 去官留杀：用食神合官

**举例：**
- ✅ 慈禧：官杀混杂，用丁火去杀留官 → 大贵
- ❌ 万里浪：官杀混杂，无去留 → 大凶

---

### 类型3：财印混杂

**定义：**
- 正财偏财混杂
- 正印偏印混杂

**判断标准：**
```python
if 正财透干 and 偏财透干:
    return "财星混杂"
if 正印透干 and 偏印透干:
    return "印星混杂"
```

**严重程度：** ⭐⭐⭐⭐

**去留方法：**
- 财星混杂 → 用比劫去财
- 印星混杂 → 用财星去印

```python
if 食神透干 and 伤官透干:
```

### 类型5：比劫混杂

**定义：**
- 比肩和劫财同时透干

**判断标准：**
```python
if 比肩透干 and 劫财透干:
```

**严重程度：** ⭐⭐⭐⭐

## 🎯 识别流程

### 步骤1：检查天干混杂

```python
# 检查所有天干
for 五行 in ['木', '火', '土', '金', '水']:
    阳干 = get_yang_stem(五行)
    阴干 = get_yin_stem(五行)

    if 阳干 in 天干 and 阴干 in 天干:
        print(f"{五行}阴阳混杂！")
        检查去留()
```

### 步骤2：检查十神混杂

```python
# 检查十神混杂
if 正官 in 天干 and 七杀 in 天干:
    print("官杀混杂！")
    检查去留()

if 正财 in 天干 and 偏财 in 天干:
    print("财星混杂！")
    检查去留()

# ... 其他十神混杂
```

### 步骤3：检查去留之神

```python
def 检查去留():
    # 检查是否有去留之神
    去留方式 = []

    # 1. 伤官制杀
    if 伤官 in 天干:
        去留方式.append("伤官制杀")

    # 2. 食神合官
    if 食神 in 天干:
        去留方式.append("食神合官")

    # 3. 印星化杀
    if 印星 in 天干:
        去留方式.append("印星化杀")

    # 4. 财星泄伤
    if 财星 in 天干:
        去留方式.append("财星泄伤")

    if 去留方式:
        print(f"有去留：{去留方式}")
        return True
    else:
        print("无去留！混杂不清！")
        return False
```
