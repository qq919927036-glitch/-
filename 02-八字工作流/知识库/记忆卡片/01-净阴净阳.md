# 记忆卡片：净阴净阳

## 核心概念

**净阴净阳**：去浊留清，不能混杂

## 记忆口诀

```
阴阳混杂要征战
去浊留清保平安
官杀混杂去一留
财印伤食亦同理
```

## 理解方法

### 形象比喻

**国家比喻**
```
命局去浊留清
= 国家有英明领袖
= 强有力的中央集权
= 政党执政，异党削弱
= 繁荣昌盛
```

**农耕比喻**
```
去浊留清
= 农民锄草
= 锄掉杂草
= 庄稼茁壮成长
```

### 历史比喻

**内乱时期（不好）**
- 春秋战国
- 楚汉争雄
- 三国鼎立
- 安史之乱
→ 群龙无首，大乱之象

**统一时期（好）**
- 单一政党执政
→ 阴阳清纯，格局高

## 操作方法

### 什么时候需要去留？

**十神混杂都需要取清：**
- 官杀混杂 → 去杀留官 / 去官留杀
- 财印混杂 → 去一留一
- 伤食混杂 → 去一留一
- 比劫混杂 → 去一留一

### 如何去留？

**方法1：生克制化**
```
甲木真神，乙木假神
→ 有制化乙木之神
→ 以甲木为格
→ 以制化乙木之五行为用神
```

**方法2：合去合留**
```
丙丁混杂
→ 丙辛一合，丙火合去
→ 丁火独旺而清纯
```

## 命例验证

### 好命（去留最清）

**慈禧**
```
乙 丁 乙 丁
未 未 丑 亥
```
- 甲乙木混杂
- 贵在丁火泄去乙木
- 留下甲木清纯
- **十二级富贵**

### 坏命（不能取清）

**童民联命造**
```
癸 甲 丁 壬
卯 子 未 寅
```
- 甲乙木混杂同旺
- 没有去留取清之神
- 正偏印阴阳交战
- **贫贱夭寿**

## 关键金句

> "命局的格用喜神要取清而不能混杂，混杂就会引起征战"

> "去浊留清，净阴净阳的原理，是子平格局理论的精髓"

> "如果原局混杂无法去留取清者，则下命可决"
