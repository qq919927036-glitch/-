# 记忆卡片：十二长生

## 核心概念

**十二长生**：量化计算工具
- 计算富贵贫贱等级
- 计算六亲数量
- 计算寿命长短
- 计算教育程度

## 口诀（必背）

```
长生四子中旬半, 沐浴一双保吉祥。
冠带临官三子位, 旺中五子自成行。
衰中二子病中一, 死中至老没儿郎。
绝中取养他人子, 入墓之时命夭亡。
胎中头女有姑娘, 养中三子只留一。
男女宫中仔细详。
```

### 记忆方法

**前4个（吉）**
- 长生：新生，4个孩子
- 沐浴：洗澡，2个孩子
- 冠带：冠礼，3个孩子
- 临官：当官，3个孩子
- 旺：旺盛，5个孩子

**中间2个（下坡）**
- 衰：衰弱，2个孩子
- 病：生病，1个孩子

**后4个（凶）**
- 死：死绝，0个孩子
- 绝：绝境，养别人孩子
- 墓：入墓，夭亡
- 胎：怀胎，生女孩
- 养：养育，3留1

## 应用实例

### 例1：慈禧命造

```
乙 丁 乙 丁
未 未 丑 亥
```

**计算：**
- 甲木在亥月 → 长生（4级）
- 甲木在亥时 → 长生（4级）
- 壬水在亥月 → 禄（临官级）
- 壬水在亥时 → 禄（临官级）
- 丁火清而透干

**结果：**
- 格神甲木在月时两逢长生
- 壬水月时再居禄位
- **十二级富贵**

### 例2：毛泽东命造

```
癸 甲 丁 甲
巳 子 酉 辰
```

**计算：**
- 地支拱未库（6级）
- 亥水长生（4级）
- 甲木坐辰（2级）

**结果：**
- 6 + 4 + 2 = **12级**
- **帝王之命**

## 关键要点

### 1. 表面含义 vs 实际含义

**表面：**
- 论子女数量

**实际：**
- 五行十神尺度含量
- 计算富贵等级

### 2. 独特价值

> "除了表明阴阳五行的属性和状态之外，还是我们计算人命富贵贫贱等级高低和六亲数量缘分的量化工具"

### 3. 精确掌握的重要性
