# 记忆卡片：冲战合好

## 核心概念

**冲战合好**：大运流年的四种关系
- 冲
- 战
- 合（和）
- 好

## 一、冲（冲撞）

### 定义
- 大运流年与原局发生冲撞

### 判断原则

**冲去忌神 = 吉**
```
大运/流年冲去原局忌神
→ 喜
→ 忌神被制，格局转清
```

**冲去喜神 = 凶**
```
大运/流年冲去原局喜神
→ 不吉
→ 喜神受伤，格局破败
```

**大运冲真神，流年冲假神 = 凶**
```
真神被冲，假神也被冲
→ 混杂不清
→ 不吉
```

### 命例

**例：某命造**
```
原局：庚金为格，用丙火
大运：冲去丙火
→ 不吉
→ 用神被伤
```

---

## 二、战（战斗）

### 定义
- 大运流年与原局形成征战

### 判断原则

**真假同旺 = 战**
```
大运旺真神
流年旺假神
→ 真假神混战
→ 必然战斗
```

**旺衰有差别 = 不战**
```
真神旺，假神衰
→ 不形成战斗
→ 可以共存
```

**核心要求：**
> "真假神旺衰稍有分别，就不会形成战斗"

**战斗条件：**
> "战斗必然是真假同旺无去留才会战斗"

### 危害

**阴阳征战**
```
命局战斗
→ 内乱
→ 国力衰败
→ 民不聊生
→ 严重时灭亡
```

**历史比喻**
- 春秋战国
- 楚汉争雄
- 三国鼎立
- 安史之乱

---

## 三、和（合）

### 定义
- 大运流年与原局发生相合

### 判断原则

**应当合的时候，合 = 吉**
```
该合的时候
→ 合就吉
→ 格局清纯
```

**不应当合的时候，合 = 凶**
```
不该合的时候
→ 合就凶
→ 破坏格局
```

**具体判断**

| 类型 | 结果 |
|------|------|
| 合留喜神 | 吉 |
| 合去忌神 | 吉 |
| 合留忌神 | 凶 |
| 合去喜神 | 凶 |

### 命例

**例：董建华命造**
```
丁 丙 乙 辛
丑 午 未 巳
```

**关键：**
- 丙辛一合
- 丙火合去
- 丁火独旺
- **格局转清**

---

## 四、好（吉利）

### 定义
- 大运流年同为一气，皆为吉利

### 判断原则

**同气 = 吉**
```
大运流年同为一气
→ 吉利
→ 好年运
```

**同入喜神生旺地 = 大吉**
```
大运流年同入喜神生旺之地
→ 非常大吉
→ 事业兴旺
```

**同入坏地 = 极不吉**
```
大运流年同入坏地
→ 极不吉
→ 灾祸临身
```

---

## 综合判断

### 总原则

> "原局格用神，与大运流年顺逆同道，有体用者吉利"

> "与岁运不和克害而无体用者不利"

### 吉利情况

1. **行格神之运** → 吉利
2. **行用神之运** → 吉利
3. **行格神用神双全之运** → 大吉
4. **行运能去忌神，旺格用神** → 上吉
5. **原命不好能得大运相顾** → 小草春风

### 不利情况

1. **行运冲喜神** → 不利
2. **行运合去喜神** → 不利
3. **行入坏大运流年** → 极不吉
4. **真假神混杂战斗** → 凶险

---

## 其他要素

除了冲战合好，还需要考虑：

### 刑
- 刑喜神 → 不吉（要有救应）
- 刑忌神 → 吉

### 克
- 克喜神 → 要有救应神
- 克忌神 → 吉

### 会
- 会留喜神 → 吉
- 会去忌神 → 吉
- 会留忌神 → 凶
- 会去喜神 → 凶
