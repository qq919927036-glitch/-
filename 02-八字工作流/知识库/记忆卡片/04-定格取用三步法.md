# 记忆卡片：定格取用三步法

## 核心流程

```
八字到手
  ↓
【第一步】全面分析
  ↓
【第二步】定格取用
  ↓
【第三步】明喜忌
  ↓
开始断命
```

---

## 第一步：全面分析

### 1.1 分析内容

**五行分析：**
- 旺相休囚
- 生克制化
- 刑冲会合

**十神分析：**
- 察旺衰
- 明变化
- 辨真假

### 1.2 分析目的

**找到最强最清纯的十神**
```
不是看月令
不是看日干强弱
而是看整体结构
```

---

## 第二步：定格取用

### 2.1 定格局

**什么是格？**
```
格 = 命局中最强、最清纯、最有力的十神
= 真神
```

**如何定？**

**原则1：净阴净阳**
```
十神混杂？
→ 需要去留取清
→ 谁清纯谁为格
```

**原则2：察旺衰**
```
谁最旺？
→ 旺者为真神
→ 可以为格
```

**原则3：明变化**
```
有合化？
→ 合去的是假神
→ 合留的是真神
```

**原则4：辨真假**
```
甲木为真神，乙木为假神
→ 有制化乙木之神
→ 以甲木为格
```

### 2.2 取用神

**什么是用？**
```
用 = 辅佐格神的五行
= 制化忌神的神
= 生扶格神的神
```

**如何取？**

**情况1：官杀混杂**
```
正官为格，七杀混杂
→ 用偏印泄七杀
→ 或用食神制杀
```

**情况2：正官弱**
```
正官为格，但弱
→ 用正财生官
```

**情况3：正官遇伤官**
```
正官为格，伤官克
→ 用正财解救
```

**情况4：七杀格**
```
七杀为格，正官混杂
→ 用伤官制官
→ 或用正印泄官
```

---

## 第三步：明喜忌

### 3.1 什么是喜神

**喜神 = 对格神有利的五行**

### 3.2 什么是忌神

**忌神 = 对格神不利的五行**

### 3.3 如何判断

**有利于格神 = 喜**
```
生扶格神 → 喜
制化忌神 → 喜
保护格神 → 喜
```

**不利于格神 = 忌**
```
克制格神 → 忌
泄耗格神 → 忌
混杂格神 → 忌
```

### 3.4 重要原则

> "忌神被克制越净八字格局越高"

> "格神高大，用神有力，是大富大贵"

## 命例演示

### 例：董建华

```
丁 丙 乙 辛
丑 午 未 巳
```

**第一步：全面分析**
- 地支巳午未会丙丁火方
- 年时巳丑会庚金正官局
- 时干透辛金七杀
- 丙丁火混杂

**第二步：定格取用**
- 庚金正官因会局而有力
- 丙辛一合，丙火合去
- 丁火克尽辛金
- **格：庚金正官**
- **用：丁火食神制杀**

**第三步：明喜忌**
- **喜：丁火**（制杀）
- **忌：丙火**（混杂，已被合去）

**结果：**
- 格局清纯
- 十级富贵

## 关键金句

### 定格原则
> "要知人命，必定格局；要定格局，必晓阴阳"

> "令上寻真聚的真，假神休要乱真神"

> "真神得用平生贵，用假终为碌碌人"

### 取用原则
> "道有体用，不可一端论也"

> "格神忌而用神喜"

### 判断标准
> "大贵之人必然有大贵之处"

## 常见错误

### 错误1：只看月令
```
❌ 甲木日干生于酉月 = 正官格
✅ 全面分析，最强最清纯者为格
```

### 错误2：日干强弱论命
```
❌ 日干强还是弱？
✅ 格神是什么？用神是什么？
```

### 错误3：不讲净阴净阳
```
❌ 官杀混杂也可以
✅ 必须去留取清
```
