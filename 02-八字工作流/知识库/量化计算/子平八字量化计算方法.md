## 核心概念

**量化计算**是子平格局的独特之处：
```
传统命理：大概说"富贵贫贱"
子平格局：精确计算"十二级富贵"
```

**可计算的方面：**
1. 富贵等级（财富、官贵）
2. 六亲数量（父母、兄弟、夫妻、子女）
3. 寿命长短
4. 教育程度
5. 相貌美丑

## 一、计算基础

### 1.1 单位换算

```
1级 = 10分 = 10年（寿命）
1分 = 1年（寿命）
```

### 1.2 级vs分的区别

| 级 | 分 | 适用场景 |
|---|---|----------|
| **级** | 完美状态 | 喜神无刑冲克害混杂 |
| **分** | 有瑕疵 | 喜神有刑冲克害混杂 |

**规则：**
> 喜神有刑冲破坏混杂以分计算
> 没有刑冲破坏混杂者以级计算
> 暗拱暗会也可加入计算

### 1.3 计算公式

```
原局所得级数
  + 暗拱暗会级数
  - 克泄刑害扣除级数
  = 实际级数
```

### 2.2 特殊规则

**库地规则（戌、丑、辰、未）**
- 有混杂者 → 无分
- 有克泄者 → 宜闭库，为级
- 无克泄者 → 宜开，以库神清得用

**绝处逢生**
```
亥月本为绝（0级）
但若地支得寅合留甲木
→ 甲木生丙火
→ 以级计算（不为0级）
```

**同旺混杂**
```
午月帝旺（6级）
但午月与丁火同量
原局有丁火混杂无法去除
→ 不以丙火计，以丁火计
```

## 四、六亲计算方法

### 4.1 基本规则

> "父母、兄弟、夫妻、子女统称为六亲"
> "以宫位和六亲星，配合原局计算"
> "人数多少以一级为一人，缘份长短以一级为十年"

### 4.2 父母

**基本原则：**
```
正印为母
偏财为父
```

**变通规则：**
1. 原命不见正印偏财
   → 有偏印正财为喜神，可作父母星

2. 财印星都是忌神
   → 找克泄财印之神为父母星

3. 财印星都是忌神，宫位也是忌神
   → 父母早亡，幼失怙恃

### 4.3 兄弟

**基本原则：**
```
比劫为兄弟星
兄弟宫位：年月（男）/ 月时（女）
```

**判断方法：**
1. 比劫为喜神，宫位为喜
   → 兄弟相亲相爱

2. 一喜一忌
   → 关系平常

3. 宫位与兄弟星为忌神
   → 反目成仇

**验证时辰：**
> "要是命主的兄弟数量有差错，富贵等级也不相符时"
> "即应该将所报时辰上下推一推"

### 4.4 夫妻

**男命：**
```
官杀为夫
正偏财为妻
```

**女命：**
```
官杀为丈夫
食神伤官为夫（官杀为忌时）
```

**夫妻星位置：**
- 居年月 → 年龄偏大
- 居日时 → 年龄偏小

**多见则忌：**
> "要是夫妻星在原命局要生旺，但不喜多见"
> "要多见的话，就会多夫多妻了"

**特殊情况：**
- 财星混乱 → 用克制财星的比劫为妻
- 财星为忌神不见 → 以生财的食伤为妻
- 财星为忌神，食伤遭刑克 → 倒插门女婿，严重则无妻

### 4.5 子女

**基本原则：**
```
食神伤官为子女星
```

**古人错误：**
> "古人和古书中有以官星七杀看子女的，这不正确"

**各种情况：**

1. 食伤为喜神但很弱，有旺印克，无财星救助
   → 无子女

2. 食伤混乱或为忌神
   → 以印星看子女

3. 食伤混乱，有财星泄化
   → 以财为子女星

4. 财星又混乱无比劫制化
   → 没有子女

5. 食伤混乱用财，财星为忌神
   → 以官杀喜神为子女星（五行流通法则）

## 五、计算实例

### 例1：慈禧（十二级富贵）

**八字：**
```
乙 丁 乙 丁
未 未 丑 亥
```

**计算：**
1. 格神：甲木劫财
   - 月时两逢长生 → 4+4=8级
2. 用神：壬水正印
   - 月时再居禄位 → 4+4=8级
3. 丁火食神
   - 清而透干 → 已计入

**总计：**
```
地支拱未库 → 6级
亥水长生 → 4级
甲木坐辰 → 2级
----------
共计：6+4+2 = 12级
```

**结果：** 十二级富贵，帝王之命

---

### 例2：毛泽东（十二级富贵）

**八字：**
```
癸 甲 丁 甲
巳 子 酉 辰
```

**计算：**
1. 格神：癸水七杀
   - 月令建禄 → 已计入计算
2. 暗拱：
   - 地支拱未库 → 6级
   - 亥水长生 → 4级
   - 甲木坐辰 → 2级

**总计：** 6+4+2 = **12级**

## 六、核心要点

### 6.1 计算三要素

1. **原局级数**：格神、用神在原局的级数
2. **暗拱暗会**：地支拱合的级数
3. **克刑扣除**：忌神刑克需要扣除的级数

### 6.2 关键原则

> "原局所得级数，加上暗拱暗会的级数"
> "若有克泄刑害忌神，虽在原局有制化，亦需扣除"
> "而后所得的，才是实际级数"

### 6.3 注意事项

1. **真假神影响**
   - 真神可计算
   - 假神需扣除

2. **混杂影响**
   - 混杂无去留 → 不以级计算
   - 去留得宜 → 以级计算

3. **刑冲影响**
   - 有刑冲克害 → 降为分计算
   - 无刑冲克害 → 保持级数
