## 核心概念

**暗拱暗会**是量化计算的重要组成部分：
```
原局级数 + 暗拱暗会级数 - 扣除级数 = 实际级数
```

> "暗拱暗会也可加入计算，一级是十分"
> "有刑冲克害混杂者为分，无克害者为级"

---

## 一、什么是暗拱暗会

### 1.1 定义

**暗拱**：地支之间暗中形成的相合关系
**暗会**：地支之间暗中形成的相会关系

**特点：**
- 不直接相邻
- 隐藏在命局中
- 需要仔细查找
- 可以增加级数

### 1.2 作用

1. **增加级数**
   - 暗拱暗会所得的级数可加入计算
   - 这是其他学派没有的方法

2. **改变五行状态**
   - 让假神变为真神
   - 让无力变为有力

3. **影响命局层次**
   - 决定格局高低
   - 决定富贵等级

## 三、暗拱暗会类型

### 3.1 拱局

**定义：** 相隔两个地支的关系

**公式：**
```
地支A 与 地支A+3 地支A+3 暗拱
```

**示例：**
```
子 → 与 卯、辰 暗拱
丑 → 与 辰、巳 暗拱
寅 → 与 巳、午 暗拱
```

**12组拱局：**
- 子拱辰（子辰中间夹寅卯）
- 丑拱巳
- 寅拱午
- 卯拱未
- 辰拱申
- 巳拱酉
- 午拱戌
- 未拱亥
- 申拱子
- 酉拱丑
- 戌拱卯
- 亥拱寅

### 3.2 暗会

**定义：** 通过相合、相冲、三会等暗中形成的会局

**类型：**

**类型1：通过六合**
```
示例：寅 丑 卯
     ↑ 合 ↑
     卯亥 → 暗会亥
```

**类型2：通过三会**
```
示例：卯 辰 巳
        ↑
        卯未 → 暗会未
```

**类型3：通过相冲**
```
示例：子 午 酉
           ↑
           子冲午，午戌暗会戌
```

## 四、查找方法

### 4.1 全面扫描法

**第一步：列出地支**
```
八字：甲 乙 丙 丁
地支：子 丑 寅 卯
```

**第二步：检查所有组合**
- 检查是否三合
- 检查是否六合
- 检查是否相冲
- 检查是否三会

**第三步：找出暗拱暗会**
- 拱局查找
- 隔支查找
- 拱层查找

### 4.2 系统查找法

**按五行查找：**

1. **水局暗拱暗会**
   - 查看申子辰
   - 查看亥子丑
   - 查看相关地支

2. **木局暗拱暗会**
   - 查看亥卯未
   - 查看寅卯辰
   - 查看相关地支

3. **火局暗拱暗会**
   - 查看寅午戌
   - 查看巳午未
   - 查看相关地支

4. **金局暗拱暗会**
   - 查看巳酉丑
   - 查看申酉戌
   - 查看相关地支

5. **土局暗拱暗会**
   - 查看辰戌丑未
   - 查看相关地支

## 五、实例演示

### 例1：慈禧命造

**八字：**
```
乙 丁 乙 丁
未 未 丑 亥
```

**暗拱暗会分析：**

1. **未丑拱**
   - 未 和 丑 中间夹 未亥巳
   - 暗拱：亥、巳

2. **亥未会木局**
   - 亥 + 未 → 暗会卯（亥卯未三合）

3. **巳亥冲**
   - 年支巳，时支亥
   - 巳亥暗拱：未、戌

4. **综合计算**
   - 拱未库：6级
   - 亥水长生：4级
   - 甲木坐辰：2级

### 例2：毛泽东命造

**八字：**
```
癸 甲 丁 甲
巳 子 酉 辰
```

**暗拱暗会分析：**

1. **子辰会水局**
   - 子 + 辰 → 暗会申（申子辰三合）

2. **巳酉会金局**
   - 巳 + 酉 → 暗会酉（巳酉丑三合）

3. **拱未库**
   - 辰戌丑未四支
   - 暗拱未库

4. **综合计算**
   - 拱未库：6级
   - 亥水长生：4级
   - 甲木坐辰：2级

2. **六合**（较明显）

4. **拱局**（需要找）
5. **暗会**（最隐蔽）

## 七、常见错误

### 7.1 忽略暗拱暗会

**错误：**
```
❌ 只看明显的三合六合
❌ 忽略了隔支关系
❌ 漏掉了隐藏的级数
```

**正确：**
```
✅ 系统扫描所有地支关系
✅ 仔细查找暗拱暗会
✅ 确保不遗漏级数
```

### 7.2 混淆刑冲与暗拱

**错误：**
```
❌ 把相冲当作暗拱
❌ 把刑克当作相会
```

**正确：**
```
✅ 相冲：子午冲、丑未冲等
✅ 暗拱：隔两个地支的关系
✅ 明确区分不同关系
```

## 总结

### 核心要点

1. **系统扫描**
   - 不遗漏任何关系
   - 三合、六合、三会、相冲都要查

2. **重点查找**
   - 拱局要仔细找
   - 隔支关系容易漏

3. **验证计算**
   - 找到后要验证
   - 确认是否真的有这个关系

4. **持续练习**
   - 多分析实例
   - 逐步熟练掌握
