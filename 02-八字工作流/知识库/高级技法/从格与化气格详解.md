## 核心概念

### 什么是从格？

> "从格是日干弱无气、四柱五行克泄日干、见他神生旺弃身从命"

**基本定义：**
- 日干衰弱无气
- 四柱尽是克泄交加
- 无印比生扶
- 不得不弃命从人之象

**可从之神：**
- ✅ 食神
- ✅ 伤官
- ✅ 正财
- ✅ 偏财
- ✅ 正官
- ✅ 七杀

**不可从之神：**
- ❌ 正偏印
- ❌ 比劫

## 一、从格的判断

### 1.1 真从vs假从

#### 真从格

**条件：**
```
1. 日干弱无气
2. 四柱克泄交加
3. 无印比生扶
4. 从神生旺
5. 无刑冲克泄和混杂
6. 或虽有刑害混杂而有救应
```

**结果：**
> "真从之家有几人、假从亦可发其身"
> "从得真者只论从、从神又有吉和凶"

#### 假从格

**条件：**
```
1. 从神有根（日干有印比）
2. 原能去根者
3. 有克害从神之神
```

**结果：**
- 原局为假从
- 行去根和从神生旺之运 → 亦能大发
- 即所谓：**"孤儿异性能出类"**

### 1.2 从格的六大格局

滴天髓云：
> "一出门来只见儿、吾儿成气构门闾"
> "从儿不论身强弱、只要吾儿又见儿"

**六格：**
1. 从食神格
2. 从伤官格
3. 从正财格
4. 从偏财格
5. 从正官格
6. 从七杀格（从偏官格）

## 二、从格的取用原则

### 2.1 核心原则

> "从格忌混浊，更怕日干生旺之运"

**混浊处理：**
- 从正不得混偏
- 从偏不得混正
- 有混杂必当有去留制化之神

### 2.2 取用方法

| 情况 | 用神 |
|------|------|
| 从格有混杂 | 去留取清之神 |
| 从神有克泄 | 去克泄之神 |
| 从官杀混杂 | 食伤制官杀或印泄官杀 |
| 从财混杂 | 官杀泄财或比劫夺财 |

### 2.3 从格的忌讳

**三大忌讳：**
1. **忌根多**
   - 根即日干之印比
   - 宜有制化印比之神

2. **忌克害从神**
   - 原局有克害从神之神
   - 必须制化之

3. **忌正偏混杂无制**
   - 正偏混杂
   - 有根无制
- 克害从神无解
- **皆谓之假从**

## 三、从格各论

### 3.1 从儿格（从食伤）

**核心理论：**
> "从儿不论身强弱，只要我儿又遇儿"

**特点：**
- 日干克泄交加
- 四柱尽是食伤财官
- 日主无一点生扶之气
- 不得不从

**从食伤格：**
- 忌见：官杀、印
- 不忌：比劫
- 局中见财为上
- 运行财地最佳
- 食伤次之
- 比劫之地又次之
- 逢官杀印乡必凶

**为什么从儿格不忌比劫？**
> "是以不忌比劫、因为能生食伤为喜"

### 3.2 从财格

**判断：**
- 财官食伤大旺
- 日干无气

**取用：**
- 从正财混偏财 → 用七杀泄偏财
- 从偏财混正财 → 用官杀泄正财或比劫夺正财

**特点：**
- 从财格最喜财旺
- 忌印比生身

### 3.3 从官杀格

**判断：**
- 官杀大旺
- 日干无气

**取用：**
- 从正官混七杀 → 用伤官制七杀或正印泄七杀
- 从七杀混正官 → 用食神制正官或偏印泄正官

**特点：**
- 从官杀格最喜官杀旺
- 忌食伤伤官制官杀
- 忌印比生身

## 四、化气格

### 4.1 什么是化气格？

> "化气格是日干弱，四柱干支无气，与月干或时干作合，而化为他神"

### 4.2 化气的条件

#### 真化条件

**必须同时满足：**
```
1. 日干弱无气
2. 四柱克泄交加
3. 无印比生
4. 日干与月干或时干作合
5. 天干透出化神及喜神
6. 地支成方成局或与化气相生
```

#### 假化条件

**满足以下之一：**
```
1. 化气之神有克泄，而有救应之神
2. 化气之神有混杂，有去留之神
3. 日干原局有气，有克泄日干之神
```

**结果：**
- 假化行真化之运岁 → 亦能富贵

### 4.3 化气不成的情况

**以下情况化气不成，以正格论：**
```
1. 日干在四柱干支有气
2. 原局又无化神
3. 日干虽与时干或月干作合，也不能化
4. 月干与时干是同一字同时合日干谓之破
```

### 4.4 天干五合化气

**五合化气：**
```
甲己合化土
乙庚合化金
丙辛合化水
丁壬合化木
戊癸合化火
```

**化气格的要求：**
- 日干与月干或时干作合
- 四柱皆见化神同五行
- 始能成化

**合而不化：**
- 日干与他神作合
- 柱中有克神破合
- 化气不成
- 只能论为偏好

---

## 五、从格与化气的区别

### 5.1 相同点

- 都要求日干弱无气
- 都要求无印比生扶
- 都忌混杂
- 都需要去留得宜

### 5.2 不同点

| 从格 | 化气格 |
|------|--------|
| 不需要合 | 必须有合 |
| 从财官食伤 | 化为他神 |
| 有六格 | 有五化 |
| 弃命从人 | 合化变质 |

## 六、经典命例分析

### 例1：马英九（从儿格）

**八字：**
```
庚 癸 己 辛
寅 未 酉 未
1950年五月二十九日未时
```

**定格取用：**
- 月日地支卯木双排（注：此处可能有误，原命是未月）
- 辛金坐绝又有午中丁火之制
- 丁火护乙，为"儿能救母泄天机"
- **格神：以从乙木食神为格**
- **用神：以丁火偏财为用神**

**判断：**
- 格旺月令有丁火偏财之护
- 是为老板求财之人
- 官杀为忌，不能求官

**婚姻：**
- 婚姻不吉
- 夫星忌，夫宫为喜
- 既然是从格，官杀也为喜神
- 但是起的作用都是坏作用
- 丈夫对命主没有帮助，虽不和却不会离婚

**六亲：**
- 年月双透辛金，辛金为忌神
- 辛为母亲
- 从格不喜印生身，所以母亲无缘
- 亲母早亡，又娶继母
- 母宫为喜，关系还算可以

**子女：**
- 食神卯木是儿子
- 有卯木克
- 癸亥年生子
- 癸水透出泄辛金生乙木食神而生子

### 例2：台湾总统（另一例）

**八字：**
```
丁 丙 辛 甲
未 午 卯 午
```

**定格取用：**
- 本造辛日干
- 干透甲丙丁
- 地支卯未会
- 卯木生二午
- 官杀混杂
- 日干弱极
- 丙辛之合因天干透财杀破化气

**判断：**
- **化气不成以正格论**
- 地支卯未会财局
- 因午未合，未土印合留
- **格神：以丙火正官为格**
- **用神：用未土偏印泄杀为用神**
- 既用印不能论从

**大运分析：**
**行卯运壬申年：**
- 流年壬水克正官格神
- 运支卯会年未
- 原局午未合留之未土用神，因解合会卯而去
- 转生午火七杀
- 运支卯木克用神未土
- **用神尽去**

**结果：**
- 行运至此
- 七杀忌神大旺
- 正官用神衰弱
- 真衰假旺
- 用神尽去而凶亡

### 例3：三戌知县（从官格）

**八字：**
```
癸 丙 戊 壬
戌 戌 巳 戌
```

**定格取用：**
- 癸生戌月
- 地支三戌
- 日坐巳宫
- 土旺逢火
- 干透丙戊壬
- 丙生戊，戊克壬
- 癸水虚浮
- 四柱无一点生扶之气

**定格：**
- **格神：当从戊土正官为格**
- **用神：以丙火为用**

**推论理由：**
> "从格忌行比劫印地、忌其生助日干而不从"

**大运分析：**
- 此造早行北方亥子丑
- 日干禄旺之地
- 何以无恙？
- 乃原局无金生水泄戊
- 所谓水来土掩
- **从的真之故**（虽无恙，亦无用）

**交壬寅运：**
- 命运制化合宜
- 不悖格局
- 方得百里之尊
- 其历三任知县而不升
- 乃从格不行从神生旺之地，而显逊色也

### 例4：丁壬皆留（化气不成）

**八字：**
```
丁 丁 壬 壬
巳 未 寅 寅
```

**定格取用：**
- 时干壬水坐寅
- 月干丁火坐未
- 月时二干丁壬一合
- **丁壬皆留**

**判断：**
- 壬水既留在原局
- 有比帮身
- **不能做化气论**
- **此以正财为格**
- **用神：用比肩壬水制偏财**
- 巳宫庚金偏印制甲木生壬水为用神

**婚姻：**
- 此造官杀皆忌
- 以巳宫庚印化夫宫寅中戊杀为夫星
- 惜巳宫有丙火带忌
- 这是克夫的基因

**大运分析：**
**行庚戌运：**
- 运支戌与夫宫寅会成丙火
- 生戊土七杀忌神
- 克夫星庚金
- **而于该运克夫**

---

### 例5：前清洛秉章（不能论化气）

**八字：**
```
癸 丙 辛 戊
丑 辰 亥 子
```

**定格取用：**
- 辰月丙戊进气
- 丙火生戊土
- **戊土正印生身**
- **克制化气化神**

**判断：**
> "所以此造不能论化气，只以正格论"

- 地支亥子丑全
- 子辰会局
- 辰月癸旺壬衰
- **格神：当以癸水食神为格**
- **用神：用丙火生戊土，戊土制壬生身为体为用**

**结果：**
- 体用精神和谐
- 大运一路非格旺地即用神旺地
- 尤其是戌运
- 丙戊入库

---

### 例6：假化之例

**八字：**
```
戊 辛 丙 己
子 酉 申 丑
```

**定格取用：**
- 时干己土有月柱辛金之泄
- 年干戊土月令死地
- 时支丑土会酉成金局
- 日干与月干丙辛一合
- **成化气格**

**假化之因：**
```
1. 原局有戊己土逆化气化神
2. 虽经泄化得宜，仍属假化
3. 壬癸水混杂无法取清
```

**大运分析：**
**甲子、乙丑二十年大运：**
- 非泄壬留癸
- 即泄癸留壬
- 去留得宜

**丙寅运：**
- 运干与原局争合
- 日干丙火与戊土忌神至寅得长生
- **化气格破局**

## 七、从格与化气的运用

### 7.1 大运流年判断

**从格喜忌：**
```
喜：从神生旺之运
忌：印比生身之运
   克害从神之运
```

**化气格喜忌：**
```
喜：化神生旺之运
忌：破化气之运
   争合之运
```

### 7.2 从格的成败

**成功因素：**
1. 从神真旺
2. 无混杂
3. 无刑冲克害
4. 大运配合

**失败因素：**
1. 从神不旺
2. 有混杂无制
3. 有刑冲克害
4. 大运不配合

## 八、核心要点总结

### 8.1 从格核心

1. **日干无气**
   - 弱而极弱
   - 无生扶

2. **从神当令**
   - 财官食伤大旺
   - 不得不从

3. **清纯为贵**
   - 正偏不混
   - 去留得宜

4. **大运配合**
   - 行从神旺地吉
   - 行印比旺地凶

### 8.2 化气核心

1. **日干弱无气**
   - 克泄交加
   - 无印比生

2. **必须有合**
   - 日干与月时干合
   - 透出化神

3. **地支配合**
   - 成方成局
   - 生助化神

4. **真假之分**
   - 真化富贵
   - 假化亦有富贵

## 十、常见错误

### 错误1：日干稍有弱就论从

```
❌ 日干弱就论从
✅ 日干弱无气，四柱克泄交加，无印比生扶，才论从
```

### 错误2：只要有合就论化

```
❌ 日干与月时干合就论化
✅ 必须透出化神，地支成方成局，才论化
```

### 错误3：从格不取用

```
❌ 从格不需要取用
✅ 从格也需要取用，去留取清为用
```

### 错误4：假从一定不好

```
❌ 假从就是不好
✅ 假从行真从之运，亦可大发
```
