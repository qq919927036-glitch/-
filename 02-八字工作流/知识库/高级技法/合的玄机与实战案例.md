## 核心概念

### 合的重要性

> "命局之中天干地支的五合与六合，在八字的测算中是非常重要的"

**为什么重要？**
1. **根本性变化**：相合会产生根本变化，就像中药配方、化学元素合成
2. **合留vs合去**：决定吉凶祸福
3. **玄机所在**：暗藏的成败关键

---

## 一、认识合神

### 1.1 合的类型

**天干五合：**
```
甲己合化土
乙庚合化金
丙辛合化水
丁壬合化木
戊癸合化火
```

**地支六合：**
```
子丑合
寅亥合
卯戌合
辰酉合
巳申合
午未合
```

### 1.2 合的变化

#### 合留（吉）

**定义：** 格神或用神遭到克泄，命局中有合来救助

**示例：**
```
以丙火为用，生于冬令地支见亥子水克火
天干逢辛金相合 → 丙火为留 → 非常好的合
```

**结果：** 喜神合留则有福

#### 合去（凶）

**定义：** 格用之神没有遭克泄，而与他神作合

**结果：** 喜神被合去 → 英雄无用武之地

## 二、合的吉凶判断

### 2.1 核心原则

> 经云：**"合有宜不宜，合多不为奇"**

| 情况 | 结果 | 判断 |
|------|------|------|
| 合留喜神 | ✅ 吉 | 喜神得救，可以发挥作用 |
| 合去忌神 | ✅ 吉 | 忌神被去掉，命局变清 |
| 合留忌神 | ❌ 凶 | 忌神留在局中，为害 |
| 合去喜神 | ❌ 凶 | 喜神被去掉，英雄无用武之地 |

### 2.2 大运流年的影响

**喜神合留者：**
- 喜：后天大运行喜神生旺之运
- 忌：行克泄之岁运

**合去的喜神：**
- 得大运之生扶 → 也可小草春风
- 再遇克害衰绝 → 真正合去无疑

**喜神合留、忌神合去：**
- 岁运不可再遇合神
- 再遇就是**解合、争合** → 不吉利

**原局无合，大运流年有合：**
- 合留喜神或合去忌神 → 大吉
- 反之则凶

## 三、合的实战技巧

### 3.1 地支六合的运用

**示例1：适宜之合**
```
地支：寅 亥
生于：子月
天干：透甲癸

定格：以癸水为格
用神：用甲木泄壬为用神

问题：寅亥一合 → 壬水合留在原局混杂癸水
判断：此一合为不宜之合 ❌
```

**示例2：适宜之合**
```
地支：申 寅 亥（相邻）
格用：以甲木为格用喜神

变化：申来冲寅，而有亥来合寅
结果：甲木因寅亥之合为留
判断：此一合谓之适宜之合 ✅
```

### 3.2 天干五合的运用

**关键要点：**
- 看合留的是谁
- 看合去的是谁
- 看合后的五行变化

## 四、经典命例分析

### 例1：和珅（十一级富贵）

**八字：**
```
庚 乙 庚 壬
午 酉 子 午
```

**定格取用：**
- 月令：庚辛金禄旺之地
- 天干：乙庚合，皆合留在局
- 乙木作用：泄子水生丁火
- 格神：以庚金为格
- 用神：用丁火克酉金

**乙木为何可用？**
- 乙木年支得长生（阴干之长生带泄）
- 乙木生午，因有酉金之克
- 得午火制酉护乙而得长生
- **"儿能生母泄天机"**

**合的玄机：**
- 日时子午相冲 → 子午皆冲去
- 子水不会破害年支午火
- 可以寻暗拱暗会为真神

**计算级数：**
```
午火月令：4级
乙木年支得生：4级
子午冲去：3级
----------------
共计：11级
```

**大运分析：**
- **戊子运**：庚金死辛金生，格神死地假神当真。子冲午，辛金假神无受破害，可以当真
- **己丑运**：庚金丁火格用神入库，二十年好运，威权显奕
- **庚寅运**：庚运尚可，寅运会午，官杀混杂，制比刃，失势而退
- **辛卯运**：壬申年，运干辛金克乙木，运支卯酉冲，壬申年申子会局，壬水泄庚金，格用去，比刃体神也去，遭赐死

## 五、核心要点总结

### 5.1 合的判断流程

```
第一步：发现合
  ↓
第二步：判断是合留还是合去
  ↓
第三步：判断合的是喜神还是忌神
  ↓
第四步：综合判断吉凶
```

### 5.2 核心原则

1. **高度关注合的现象**
   - "凡是命局中出现合的现象，一定要高度关注"

2. **详细察看合去合留**
   - "详细察看喜忌神是合留还是合去"

3. **关系命局成败**
   - "合去合留的玄机极大，关系着命局的成败和层次问题"

4. **可能决定生死**
   - 合去喜神 → 英雄无用武之地
   - 合留喜神 → 喜神得救

5. **配合大运流年**
   - 原局无合，大运有合 → 也要判断
   - 岁运配合 → 共同决定吉凶

### 5.3 记忆口诀

```
合留喜神最为吉
合去忌神也是喜
合去喜神最不吉
合留忌神要注意

原局无合运有合
判断吉凶照此理
合多不为奇中奇
宜与不宜细分析
```
