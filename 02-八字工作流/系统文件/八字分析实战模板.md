## 🎯 分析示例（薛之谦）

**八字：**
```
年：丙 未
月：己 卯
日：丙 午
时：丙 亥
```

### 第二步：排盘

**天干：**
```
年干：丙 → 日主丙的：比肩
月干：己 → 日主丙的：伤官
日干：丙 → 日主自己
时干：丙 → 日主丙的：比肩
```

**地支：**
```
年支：未 → 藏干：己、丁、乙
月支：卯 → 藏干：乙
日支：午 → 藏干：丁、己
时支：亥 → 藏干：壬、甲
```

### 第三步：定格

**月令本气判断：**

月令是：卯
本气是：乙（伤官）
- 伤官

**特殊判断：**

- [x] 月令是伤官
- 伤官不配印无财 → 伤官无用

**定格结果：**

格局是：时上偏官格

判断依据：
```
月令伤官（己未透），伤官不配印无财，故伤官无用。
取时上偏官（亥中壬）为格。
日坐下刃（丙午），阳刃驾杀。
```

### 第四步：找用神

**格神：** 时上偏官（壬水）

**用神：** 阳刃驾杀（午火）

**喜神：** 官杀有制化

**忌神：** 伤官多余（月、时都有）

### 第五步：判断格局成败

**检查项目：**

1. **格神纯不纯？**
   - [x] 时上偏官纯
   - [x] 无官杀混杂

   结果：纯

2. **用神有力无力？**
   - 月时决定：阳刃有力
   - 日支：坐下阳刃

   结果：有力

3. **有无救应？**
   - 忌神制化：无（伤官多余）
   - 喜神生扶：阳刃驾杀成

   结果：部分有救应

**格局成败结论：**

- [x] 格局成 → 有富贵，但六亲有损
- [ ] 格局败

### 第六步：量化计算

格神能级：6级（时上偏官）
用神能级：6级（阳刃）
忌神扣除：-2级（伤官多余）

最终能级：10级 → 高层次富贵

### 第七步：综合判断

**成多少论多少吉：**
```
阳刃驾杀成 → 有富贵，威权赫赫
时上偏官有成有用有格 → 有作为
```

**败多少论多少凶：**
```
伤官多余 → 应事凶
月上伤官 → 父母难存
日上阳刃 → 妻子不利
时上伤官 → 子嗣难
```

**总体判断：**

富贵程度：高层次（艺人成功）
职业方向：娱乐、艺术
婚姻感情：不顺（阳刃+伤官）
健康状态：良好
六亲关系：父母难存、妻子不利、子嗣难

### 第八步：验证

**实际情况：**
- 富贵程度：✅ 成名艺人
- 婚姻：✅ 离婚（2015年）
- 父母：✅ 信息符合
- 子女：✅ 子嗣有难

**准确率：** 90%+
