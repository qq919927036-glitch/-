# 记忆库架构说明

## 三层记忆架构

### 情景记忆 (Episodic Memory)
**用途**: 记录具体事件和对话过程
**命名**: MMDD-简短描述.md
**归档**: 按月归档（2026-01/、2026-02/）
**何时读取**: 需要回溯"当时怎么想的"时

### 语义记忆 (Semantic Memory)
**用途**: 提炼的通用知识和模式，可复用
**命名**: [主题]_[类型].md
**何时读取**: 需要复用知识时，直接读取

### 强制规则 (Procedural Memory)
**用途**: 必须遵守的行为约束
**命名**: 强制规则_[领域].md
**何时读取**: 每次会话自动加载，不可违反

## 记忆提炼流程

```
情景记忆 → 识别模式 → 语义记忆
语义记忆 → 发现必须遵守的规则 → 强制规则
```

## 触发条件
- 每次自我迭代时
- 发现重复模式时（≥3次）
- 完成重大项目后
