# 强制规则：创作任务触发条件

> 何时进入创作工作流，何时记录即可

## 进入完整创作流程的条件

当用户说以下内容时，进入创作任务：

### 明确创作意图
- "写一篇文章"
- "帮我写..."
- "我要写..."
- "开始创作..."

### 有具体选题
- 用户提供标题或主题
- 用户提供详细需求

### 工作流程
1. 询问/确认基本信息（目标读者、长度、风格）
2. 生成大纲
3. 写作内容
4. 优化标题
5. 保存到 `02-写作中/`

## 仅记录选题的条件

当用户说以下内容时，只记录到选题记录：

### 碎片想法
- "有个想法..."
- "可以写..."
- "是不是可以写..."
- 没有具体展开的想法

### 工作流程
1. 追加到 `00-选题记录.md`
2. 快速记录核心想法
3. 不展开写作

## 判断逻辑

```
用户输入
  ↓
检测关键词
  ↓
有明确创作意图? → YES → 进入创作流程
  ↓ NO
有具体选题? → YES → 进入创作流程
  ↓ NO
仅记录到选题记录
```

## 示例

### 进入创作流程
- **用户**: "写一篇关于雨水节气祛湿的文章"
- **AI**: 进入创作流程，询问细节，生成内容

### 仅记录选题
- **用户**: "可以写写雨水节气祛湿"
- **AI**: 记录到 `00-选题记录.md`，"已记录：雨水节气祛湿"
