### 二、从格与化气格

#### 1. 从格判断

**真从格：**
```
日干弱无气
四柱克泄交加
无印比生扶
从神生旺
无刑冲克害
```

**假从格：**
```
从神有根（有印比）
原能去根
有克害从神之神
行真从之运亦可大发
```

#### 2. 从格六格

1. 从食神格
2. 从伤官格
3. 从正财格
4. 从偏财格
5. 从正官格
6. 从七杀格

**核心理论：**
> "从儿不论身强弱，只要我儿又见儿"

> "一出门来只见儿，吾儿成气构门闾"

#### 3. 化气格判断

**真化条件：**
```
日干弱无气
四柱克泄交加
无印比生
日干与月干或时干作合
天干透出化神
地支成方成局
```

**假化条件：**
```
化气之神有克泄，而有救应
化气之神有混杂，有去留
日干原局有气
假化行真化之运亦能富贵
```

#### 4. 经典案例

**马英九（从儿格）：**
- 从乙木食神为格
- 丁火偏财为用
- 官杀为忌
- 虽不从却不会离

**三戌知县（从官格）：**
- 从戊土正官为格
- 丙火为用
- 水来土掩，从的真
- 三任知县而不升

## 💡 关键洞察

### 1. 合的重要性远超想象

之前以为合只是普通的地支关系，现在认识到：

**合是根本性变化：**
- 就像中药配方
- 就像化学元素合成
- 相合就会产生根本变化

**合决定成败：**
- 和珅的十一级富贵，关键在合
- 李鸿章的取清成格，关键在合
- 很多命局的成败，都在一合之间

### 2. 从格化气不是另类

之前以为从格化气是特殊例外，现在认识到：

**从格化气也是定格取用：**
- 同样需要净阴净阳
- 同样需要去留取清
- 同样需要判断格用喜忌

**只是日干状态不同：**
- 正格：日干有气
- 从格：日干无气
- 化格：日干合化

### 3. 理论体系高度统一

子平格局理论的强大之处在于：

**一套理论贯穿始终：**
- 净阴净阳 → 适用于所有格局
- 定格取用 → 适用于所有格局
- 量化计算 → 适用于所有格局

**没有例外和矛盾：**
- 正格如此
- 从格如此
- 化格也如此
