## 📊 核心概念：因缘果

### 因缘果理论

**核心口诀：**
> "原局是因，大运是缘，流年是果。"
> "因缘果就是应事。"

**解释：**
一件事的发生，必须：
1. **原局有**（因）- 命局本身具备这个因素
2. **大运体现**（缘）- 大运把这件事体现出来
3. **流年引发**（果）- 流年最终促成这件事

**例子：**
```
原局有官星（因）
├── 大运官星旺（缘）
└── 流年官星透（果）
    └── 升官发财
```

## 🎯 大运的本质

### 1. 大运是什么？

**比喻1：党大会**
> "大运就像我们党大会，换一下大运就换代表了。"
> "大运像联合国的轮值委员。"

**比喻2：常委班底**
- 月令 = 习大大（核心领导）
- 天干透出 = 常委（有发言权）
- 地支 = 班底（内在实力）

### 2. 大运重地支，但天干也看

**核心原则：**
> "大运主看的是地支，但天干也看。"

**为什么？**
- 地支 = 性质，天生是哪一派的
- 天干 = 代表，轮值委员上台
- 有根的 = 这一柱开始值事，体现出来

**判断：**
```
大运透某天干
├── 原局地支有根 → 值事，体现明显
└── 原局地支无根 → 值事，但力度弱
```

### 3. 大运五行的"进气"

**本气取：**
- 申酉 = 金进气
- 亥子 = 水进气
- 寅卯 = 木进气
- 巳午 = 火土进气

**判断：**
> "凡这些大运基本上都是这些本气开始旺的。"
> "就看这些进气在原局的作用。"

---

## 💡 天干 vs 地支

### 天干的作用

**天干 = 值班的（常委）**
- 有发言权
- 显像明显
- 透出 = 进入核心权力圈
- 透忌神 = 凶神有发言权
- 透喜神 = 吉神值班

**核心：**
> "天干就是值班的。"
> "你看哪个常委牛逼给力，就看其班底（地支）有没有旺气。"

### 地支的作用

**地支 = 班底（内在实力）**
- 是内在的斗争，不上台面的
- 通过天干作用
- 地支旺 = 天干有班底，有力
- 地支弱 = 天干无班底，无力

**核心：**
> "地支是通过天干作用的，因为天干是显像。"
> "地支的生克作用是内在的斗争。"

## 🔍 判断吉凶的标准

### 一切围绕格局

**核心原则：**
> "看八字以月令为习大大，以习大的喜好定吉凶。"
> "一切以格局为中心点。"

**判断流程：**
```
第一步：找格局（月令习大大）
第二步：看格局喜忌
第三步：看大运流年对格局的影响
```

### 组合的意义

**为什么有组合？**
> "组合的意义在于促成格局。"

**常见组合：**
- 伤官格见印（印保护官）
- 印格带官（印需要官贵）
- 财格带印（财需要官保护）
- 官杀带印（官印双全）
- 官杀带财（财生官）

**核心逻辑：**
- 都是围绕格局
- 都是为了使格局发挥作用
- 就像习大大需要辅助之臣

### 格局不喜欢相神旺于自己

**核心原则：**
> "格局不喜欢相神旺于自己。"

**例子：**
- 官格喜印 → 但印不能太旺，超过官
- 财格喜官 → 但官不能太旺，超过财
- 七杀喜食神制 → 但食神不能太旺

**原因：**
> "习大会喜欢辅助我为我护驾的人权力比我大吗？班底比我厚吗？"

## ⚠️ 大忌

### 1. 天干忌神透出无制

**核心判断：**
> "透忌神就算他没有班底，他也有发言权，只要不被制就是凶。"

**判断：**
- 忌神透干 = 进入常委
- 有力凶大，无力凶小
- 但只要不被制就是凶
- 可以反复断这个凶所代表的事物

### 2. 地支忌神值事

**判断：**
- 地支忌神 = 小委员，不是常委
- 可能很有实力，但没有发言权
- 等大运流年透出 = 轮值上位 = 体现凶

### 3. 天克地冲年柱

**核心：**
> "年支凶是大凶。尤其是天克地冲年柱不好。"
> "如果年上是忌还好，如果年上是吉那是大灾。"

**原因：**
> "你们可以理解为，年是天授。"

### 4. 格局根被刑冲

**核心：**
> "官格的根怕刑，怕冲。"

**判断：**
- 格局根被冲刑 = 班底内耗
- 如果是制忌神的刑冲 = 吉
- 如果是伤格局的刑冲 = 凶

**例子：**
- 甲生申月，巳来刑申 → 食神制杀 → 吉

## 📋 日支的特殊地位

### 日支 = 自己

**核心：**
> "日支和我是一体，等于我本身就是日柱，而不是只看天干。"
> "日支代表我拥有的能力，特点，内在的。"

**为什么重要？**
- 日支 = 自己拥有的东西
- 感应自己最深
- 日支的东西最有用

**制化在地支：**
```
七杀格
├── 食神不透
└── 食神在地支
    ├── 日支食神 = 最有用（是我自己）
    ├── 时支食神 = 有用（等轮值上岗）
    └── 年支食神 = 无用（太远）
```

## 🎯 大运流年判断实战流程

### 第一步：分析原局（因）

**必须明确：**
1. 格局是什么？（月令本气）
2. 格局喜什么？
3. 格局忌什么？
4. 有无组合？
5. 有无救应？

### 第二步：看大运（缘）

**判断大运对格局的影响：**
```
大运来了
├── 大运天干 = 轮值委员
│   ├── 透喜神 → 吉神值班
│   └── 透忌神 → 凶神值班（需看有无制化）
└── 大运地支 = 这一派的性质
    ├── 看本气是什么（金木水火土）
    ├── 看原局是否有根
    └── 看对格局的影响（喜忌）
```

### 第三步：看流年（果）

**流年引发：**
```
流年来了
├── 原局有 + 大运体现 + 流年引发 = 事成
└── 三者缺一 = 事不成
```

**判断：**
1. 流年天干 = 今年的值班常委
2. 流年地支 = 今年的性质
3. 综合判断对格局的影响

### 第四步：细论应事

**围绕格局论：**
> "原局所有的吉利都是围绕用神格局是不是，不管升官还是发财。"

**方法：**
- 升官 = 宫星
- 发财 = 财星
- 婚姻 = 宫星（女）或财星（男）
- 六亲 = 各自星
- 健康 = 各自代表五行

## 💡 核心理论总结

### 1. 一切围绕格局

**判断标准：**
> "你们找出格局就要去找格局忌什么，如果有忌就要找需要什么。"

### 2. 先看天干，再论地支

**核心：**
> "先看天干，再论地支，不要想的太复杂。"
> "看八字先看天干，地支也要看，但天干的象更为明显，主看天干，次看地支。"

### 3. 简单看，不要复杂

**核心：**
> "看的太复杂就扑街了，就要简单看。"
> "所有的关系都围绕七杀，而不是其它。"

### 4. 组合只针对原局

**核心：**
> "组合只针对原局，在一透的情况下不要轻易更换原局你所选择组合。"
