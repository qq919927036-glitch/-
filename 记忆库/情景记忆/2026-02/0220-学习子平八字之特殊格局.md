## 📊 特殊格局概述

### 什么时候用特殊格局？

**核心原则：**
> "八字有财官格局都没有，这人就没活路了。不管明的暗的，都要有才成。"

**取格顺序：**
1. **月令正格**（优先）
   - 正官、七杀、正财、偏财、正印、偏印、食神、伤官、比肩、劫财
   - 时上偏财、时上偏官

2. **外格**（最后选择）
   - 当完全无法取财官格局时
   - 原局无财官，不能入印食格
   - 此时必须论外格

**概率：**
> "这类格局，大概十个里面遇到一个，最多这个概率。"

## 🎯 魁罡格

### 定义

**魁罡日（四天）：**
- 壬辰日
- 庚戌日
- 戊戌日
- 庚辰日

### 特点

**性格：**
- 性子会爆些
- 脾气大，胆子也大
- 在家里和父亲完全不对付

**女命影响：**
- 对婚姻有影响
- 加重婚姻不顺

**判断：**
- 魁罡是神煞，不是格局
- 可以与格局叠加论断
- 神煞叠加影响更大

### 案例

**女命：戊戌日**
- 日支戌为墓，心态为拒绝、消沉（对感情）
- 魁罡 + 寡宿（戌）= 性子更爆，婚姻更不顺
- 日坐墓 + 魁罡 = 婚姻宫有问题

## 🎯 倒冲禄格

### 定义

**核心概念：虚邀禄马**

> "虚邀财官，就是暗冲、暗刑、暗邀、暗合。"
> "虽财官不见，而实际暗中已经得了财官，我得了好处你还不晓得，这种好处是最大的。"

### 成格条件

**倒冲禄格（以巳亥为例）：**
```
原局无亥（禄）
├── 两个巳（或三个）
├── 暗冲亥（巳亥冲）
└── 有字合助亥（如卯）

结论：巳旺 = 亥旺，得禄
```

**要点：**
1. 原局不见财官
2. 主强（身旺），可以邀请别人上门
3. 两个或三个才能冲一个
4. 有字合住即可

### 判断原则

**多则不中：**
- 如两个卯争合亥 = "多则不中"
- 会减分数
- 但仍有格局

**男女不同：**
- 男命：大富大贵
- 女命：小富小贵（女命格局不如男命给力）

### 实战案例

**女命：**
```
年：乙 卯
月：甲 巳
日：乙 巳
时：丁 卯
```

**分析：**
1. 夫星七杀弱，夫子不成
2. 只能论格局
3. 两个巳暗冲亥（巳亥冲）
4. 卯合助亥
5. 倒冲禄格成

**判断：**
- 格局成 → 下限不会低
- 合多（卯卯巳巳亥）= 官不纯 → 虚名虚利
- 女命格局 → 小富小贵

**六亲：**
- 巳冲亥 = 夫星
- 两卯争合亥 = 有争夫危险
- 两巳孤鸾孤辰 = 守空房

## 🎯 六乙鼠贵格

### 定义

**条件：**
- 乙日生丙子时
- 不见官星

### 原理

**虚邀财官：**
```
丙子时
├── 丙禄巳
├── 子与巳暗合（子中癸合巳中戊）
└── 巳合申

暗得：
├── 巳中戊土正财
└── 申中庚金正官
```

**核心：**
- 财官不见
- 暗中已得
- 好处最大

## 💡 核心理论

### 1. 为什么要用外格？

**原因：**
> "没有格局要有财官，财官格局都没有，这人就没活路了。"

**逻辑：**
- 正格取不了
- 财官明不见
- 但必须有得财官的手段
- 故有外格论法

### 2. 暗冲暗合的本质

**理解：**
- 主强（身旺）= 邀请别人上门
- 虚邀 = 暗中得财官
- 得了好处，别人还不晓得
- 这种好处是最大的

### 3. 外格的使用原则

**先玩好六神：**
> "一般概率不大，但不可不知，真遇到而不晓得断，就悲催了。"
> "根本上，还是要先玩好六神。"

**论作用思路一样：**
> "这些暗冲暗合暗刑等等，都是虚邀禄马，禄马就是财官，要别位不见财官才能如此论的。"
> "以上呢，只是入手取禄马的方式不同，取出来后，论作用的思路还是一样的。"

## 📋 实战分析流程

### 判断是否用外格

**第一步：检查正格**
```
月令有财官杀印？
├── 有 → 取正格
└── 无 → 下一步
```

**第二步：检查时上偏财偏官**
```
时有偏财偏官？
├── 有 → 取时上格
└── 无 → 下一步
```

**第三步：检查外格**
```
完全无财官？
├── 是 → 论外格
│   ├── 倒冲禄
│   ├── 六乙鼠贵
│   ├── 刑合格
│   ├── 飞天禄马
│   └── 其他
└── 否 → 能否入印食格？
```

### 分析外格成败

**第一步：判断成格条件**
- 是否符合该外格的条件？
- 是否有配合（如合助）？
- 是否有破坏（如多冲）？

**第二步：论作用**
- 取出财官后
- 按正常格局论
- 看纯清、有力、救应

**第三步：加加减减**
- 神煞叠加
- 外格 + 神煞
- 综合判断

## 🎯 今日核心收获

### 1. 理解外格的本质

**外格不是玄学：**
- 是财官取不到时的补充方法
- 是虚邀财官的技术
- 论作用思路与正格相同

### 2. 掌握常见外格

- 魁罡格（神煞，非格局）
- 倒冲禄格
- 六乙鼠贵格
- 刑合格

### 3. 建立判断逻辑

**先正格，后外格：**
1. 月令正格
2. 时上偏财偏官
3. 外格（虚邀财官）
4. 印食格
