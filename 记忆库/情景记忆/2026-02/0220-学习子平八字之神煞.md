## 📊 神煞的核心原则

### 1. 神煞的起法

**核心原则：**
> "神煞一般是以年支起，以落在日时为重。"

**判断：**
- 以年支起神煞
- 落在日时才很应验
- 日时占其一就很重
- 日时全备更加重

### 2. 神煞与格局的关系

**核心原则：**
> "神煞可以直接用，尤其是婚姻上的。"
> "有些八字，找出神煞，直接断可以了。"

**判断流程：**
```
先看格局
├── 格局成 + 神煞不重 → 论格局
└── 格局不成或神煞太重 → 直接论神煞
```

**重要：**
> "外面的算命都不太重视神煞，偶然说了几个神煞都是曲解了煞的本意。"

## 🎯 孤辰寡宿

### 定义与起法

**孤辰寡宿：**
- 以年支起
- 三会局的左右两个字

**起法：**
| 三会局 | 寡宿 | 孤辰 |
|--------|------|------|
| 亥子丑 | 戌 | 寅 |
| 寅卯辰 | 丑 | 巳 |
| 巳午未 | 辰 | 未 |
| 申酉戌 | 未 | 亥 |

**记忆口诀：**
> "辰戌丑未，永远是寡，寅申巳亥，永远是孤。"

### 应验

**男怕孤辰，女怕寡宿：**
- 男人犯孤辰 → 很难娶老婆
- 女人犯寡宿 → 很难嫁

**具体克应：**
1. 很难结婚
2. 要到外省、外国找对象
3. 不外出 → 克父母、克配偶
4. 对婚姻阻隔

**特殊情况：**
> "女命夫星坐下或本身就是寡宿的话，100%克死老公。"

**化解：**
- 助养孤寡老人一年（上门关心）
- 嫁给外省人（最好有点残疾）
- 找健全男人不会有好结果

### 实战案例

**案例1：男命未娶**
```
己未 辛未 乙酉 甲申
└── 申孤辰
```
男命孤辰在时，至今未娶。

**案例2：女命老处女**
```
辛卯 癸巳 己巳 乙丑
├── 丑寡
├── 巳孤
└── 日时全占
```
和小姑一起住，一直是处女，万年处女身。

## 🎯 隔角煞

### 定义与起法

**隔角：**
- 三会局的交接处
- 木局、火局交接处 = 辰巳

**起法：**
- 辰巳（木火局）
- 未申（火金局）
- 戌亥（金水局）
- 丑寅（水木局）

**核心：**
> "隔角，也是一个克六亲很厉害的神煞，八字里面集齐两个便是。"

### 应验

**判断：**
- 字在什么宫 → 克什么人
- 什么六神坐上去 → 也照样被克
- 怕在日时，尤其在日 → 肯定克老公老婆

**具体克应：**
1. 克六亲很重
2. 集齐两个便是
3. 在日时最重

### 实战案例

**案例1：多煞并发**
```
己未 辛未 乙酉 甲申
├── 申孤辰
└── 未申隔角
```
未是财，又是隔角，又是孤辰。

**案例2：外省离婚**
```
壬戌 丁未 辛亥 辛卯
├── 孤辰寡宿
└── 隔角煞
```
嫁了外省老公，15年离婚。

## 🎯 阴差阳错

### 定义与起法

**阴差（6个）：**
丁丑、丁未、辛酉、辛卯、癸亥、癸巳

**阳错（6个）：**
丙子、丙午、戊寅、戊申、壬戌、壬辰

**核心：**
> "四柱见之则是，尤其日时为重，日时之中，以日为重。"

### 应验（男命）

**核心：**
> "男人犯这个神煞，会被女友或老婆坑，不是有心坑的，是被拖累。"

**三个坑：**
1. **第一坑：** 夫妻不和互克，两家不和
2. **第二坑：** 杀对方父母（克死人）
3. **第三坑：** 被连累（官司、破财等）

**具体：**
- 结婚后女家很多事，被牵连
- 女家出官司，无辜躺枪
- 婚姻波折，不得外家力
- 男人最好入赘否则克妻

### 应验（女命）

**核心：**
> "女人犯这个也是被老公连累。"

**具体：**
- 被老公连累
- 老公赌博、婚外情
- 夫妻两家好似仇人

### 特殊组合

**阴差阳错 + 桃花：**
> "桃花并阴差阳错，是十分差的组合。"

**具体：**
- 婚前婚后打官司
- 老婆做生意破大财
- 老婆亲戚连累自己
- 三次招女人官司

**注意：**
> "反正你们记好，阴阳差错都算了，还加桃花的话，坑坑坑坑。"

### 实战案例

**案例1：被连累**
```
戊午 乙丑 丁丑 丙午
├── 丁丑阴差
└── 丙午阳错
```
结婚后两家是非多，老婆外面找男人，2017年离婚。

**案例2：克死老婆**
```
丙子 庚寅 丁丑 丙午
├── 丙子阳错
├── 丁丑阴差
└── 日时两犯
```
克死两个老婆，娶了三妻。

**案例3：洗脚婢**
```
癸亥 癸亥 辛酉 戊子
├── 一点夫星都没有
└── 三犯差错
```
洗脚婢（身份低贱）。

## 🎯 孤鸾煞

### 定义与起法

**孤鸾日（8个）：**
乙巳、丁巳、辛亥、戊申、甲寅、丙午、戊午、壬子

**特征：**
- 丁巳、甲寅、丙午、戊午、壬子 = 自坐旺地
- 乙巳、辛亥 = 自坐伤官
- 戊申 = 夫星绝地

**核心：**
> "就是没有老公没有老婆的鸟。"
> "八字见夫星，见妻星，不是真的孤鸾。"

### 应验

**判断条件：**
1. 没有夫妻星
2. 夫妻宫坏
3. 见孤鸾丧夫煞

**具体：**
> "如果女命没有夫星，夫宫又烂，又见孤鸾丧夫煞。"
> "就算夫星也很好，也是各种怄气。"

**特殊情况：**
> "有些命就是要多吵架多打架才身体健康。"
> "如果你们感情很恩爱很顺——丧夫。"

### 辛亥、甲寅的特殊

**核心：**
> "辛亥、甲寅——若不淫乱，多主丧夫。"

**判断：**
- 比较贞洁、孤单
- 不淫乱 = 丧夫
- 老公出轨可以化解
- 如果都不出轨 = 丧夫

### 戊午、壬子的特殊

**核心：**
> "戊午、壬子的孤鸾，最好带财星，否则多嫁穷夫。"

### 孤鸾的特殊状态

**孤鸾无巢：**
- 没有夫妻星
- 夫妻宫坏了
- 真正的孤鸾

**孤鸾带箭：**
- 孤鸾见刑
- 多数丧夫离异不堪重击
- 全家阵亡后自己自杀

**孤鸾 + 桃花：**
- 红艳、七杀
- 失恋失婚后被人奸杀

### 品格问题

**核心：**
> "孤鸾的人，看她是离异还是丧夫，可以看看样子。"

**判断：**
- 越美丽越克夫（美貌=离婚）
- 漂亮温柔眼神有点哀伤=丧夫
- 长得像和尚尼姑=一定丧夫

**人品：**
> "孤鸾要么被人背叛，要么背叛人。"
> "很容易做出出卖朋友的事，或者挑拨离间的事。"

**注意：**
> "你们交友的话，看到日时有孤鸾的，要谨慎。"
> "孤鸾最容易做内奸，无间道。"

### 实战案例

**案例1：离婚**
```
己未 丁丑 丙午 甲午
└── 孤鸾逢刑
```
2016年离婚。

**案例2：未婚**
```
丙申 戊戌 甲寅 乙丑
└── 61岁，孤鸾日
```
至今未婚。

**案例3：背叛**
```
辛酉 己亥 壬子 丙午
└── 日时孤鸾
```
收徒弟，为了利益背叛师父，还挑拨离间。

## 💡 神煞使用原则

### 1. 什么时候用神煞？

**直接用神煞：**
- 找出神煞 → 直接断
- 不必找喜忌格局什么的

**例子：**
> "有些八字，找不出神煞，直接断可以了。"
> "反正犯了，要么难结婚，要么克死对象，要么离婚，就是那么简单。"

### 2. 神煞与格局的关系

**核心：**
> "就算格局很高，都无可避免地做小三，或嫁二手男的。"

**判断：**
- 神煞太重 → 直接论神煞
- 格局成 + 神煞不重 → 论格局
- 两者参看

### 3. 神煞的轻重

**判断标准：**
- 日时为重，日尤重
- 年月较轻
- 集齐两个很重
- 多个神煞叠加更重
