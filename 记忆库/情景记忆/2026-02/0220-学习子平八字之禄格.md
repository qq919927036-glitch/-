## 🎯 三种禄的核心区别

### 1. 月令禄（甲丙庚壬）

**定义：** 日干在月令临官之地

**特点：**
- 日主旺相，但不是最旺
- 可以取食神泄秀
- 优先取官杀

**取格顺序：**
```
1. 食神透出 → 取食神格
2. 不透食神 → 取官杀
3. 无官杀 → 取财
4. 以上都无 → 取印
```

### 2. 建禄格（乙丁辛癸）

**定义：** 日干在月令禄位（阴干）

**特点：**
- 日主已经够旺了
- 不需要食神泄秀
- 必须**另寻财官**

**取格顺序：**
```
1. 官杀（最优先）
2. 财（次之）
3. 食伤（再次）
4. 印（最后）
```

**核心口诀：**
> "乙生卯月，丁生午月，辛生酉月，癸生子月，都是月令建禄，要另寻财官，天干见官杀为优先，财次之，食伤再次之，印最后。"

### 3. 阳刃格（甲丙庚壬）

**定义：** 日干在月令帝旺之地（阳干）

**特点：**
- 日主最旺（过度）
- 需要官杀制刃
- 无官杀则食伤泄秀

**取格顺序：**
```
1. 官杀（最喜，制刃）
2. 食伤（泄秀）
```

**核心口诀：**
> "羊刃是日主最旺的状态，所以需要官杀来制。如无制约，那么就喜欢食伤来泄秀。"

## 🔍 实战判断流程

### 第一步：判断是什么禄

```
拿到八字 → 看日干 → 看月令 → 判断类型

例1：甲日 申月
→ 甲在申为临官禄
→ 月令禄类型
→ 透食神先取食神，不透先取官杀

例2：乙日 卯月
→ 乙在卯为建禄
→ 建禄格类型
→ 另寻财官，官杀优先

例3：甲日 卯月
→ 甲在卯为阳刃
→ 阳刃格类型
→ 直接看官杀，无官杀取食伤
```

### 第二步：按类型取格

**月令禄（甲申、丙巳、庚申、壬亥）：**
1. 看天干是否透食神？
   - 透 → 取食神格
   - 不透 → 下一步

2. 看天干是否有官杀？
   - 有 → 取官杀格
   - 无 → 下一步

3. 看天干是否有财？
   - 有 → 取财格
   - 无 → 取印格

**建禄格（乙卯、丁午、辛酉、癸子）：**
1. 看天干是否有官杀？
   - 有 → 取官杀格（最优先）
   - 无 → 下一步

2. 看天干是否有财？
   - 有 → 取财格
   - 无 → 下一步

3. 看天干是否有食伤？
   - 有 → 取食伤格
   - 无 → 取印格（最后选择）

**阳刃格（甲卯、丙午、庚酉、壬子）：**
1. 看天干是否有官杀？
   - 有 → 取官杀格（制刃，最贵）
   - 无 → 下一步

2. 看天干是否有食伤？
   - 有 → 取食伤格（泄秀）
   - 无 → 无格（不成格局）

## 💡 核心理论

### 1. 为什么建禄要"另寻财官"？

**原因：**
- 建禄日主已经够旺了
- 不需要食神泄秀（那样还是向身）
- 必须找到财官，才能体现价值
- 财官 = 富贵

**口诀：**
> "禄是比劫的根，比劫是分夺财官的东西"
> "建禄无祖业，靠自己"

### 2. 为什么阳刃要官杀制？

**原因：**
- 阳刃是日主最旺的状态（过旺）
- 过度则失，中庸为贵
- 必须有官杀来制约
- 无制约则凶神逞凶

**口诀：**
> "阳刃最喜官杀"
> "阳刃驾杀，威权赫赫"

### 3. 三种禄的本质区别

| 类型 | 日主状态 | 核心需求 |
|------|----------|----------|
| 月令禄 | 旺相 | 食神泄秀或官杀贵气 |
| 建禄 | 够旺 | 另寻财官，不向身 |
| 阳刃 | 最旺 | 官杀制刃 |

## 📋 实战分析模板

### 八字分析标准流程（可执行版本）

**第一步：排盘**
```
输入：出生时间
输出：四柱八字
```

**第二步：定格**
```
1. 看月令本气是什么？
   - 正官 → 正官格
   - 七杀 → 七杀格
   - 正财 → 正财格
   - 偏财 → 偏财格
   - 正印 → 正印格
   - 偏印 → 偏印格
   - 食神 → 食神格
   - 伤官 → 伤官格
   - 比肩 → 比肩格
   - 劫财 → 劫财格

2. 特殊判断：
   - 月令是禄？
     * 甲丙庚壬 → 月令禄
     * 乙丁辛癸 → 建禄格
   - 月令是阳刃？
     * 甲丙庚壬 → 阳刃格

3. 按类型取格：
   - 月令禄：食神 > 官杀 > 财 > 印
   - 建禄：官杀 > 财 > 食伤 > 印
   - 阳刃：官杀 > 食伤
```

**第三步：判断格局成败**
```
检查项目：
1. 格神纯不纯？
   - 有无混杂？
   - 有无刑冲？
   - 有无空亡？

2. 用神有力无力？
   - 月时决定旺衰
   - 日支决定辅助

3. 有无救应？
   - 忌神有无制化？
   - 喜神有无生扶？
```

**第四步：量化计算**
```
如果有数据：
1. 计算富贵等级
2. 计算六亲数量
3. 计算能级
```

**第五步：综合判断**
```
1. 成多少论多少吉
2. 败多少论多少凶
3. 多格参看（n个格局互相参考）
```
