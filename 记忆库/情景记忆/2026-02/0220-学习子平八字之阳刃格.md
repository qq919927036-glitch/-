## 📊 核心概念

### 一、阳刃的定义

**阳刃（非羊刃）**
- 五阳干才有：甲刃卯、丙戊刃午、庚刃酉、壬刃子
- 阳干帝旺之地即是阳刃
- 过旺则失，中庸为贵

**命理含义：**
- 阳 = 阳干
- 刃 = 刀刃
- 阳刃是凶神，主背禄逐马

### 二、阳刃为什么凶？

**暗藏的凶性（不见时）：**

以甲人刃卯为例：
```
卯刃会做什么？
├── 冲酉 → 坏官禄
├── 卯中乙克己 → 坏财马
└── 卯中乙合庚 → 引来七杀克命
```

**总结：冲官冲财合杀，全是背禄逐马**

### 三、阳刃的明暗变化

**不见时（暗）：**
- 纯坏人，干的都是坏事
- 背禄逐马，与七杀狼狈为奸
- 合而引来七杀克我，又自身以刃刃我

**明见时（有制）：**
- 见庚杀 → 七杀制刃，刃不能呈凶
- 见辛官 → 官星制刃
- 见己财 → 需有丙丁转刃生财，否则被克

**核心口诀：**
> "见与不见，吉凶大有不同"

## 🎯 阳刃格的成格条件

### 一、阳刃驾杀格

**经典案例：薛之谦**
```
丙  己  丙  丙
未  卯  午  亥
```

**格局分析：**
- 月令伤官（己未透）
- 取时上偏官（亥中壬）
- 日坐下刃（丙午），阳刃驾杀
- 时偏官有成有用有格 → 有富贵

**格局成败：**
- ✅ 阳刃驾杀成 → 威权赫赫
- ❌ 伤官多余（月、时都有）
- 应事：父母难存、子嗣有损、婚姻不好

### 二、印绶化刃格

**经典案例：岳飞**
```
癸  甲  甲  戊
卯  卯  子  午
```

**格局分析：**
- 月令阳刃（卯）
- 年干癸印 + 日支子印 → 印绶仁德化刃
- 刀兵（刃）+ 印章（印）→ 大将军

**格局成败：**
- ✅ 印化刃成 → 武权大贵
- ❌ 时干财破印 → 最终不好
- 运限引发：辛亥运 + 辛酉年 → 刃逢冲又逢合 → 刀下之鬼

### 三、阳刃用财格

**女命案例：**
```
丙  庚  壬  丁
寅  子  寅  酉
```

**格局分析：**
- 月令羊刃（子中癸）
- 用财（丙火透年，寅长生）
- 问题：庚枭威胁食神，壬比劫财

**格局成败：**
- ✅ 财透有根 → 财有源
- ❌ 羊刃伤财 → 父亲身体不好、夫条件一般
- ❌ 庚金枭印夺食 → 负能量爆棚、自卑抑郁

## ⚠️ 阳刃格的关键技法

### 一、动刃的条件

**何时动刀子？**

阳刃逢冲逢合而动，但必须是：
```
条件1：背禄逐马之时
   +
条件2：阳刃又动
   =
结果：一刀之灾，必定难免
```

**两种情况：**
1. 原局阳刃先动 + 大运背禄逐马
2. 命局背禄逐马 + 大运流年动阳刃

**案例：岳飞之死**
- 辛亥运：亥卯未合 → 阳刃动了
- 辛酉年：酉冲卯 → 阳刃又逢冲
- 结果：刃逢冲又逢合，彻底引动 → 刀下之鬼

### 二、制刃的方法

**四种方法：**

| 方法 | 作用 | 特点 |
|------|------|------|
| 官杀制刃 | 束缚阳刃 | 最喜，阳刃驾杀 |
| 印绶化刃 | 仁德感化 | 印主仁德，化刃为权 |
| 财星引刃 | 转刃生财 | 需食伤转生，否则被劫 |
| 食神泄刃 | 泄秀生财 | 次优选择 |

**重要提醒：**
> "用凶神，最怕克制太多"
> 如月令阳刃，怕官杀太多，制刃太过则刃无功

### 三、阳刃与十神关系

**阳刃格最喜：**
1. 官杀（制刃、驾杀）→ ⭐⭐⭐⭐⭐
2. 印绶（化刃）→ ⭐⭐⭐⭐
3. 食神（泄刃生财）→ ⭐⭐⭐

**阳刃格最忌：**
1. 比肩劫财 → ⭐⭐⭐⭐⭐（大忌）
2. 偏财（易被劫）→ ⭐⭐⭐⭐
3. 枭印（夺食）→ ⭐⭐⭐

**核心原则：**
> "阳刃格，就不需要比肩和劫财"
> "我身旺了，为什么要比肩劫财来争我的财官？"

## 🔍 实战案例分析

### 案例一：女命公务员（阳刃用财）

**八字：**
```
丙  庚  壬  丁
寅  子  寅  酉
```

**分析步骤：**

1. **定格：** 月令羊刃格

2. **取用：** 用财（丙火）
   - 丙透年干，坐寅长生
   - 寅见三处，食神生财
   - 丙寅一柱 → 公门文职像

3. **找病：**
   - 羊刃伤财 → 财有伤，层次减
   - 庚金枭印 → 威胁食神，断财源
   - 壬水比肩 → 劫财分夺

4. **应事：**
   - 职业：公务员（丙寅文像，戊戌运进公检法）
   - 财富：不会很多（羊刃未制化）
   - 婚姻：全败（羊刃伤财，被劈腿）
   - 性格：自卑，负能量爆棚（枭印夺食）
   - 子女：好（食神生）

5. **流年：**
   - 2008戊子：戊癸合，七杀制刃 → 结婚
   - 2013癸巳：申子辰合局，羊刃最强 → 离婚前男友劈腿
   - 2016丙申：离婚

### 案例二：薛之谦（阳刃驾杀）

**八字：**
```
丙  己  丙  丙
未  卯  午  亥
```

**分析步骤：**

1. **定格：** 月令伤官（己未透）
   - 伤官不配印无财 → 伤官无用
   - 取时上偏官（亥中壬）

2. **格局：** 阳刃驾杀
   - 日坐下刃（丙午）
   - 时偏官有成有用有格

3. **应事：**
   - 有富贵（格局成）
   - 六亲有损（伤官应事）
     - 月上伤官 → 父母难存
     - 日上阳刃 → 妻子不利
     - 时上伤官 → 子嗣难

### 案例三：岳飞（印绶化刃）

**八字：**
```
癸  甲  甲  戊
卯  卯  子  午
```

**分析步骤：**

1. **定格：** 月令阳刃（卯）

2. **格局：** 印绶化刃
   - 年干癸印 + 日支子印
   - 刃（刀兵）+ 印（印章）→ 大将军

3. **败局：**
   - 时干财破印 → 原局有败
   - 最终还原阳刃凶性

4. **运限：**
   - 辛亥运：亥卯未合 → 刃动
   - 辛酉年：酉冲卯 → 刃逢冲
   - 结果：刀下之鬼

## 💡 核心理论总结

### 一、阳刃的本质

**为什么阳刃是凶神？**

1. **违背中庸：**
   > "帝旺，旺过分了，周易讲究中庸之道，不过则无失，过则有失"

2. **背禄逐马：**
   - 冲官 → 坏官禄
   - 冲财 → 坏财马
   - 合杀 → 引来七杀克命

3. **需制化：**
   - 不见时 → 纯坏人
   - 见制化 → 威权赫赫

### 二、阳刃与其他格局的区别

**与从格不同：**
- 阳刃格是正格，需制化
- 从格是特殊格，需顺从

**与比劫不同：**
- 比劫是分夺财官
- 阳刃是伤害性命
- 阳刃比比劫更凶

**与伤官不同：**
- 伤官伤官，可以通过生财向禄
- 阳刃背禄逐马，不向财官

### 三、阳刃格的核心口诀

**成格口诀：**
> "阳刃驾杀，威权赫赫"
> "印绶化刃，仁德感化"
> "食神泄刃，泄秀生财"

**忌神口诀：**
> "阳刃格，就不需要比肩和劫财"
> "任何羊刃格都忌比肩劫财"

**灾祸口诀：**
> "背禄逐马之时，而阳刃又动，则一刀之灾，必定难免"
