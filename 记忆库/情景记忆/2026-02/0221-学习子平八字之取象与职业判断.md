## 📊 核心概念：取象

### 什么是取象？

**核心：**
> "八字不仅论生克，更论'象'。"

**定义：**
- 取象 = 根据五行组合推断职业、性格、事件
- 象 = 具体的形象、比喻
- 八字的象可以直接对应现实

### 取象的重要性

**核心：**
> "很多八字，光看生克制化是断不出来的，必须结合取象。"

## 🎯 五行取象方法

### 1. 刀入肉象

**组合：**
- 辛酉（针/刀）+ 戊辰（土/肉/胃）

**取象：**
- 金器入土（肉）

**职业断语：**
- 外科医生
- 针灸师
- 胃肠科医生

**案例：**
> 辛酉月戊辰日，断为胃肠科医生，极验。

### 2. 土重金埋象

**组合：**
- 土多金弱（己丑、戊辰重叠，金无根或弱）

**取象：**
- 金被土埋，光芒难显

**性格/体型：**
- **懒惰**：金不流通
- **行动力差**：土重滞塞
- **易发胖**：土主肉，土重肉多

**事业：**
- 多从事服务业
- 后勤工作
- 安稳工作
- 难有大成就

**救应：**
- 有木疏土 → 发达
- 财透出 → 发达

### 3. 纳音取象

**戊子（霹雳火）：**
- 霹雳火 = 闪光/显示器
- 职业 = 影像科医生、摄影师、IT硬件（带灯）

## 💼 职业判断总纲

### 第一步：先看格局

**核心原则：**
> "职业判断，先看格局，格局定了大方向。"

**判断：**
1. 官格 → 公职、管理
2. 财格 → 经商、求财
3. 印格 → 文化、学术
4. 食伤格 → 技术、艺术
5. 杀格 → 武职、执法

### 第二步：再看取象

**核心：**
> "格局定大方向，取象定具体职业。"

**例子：**
- 官格 + 刀入肉象 → 外科医生
- 财格 + 土重金埋 → 服务业
- 食伤格 + 霹雳火 → 摄影师

### 第三步：结合神煞

**核心：**
> "神煞可以辅助职业判断。"

**例子：**
- 亡神劫杀临财 → 经商（先败后成）
- 桃花带煞 → 文艺、娱乐
- 华盖 → 艺术、宗教

## 🎯 神煞职业应用

### 1. 亡神劫杀

**查法：**
- 申子辰见亥（亡神）、巳（劫杀）
- 寅午戌见巳（亡神）、亥（劫杀）

**心性：**
- **亡神**：城府、深沉
- 亡神三重 → 不懂人情世故，过于执着

**暴富密码：**
> "亡神劫杀临财，主暴富、突发之财。"

**经商特点：**
> "带亡神劫杀经商，往往需要打一次官司或失败一次，才能开启财富大门。"

**职业倾向：**
- 经商（先败后成）
- 投资（突发之财）
- 女命忌讳：不会做人，人际关系差

### 2. 桃花煞

**含义：**
- 不单指男女情欲
- 也指人缘、魅力

**凶象：**
> "桃花带煞，主克父、慢性病、身体虚弱。"

**桃花带剑：**
- 女命癸酉日
- 酉为桃花又为剑
- 克夫厉害

**职业倾向：**
- 文艺、娱乐
- 服务行业
- 美容美发

### 3. 红艳煞

**含义：**
> "红艳煞，高级偷情。"

**特点：**
- 偷得隐秘
- 越偷越有钱
- 不同于乱搞烂桃花

## 👨‍⚕️ 具体职业判断

### 医疗行业

**取象：**
1. **刀入肉** → 外科医生、针灸师
2. **金木相战**（金克木）→ 骨科
3. **水火相济** → 内科调理
4. **土重** → 脾胃科

### 公职/管理

**格局：**
- 官格成 → 公职
- 杀格有制 → 执法部门

**判断：**
> "官格用印，官印双全，公职有职权。"
> "七杀有制就是有权。"

### 经商/求财

**格局：**
- 财格成 → 经商
- 食伤生财 → 技术求财

**取象：**
- 土重金埋 → 服务业
- 水木清华 → 文化产业
- 火土成局 → 建筑房地产

### 技术/艺术

**格局：**
- 食伤格 → 技术、艺术
- 食神秀气 → 文艺、才华

**取象：**
- 霹雳火 → 摄影、影像
- 金水相生 → 音乐、演说
- 木火通明 → 文化、教育

## 💡 核心理论总结

### 1. 取象三步法

**第一步：看格局**
- 定大方向
- 官、财、印、食伤、杀

**第二步：看取象**
- 定具体职业
- 五行组合对应

**第三步：看神煞**
- 辅助判断
- 细化职业特点

### 2. 取象的核心原则

**核心：**
> "八字不仅论生克，更论象。"

**注意：**
- 不能光看生克制化
- 必须结合取象
- 象是具体的形象

### 3. 神煞的职业应用

**核心：**
> "神煞可以辅助职业判断，但不能主导。"

**原则：**
- 先格局，后神煞
- 神煞取象为主
- 不参与旺衰计算

## 🎯 实战判断流程

### 职业判断8步法

**第1步：定格**
```
月令本气是什么？
├── 正官 → 公职方向
├── 七杀 → 武职方向
├── 正财 → 正财行业
├── 偏财 → 经商求财
├── 正印 → 文化学术
├── 偏印 → 技术才艺
├── 食神 → 智慧才华
└── 伤官 → 技术艺术
```

**第2步：看成败**
```
格局成不成？
├── 成 → 有成就
└── 不成 → 普通人
```

**第3步：看取象**
```
有什么特殊组合？
├── 刀入肉 → 医生
├── 土重金埋 → 服务业
├── 霹雳火 → 摄影/IT
└── 其他取象
```

**第4步：看神煞**
```
有什么职业相关的神煞？
├── 桃花 → 文艺娱乐
├── 华盖 → 艺术宗教
├── 亡神劫杀 → 经商
└── 其他
```

**第5步：看旺衰**
```
身旺还是身弱？
├── 身旺 → 能担财官
└── 身弱 → 喜印比
```

**第6步：看五行**
```
喜用五行是什么？
├── 喜金 → 金融、五金
├── 喜木 → 文化、教育
├── 喜水 → 流动、贸易
├── 喜火 → 能源、科技
└── 喜土 → 建筑、房地产
```

**第7步：看大运**
```
大运好不好？
├── 向禄临马 → 好
└── 背禄逐马 → 不好
```

**第8步：综合判断**
```
结合以上所有信息
├── 格局大方向
├── 取象具体职业
├── 神煞辅助
├── 旺衰调整
└── 大运加减
```
